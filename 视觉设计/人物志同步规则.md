# 人物志同步规则

视觉设计执行器必须优先读取：

1. 世界观/人物志规范.md
2. 世界观/人物志/*.md

之后才允许读取：

- 视觉设计/视觉与美术总规范.md
- 视觉设计/小队视觉/*.md
- 视觉设计/角色规范/*.md
- 具体资源规范

当前视觉体系不单独维护 `阵营规范/`。

青丘、天门及后续小队视觉统一进入：

```text
视觉设计/小队视觉/
```

---

# 同步原则

人物志变更后：

必须同步检查：

- 角色规范
- AI关键词
- 建模规范
- 棋子规范
- 小队视觉规范

同步只允许处理视觉执行，不得在视觉目录重写人物事实。

---

# 禁止事项

禁止出现：

```text
视觉规范中的角色描述
与
人物志中的角色描述
不一致
```

若冲突：

```text
人物志为准
```

禁止使用视觉规范反向修改人物志、世界观或游戏规则。
