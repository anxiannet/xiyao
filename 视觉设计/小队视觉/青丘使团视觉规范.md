## 小队视觉定位

青丘使团视觉核心不是妖艳，而是：

```text
引路
旅者
旧路
狐火残痕
黄昏裂隙
逃亡护送
寻找真相
```

青丘使团不是宫廷阵营，不是妖媚阵营，不是火焰法师团。

视觉上应表现为一支长期在外行动、擅长旧路、狐火残留与裂隙边缘路线的小队。

## 配色

青丘使团不使用“狐火蓝 / 狐火青”作为狐火主方向。

配色应遵循黄昏视觉逻辑：

| 用途 | 色彩方向 | 说明 |
|---|---|---|
| 主基底 | 夜青 / 风沙灰 / 旧布白 | 表现旅者、旧路、长期在外行动 |
| 狐火主色 | 夕照红 / 琥珀橙 / 暮金 | 狐火是黄昏残痕，不是青蓝鬼火 |
| 裂隙辅色 | 暮紫 / 暗红 / 微金 | 表现黄昏伤口与路径错乱 |
| 点缀 | 暮金 | 用于路线、残片、边缘微光 |

禁止：

- 大面积冰蓝。
- 大面积鬼火绿。
- 大面积科技青。
- 把狐火画成水系、冰系或灵气法术。

## 青丘统一识别元素

青丘使团角色至少应具备以下识别元素中的两项：

- 狐耳或狐族特征。
- 旅者装备。
- 狐火残痕。
- 路线纹。
- 裂隙纹。
- 旧路、风沙、归途相关符号。

这些元素用于保证角色缩小到战棋尺度后仍能看出属于同一小队。

---

## 角色表现原则

青丘使团角色必须看起来像长期在外行动的人。

优先表现：

- 方向感。
- 机动性。
- 生存感。
- 同伴关系。
- 对裂隙与旧路的敏锐感知。

战棋地图中，优先级为：

```text
轮廓识别
↓
小队识别
↓
战场功能识别
↓
服饰与材质细节
```

禁止把青丘使团设计成单纯妖媚阵营。
