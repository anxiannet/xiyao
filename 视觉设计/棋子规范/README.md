# 棋子规范

本目录只维护《夕妖》棋子视觉与资源执行标准。

棋子规范已按用途拆分为：

| 文档 | 用途 | 当前状态 |
|---|---|---|
| `游戏用棋子规范.md` | 战斗地图、六边形方向、2D/3D战斗资源、底座、特效、命名 | 当前执行依据 |
| `展示用棋子规范.md` | 英雄展示页、图鉴、收藏模型、宣传单体展示 | 当前执行依据 |
| `棋子审核标准.md` | 游戏用与展示用棋子的审核清单、评分标准、退回条件 | 当前执行依据 |
| `归档/` | 过期或已拆分文档 | 不作为执行依据 |

---

## 来源优先级

棋子制作必须遵循：

```text
世界观人物资料
↓
游戏规则
↓
视觉与美术总规范
↓
小队视觉规范
↓
角色专项视觉规范
↓
棋子规范
↓
生成图 / 建模资产
```

视觉设计不得覆盖人物事实、游戏规则或角色品质等级。

---

## 当前结论

### 游戏用棋子

采用：

```text
六边形地图
Dir1~Dir6 方向体系
3张基础源图 + 镜像覆盖6方向
Pitch 35°
角色高度 200px
源图绿幕背景
源图不含地台
程序统一挂载底座
```

游戏用棋子详见：

```text
视觉设计/棋子规范/游戏用棋子规范.md
```

---

### 展示用棋子

采用：

```text
角色展示优先
允许专属地台
允许轻量展示特效
允许正前 / 侧面 / 背面三视图
不得替代战斗方向资源
```

展示用棋子详见：

```text
视觉设计/棋子规范/展示用棋子规范.md
```

---

## 禁止

- 在多个文档重复维护同一套棋子规则。
- 用展示图覆盖游戏用棋子规范。
- 用游戏方向资源覆盖展示三视图规范。
- 用8方向图作为六边形地图默认生产标准。
- 用“正前 / 侧面 / 正后”三面图冒充六边形战斗方向资源。
- 用棋子表现反向修改人物设定、游戏规则或角色品质等级。
