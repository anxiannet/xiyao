# 棋子规范

战棋棋子不是手办缩小版。

本文件只维护战棋棋子的执行标准。

角色资产等级以 `视觉设计/角色品质等级规范.md` 为准；本文件不另设平行等级体系，只说明各等级在棋子资产中的落地要求。

---

## 来源优先级

棋子制作必须遵循：

```text
世界观人物资料
↓
视觉与美术总规范
↓
角色品质等级规范
↓
角色专项视觉规范
↓
棋子规范
↓
生成图 / 建模资产
```

禁止用棋子表现反向修改人物设定、游戏规则或角色品质等级。

---

## MVP标准

- 3k~8k面
- GLB格式
- 统一高度
- 统一底座
- 移动端可读
- 可进入规则验证与AI对战测试

MVP棋子默认对应：

```text
L2 游戏棋子
```

---

## 识别优先级

```text
轮廓 > 武器 / 核心道具 > 姿势 > 特效 > 花纹
```

说明：

- 轮廓决定远距离识别。
- 武器或核心道具决定职业识别。
- 姿势决定战场功能印象。
- 特效只用于辅助识别，不得遮挡路线。
- 花纹只用于近看质感，不作为战棋识别核心。

---

## 可读性要求

- 200px：识别具体角色。
- 100px：识别职业 / 战场功能。
- 50px：识别阵营。

如果高品质细节影响以上三档可读性，应优先删除细节。

---

## 与角色品质等级的对应关系

| 角色品质等级 | 棋子资产要求 | 是否适合当前MVP |
|---|---|---|
| L1 测试棋子 | 单色材质、基础站姿、基础阵营轮廓、可用于规则验证 | 可用，但不作为正式展示 |
| L2 游戏棋子 | 阵营色明确、职业/核心道具明确、统一底座、移动端可读、可导出GLB | 当前MVP主标准 |
| L3 收藏棋子 | 在L2基础上增加专属底座、轻量专属特效、强轮廓、材质区分 | 正式角色目标 |
| L4 宣传角色 | 高品质材质、阵营背景、完整演出、海报构图 | 不作为战棋棋子标准 |
| L5 核心封面角色 | 品牌级封面构图、顶级展示、强识别资产 | 禁止跳过L2直接制作 |

---

## L1 测试棋子标准

用途：规则验证、AI对战验证、早期站位测试。

要求：

- 单色或少量阵营色。
- 基础站姿。
- 不要求完整材质。
- 不要求专属底座。
- 必须能区分阵营与单位类型。

禁止：

- 投入大量细节。
- 制作宣传级演出。
- 用L1图替代正式角色视觉。

---

## L2 游戏棋子标准

用途：MVP正式对战、移动端展示、GLB建模参考。

要求：

- 阵营识别明确。
- 职业或战场功能明确。
- 武器或核心道具明确。
- 统一高度。
- 统一底座。
- 3k~8k面。
- GLB格式。
- 200px / 100px / 50px 三档可读。

L2是当前MVP的主制作标准。

判断标准：

```text
玩家在移动端战棋视角下，能快速知道：

这个单位属于哪个阵营
这个单位大概做什么
这个单位是否是某个关键角色
```

---

## L3 收藏棋子标准

用途：正式角色资产、展示图、收藏感模型。

L3必须建立在L2可读性已经通过的基础上。

允许增加：

- 专属底座。
- 轻量专属特效。
- 更强角色轮廓。
- 材质区分。
- 角色专属道具强化。

限制：

- 专属底座不得遮挡格子。
- 特效不得超出角色识别主体。
- 长拖尾不得影响棋盘路线判断。
- 不得为了收藏感牺牲移动端读图。

---

## L4 / L5 与棋子的关系

L4和L5属于宣传资产、官网资产和品牌资产，不是战棋棋子的默认制作标准。

棋子可以从L3资产派生宣传图，但不能反向用L4/L5宣传图直接当棋子规范。

禁止：

- 用海报构图制作战棋棋子。
- 用大场景背景替代统一底座。
- 用满屏特效遮挡棋盘。
- 跳过L2验证直接制作L5资产。

---

## 底座规范

MVP统一底座：

- 圆形底座。
- 统一直径。
- 高度统一。
- 不出现大型景观件。
- 不遮挡相邻格子。

L3允许专属底座，但必须满足：

- 仍然保持统一外径。
- 仍然保持棋盘占格清晰。
- 专属元素低矮、轻量、可读。
- 装饰服务角色识别，不服务炫技展示。

---

## 特效规范

棋子特效用于提示阵营、职业或技能来源，不用于制造海报视觉冲击。

允许：

- 小范围脚下光痕。
- 核心道具发光。
- 低强度阵营色微光。
- 不遮挡身体轮廓的轻量粒子。

禁止：

- 超大特效遮挡路线。
- 满屏粒子。
- 大面积火焰、烟雾、爆炸。
- 特效大于角色主体。

---

## 禁止

- 超复杂底座。
- 长拖尾遮挡格子。
- 超大特效遮挡路线。
- 为展示牺牲玩法。
- 为宣传效果牺牲移动端可读性。
- 用手办细节替代战棋识别。
- 创建与角色品质等级规范冲突的新等级。
