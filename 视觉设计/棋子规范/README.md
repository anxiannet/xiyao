## 游戏用资源统一标准

战斗内所有资源统一：

```text
六边形地图
Pitch：45°
Yaw：45°
FOV：30°
```

包括：

- 棋子
- 地块
- 建筑
- 障碍物
- 战斗场景装饰

禁止地图与棋子使用不同镜头。

---

## 游戏棋子

```text
Dir1~Dir6方向体系
3张基础源图+镜像覆盖6方向
角色高度200px
源图绿幕背景
源图不含地台
程序统一挂载底座
```

---

## 展示棋子

```text
Pitch：15°~25°
允许专属地台
允许轻量展示特效
允许正前/侧面/背面三视图
```

展示规范不得覆盖游戏规范。
