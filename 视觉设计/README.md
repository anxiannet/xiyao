# 视觉设计

《夕妖》视觉设计体系。

## 职责范围

视觉设计只负责：

- 视觉体系设计
- 美术规范设计
- 角色设计
- 阵营设计
- 棋子设计
- 地台设计
- 特效设计
- UI视觉规范
- 宣发视觉规范
- 小说视觉规范

不负责：

- 世界观设定
- 游戏规则
- 数值平衡
- AI行为设计
- 前端交互逻辑
- 后端架构设计
