# 小队视觉风格路线

本文档用于定义《夕妖》新小队可选择的视觉风格路线。

视觉风格路线只负责美术方向，不负责世界观设定、规则设计和数值设计。

---

# 选择原则

新小队创建时，必须先选择一条主视觉路线。

选择依据：

1. 人物志
2. 阵营定位
3. 小队战场职能
4. 收藏展示需求

禁止每个角色单独随机选择风格。

---

# T1 裂隙旅者风

定位：当前标准主风格。

核心气质：

- 冒险
- 生存
- 可靠
- 克制
- 神秘

视觉关键词：

- 披风
- 布料
- 皮革
- 狐火
- 符文
- 旅者装备
- 地图袋
- 绳索
- 药囊

适合：

- 青丘使团
- 裂隙探索者
- 商旅组织
- 游侠
- 斥候
- 向导

禁止：

- 过度华丽
- 过度裸露
- 过度机甲化
- 花纹堆砌

- 妖火
- 月夜

- 涂山

- 夜宴组织
- 幻术师

限制：

禁止：

- 低俗卖肉
- 过度裸露
- 只靠身体暴露制造识别度
- 影响棋子阅读的复杂丝带堆叠

# S级 幻想收藏风

定位：展示层风格，不作为小队基础风格。

幻想收藏风不是阵营路线，而是角色品质等级中的展示强化层。

适用场景：

- L3 收藏棋子
- L4 宣传角色
- L5 核心封面角色
- 手办展示图
- 官网KV
- 抽卡展示页

核心气质：

- 华丽
- 梦幻
- 高级
- 收藏感
- 角色魅力最大化

视觉关键词：

- 大型专属底座
- 场景化演出
- 动态特效
- 高级材质
- 强光效
- 强构图
- 角色专属徽章

使用规则：

任何小队都可以在保留本阵营主风格的前提下，使用幻想收藏风进行展示升级。

例如：

- 青丘使团主风格为裂隙旅者风，苏绫L4宣传图可叠加幻想收藏风。
- 天门执法队主风格为天门神权风，玄照L5封面图可叠加幻想收藏风。
- 灵能机甲阵营主风格为灵能机甲风，核心角色手办图可叠加幻想收藏风。

禁止：

- 把幻想收藏风当作普通棋子量产标准
- 为了华丽牺牲战棋可读性
- 为了收藏感覆盖人物志设定
- 为了特效遮挡武器和轮廓

---

# 风格选择公式

小队基础风格 = 人物志 + 阵营定位 + 战场职能

角色展示风格 = 小队基础风格 + 品质等级

幻想收藏风 = L3以上展示强化层

# 最终原则

基础风格决定阵营识别。

幻想收藏风决定展示上限。

不得混淆两者。
