# 小队视觉风格路线

新小队必须先选择一条基础视觉路线。幻想收藏风不是基础路线，而是L3以上展示强化层。

## 基础路线

| 路线 | 关键词 | 适合 |
|---|---|---|
| 裂隙旅者风 | 引路、旅者、狐火、裂隙 | 青丘、探索者、游侠 |
| 天门神权风 | 秩序、裁决、法度、天威 | 天门、巡界司、执法者 |
| 妖族魅影风 | 夜宴、权谋、魅影、月华 | 涂山、妖族贵族、幻术师 |
| 灵能机甲风 | 灵能、机关、创造、技术 | 机关城、天工院、灵械师 |

## 展示强化层

幻想收藏风用于L3收藏棋子、L4宣传角色、L5封面角色。

公式：角色展示风格 = 小队基础风格 + 品质等级。
