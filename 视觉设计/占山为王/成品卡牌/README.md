# 《夕妖：占山为王》成品卡牌合成系统

本目录用于将纯插画资产与统一UI卡框合成为完整卡牌正面，并生成统一卡背。

## 目录

- `正面/`：完整卡牌正面 PNG。
- `背面/`：统一卡背 PNG。
- `模板/`：UI模板规范与模板预览。
- `导出/`：批量导出辅助文件与总览图。
- `card_data.json`：卡牌合成数据。
- `生成完整卡牌.py`：Python + Pillow 合成脚本。

## 使用

在仓库根目录运行：

```bash
python3 视觉设计/占山为王/成品卡牌/生成完整卡牌.py
```

脚本会读取：

- `视觉设计/占山为王/成品卡牌/card_data.json`
- `视觉设计/占山为王/批量生成/插画资产/`

输出：

- `视觉设计/占山为王/成品卡牌/正面/`
- `视觉设计/占山为王/成品卡牌/背面/xiyao_zhanshanweiwang_card_back.png`
- `视觉设计/占山为王/成品卡牌/模板/card_front_template_preview.png`
- `视觉设计/占山为王/成品卡牌/导出/all_cards_contact_sheet.png`

## 规则边界

本系统只做卡牌视觉合成，不修改规则、星级、数量或技能。

技能短文案来自：

`游戏/卡牌游戏/占山为王/V1.2-卡面技能文案.md`
