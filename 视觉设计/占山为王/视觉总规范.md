# 《夕妖：占山为王》视觉总规范

## 文件定位

本文件是《夕妖：占山为王》视觉设计的专项总规范。

本文件只维护视觉方向、卡牌气质、信息层级与执行边界。

不修改：

- 规则
- 数值
- 卡牌技能
- 世界观事实

规则与卡表以以下文件为唯一来源：

- `游戏/卡牌游戏/占山为王/V1.0-核心规则重构.md`
- `游戏/卡牌游戏/占山为王/V1.1-角色与法宝设计规范.md`
- `游戏/卡牌游戏/占山为王/V1.2-基础卡表.md`

# 视觉定位

《占山为王》的视觉定位为：

国风西游妖怪收藏卡。

核心气质：

- 东方幻想
- 妖气
- 山头势力感
- 收藏级桌游
- 清晰可读
- 神话但不压抑

---

# 与《夕妖》总视觉的关系

必须遵循：

`视觉设计/视觉与美术总规范.md`

其中最高原则为：

规则识别 > 战术阅读 > 角色魅力 > 世界观表现。

因此卡面不能为了氛围牺牲文字、星级、战力与卡牌类型识别。

---

# 禁止方向

禁止：

- 纯暗黑恐怖
- 过度写实血腥
- Q版儿童卡通
- 页游廉价堆料
- 纯概念图无可读性
- 用现代科技机甲替代西游妖怪身份
- 把地盘画成复杂战棋地图而非地盘卡

建议融合：

- 国风神话

- 桌游收藏卡
- 暮金与墨色基调
- 适度厚涂质感

- 暗金
- 墨色
- 烟云

- 山头
- 洞府
- 法宝灵光

# 视觉层级

卡牌视觉必须一眼区分：

1. 卡牌类型：妖怪 / 法宝 / 地盘
2. 星级：1星 / 2星 / 3星
3. 战力：1 / 2 / 3
4. 技能：有技能 / 无技能
5. 所属主题：妖怪来源、地盘阵营、法宝功能

---

# 卡牌类型视觉

## 妖怪卡

展示角色主体。

强调：

- 身份
- 星级
- 战力
- 出场技能

## 法宝卡

展示法宝主体。

法宝视觉等级应高于普通妖怪卡。

法宝不是妖怪，不显示战力。

## 地盘卡

展示地点与阵营。

地盘不是角色，不做人物主体。

重点是：

- 阵营识别
- 地名识别
- 可被占领的山头感

---

# 星级视觉

## 1星小妖

定位：基础驻军。

视觉：

- 青铜边框
- 小队感
- 巡山、看守、杂兵但不低劣
- 背景简洁

## 2星精英

定位：中坚驻军。

视觉：

- 银铁边框
- 更强武装
- 先锋、精怪、精英怪气质
- 动态姿态增强

## 3星首领

定位：唯一角色。

视觉：

- 暗金边框
- 专属场景
- 完整角色主体
- 技能图腾或妖气特效
- 收藏感最强

---

# 法宝视觉

法宝统一使用紫金神兵方向。

法宝必须明显区别于妖怪卡：

- 无角色站位
- 以物件为中心
- 法力漩涡、吸纳、吹散、封禁等效果清晰
- 视觉压迫感可以高于3星首领

---

# 地盘视觉

地盘卡应像一张可争夺的领地牌。

不画复杂路径，不引入相邻或距离概念。

四大阵营视觉方向：

| 阵营 | 地盘 | 视觉关键词 |
|---|---|---|
| 白骨 | 白骨岭 / 埋骨坡 / 乱葬岗 | 骨白、荒坡、鬼火、枯木 |
| 火云 | 火焰山 / 翠云山 / 芭蕉洞 | 赤红、熔岩、火云、洞府 |
| 狮驼 | 狮驼岭 / 狮驼洞 / 狮驼国 | 狮金、妖城、荒漠、巨门 |
| 盘丝 | 盘丝洞 / 黄花观 / 濯垢泉 | 毒紫、蛛网、湿雾、幽泉 |
