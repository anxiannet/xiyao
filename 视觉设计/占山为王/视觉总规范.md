# 视觉层级

卡牌视觉必须一眼区分：

1. 卡牌类型：妖怪 / 法宝 / 地盘
2. 星级：1星 / 2星 / 3星
3. 卡名
4. 技能文本
5. 插画主体

妖怪卡不额外展示攻击力、防守力与阵营区。

---

# 卡牌类型视觉

## 妖怪卡

展示角色主体。

强调：

- 名称
- 星级
- 身份
- 简洁技能文本

不展示：

- 攻击力
- 防守力
- 阵营区
- 技能名标签

## 法宝卡

展示法宝主体。

法宝视觉等级应高于普通妖怪卡。

法宝不是妖怪，不显示星级与战力。

## 地盘卡

展示地点与阵营。

地盘不是角色，不做人物主体。

重点是：

- 地名识别
- 阵营识别
- 可被占领的山头感

---

# 星级视觉

## 1星小妖

定位：基础驻军。

视觉：

- 青铜边框
- 小队感
- 巡山、看守、杂兵但不低劣
- 背景简洁

## 2星精英

定位：中坚驻军。

视觉：

- 银铁边框
- 更强武装
- 先锋、精怪、精英怪气质
- 动态姿态增强

## 3星首领

定位：唯一角色。

视觉：

- 暗金边框
- 专属场景
- 完整角色主体
- 技能效果可通过画面暗示
- 收藏感最强

---

# 法宝视觉

法宝统一使用紫金神兵方向。

法宝必须明显区别于妖怪卡：

- 无角色站位
- 以物件为中心
- 法力漩涡、吸纳、吹散、封禁等效果清晰
- 视觉压迫感可以高于3星首领
- 不显示攻击力、防守力、阵营区或技能名标签

---

# 地盘视觉

地盘卡应像一张可争夺的领地牌。

不画复杂路径，不引入相邻或距离概念。

四大阵营视觉方向：

| 阵营 | 地盘 | 视觉关键词 |
|---|---|---|
| 白骨 | 白骨岭 / 埋骨坡 / 乱葬岗 | 骨白、荒坡、鬼火、枯木 |
| 火云 | 火焰山 / 翠云山 / 芭蕉洞 | 赤红、熔岩、火云、洞府 |
| 狮驼 | 狮驼岭 / 狮驼洞 / 狮驼国 | 狮金、妖城、荒漠、巨门 |
| 盘丝 | 盘丝洞 / 黄花观 / 濯垢泉 | 毒紫、蛛网、湿雾、幽泉 |

---

# 印刷与边框

当前卡牌必须支持实体印刷。

要求：

- 63 × 88 mm
- 300 dpi
- 3 mm 出血
- 四边连续花纹
- 四角完整装饰
- 背景与花纹延伸到出血区
- 关键文字进入安全区
