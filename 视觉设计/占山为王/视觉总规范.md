# 统一美术风格锚点

本章节用于解决不同角色、地盘、卡背看起来像不同游戏的问题。

后续所有视觉设计必须统一到同一种美术语言：

**西游妖怪动画电影风。**

卡牌、角色、地盘、法宝、UI、卡背、包装、宣传图都必须像同一款游戏的资产。

禁止出现：

- 妖怪像黑神话
- 法宝像炉石
- 地盘像国风壁纸
- 卡背像玄幻小说封面
- 小妖像动画电影，首领像写实暗黑游戏

## 风格锚点描述

推荐统一风格为：

高品质3D妖怪动画电影风。

核心特征：

- 头部偏大
- 表情夸张
- 动作明确
- 轮廓清楚
- 身体比例英雄化但不写实压迫
- 材质有手办质感
- 明快自然光
- 场景有山寨、山门、栈道、洞府、妖旗等占山元素
- 角色正在做事，而不是站着展示

参考气质：

- 动画电影级3D角色
- 卡通战争节奏
- 部落战争式阅读体验
- 西游山头妖怪世界

注意：

可借鉴动画电影的质感、表情、动作和镜头语言。

禁止直接模仿任何具体商业IP角色、建筑或画面。

## 角色统一原则

所有妖怪角色必须满足：

- 像同一个世界里的妖怪
- 像同一套卡牌里的单位
- 像可以被玩家派出去抢山头的角色

角色必须正在做一件明确的事。

允许动作方向：

- 巡山
- 送信
- 抢旗
- 守洞
- 偷法宝
- 冲锋
- 攻山
- 逃跑
- 指挥
- 吹风
- 吐火
- 搅局

禁止：

- 站着拍照
- 正面证件照
- 只摆造型不参与游戏行为
- 只展示装备不表现性格
- 只靠背景和特效区分角色

## 小妖风格锚点

1星小妖和低级妖怪是统一美术风格的关键。

小妖应具备：

- 大耳朵、大鼻子、大眼神等夸张面部特征
- 明确情绪：机灵、惊慌、得意、卖力、讨好、慌张
- 明确任务感：巡山、报信、举旗、搬东西、守门、逃跑
- 轻喜剧感
- 山寨生活感

小妖不是恐怖怪物。

小妖也不是低龄儿童Q版。

小妖应像动画电影里的妖怪杂兵：

可爱但不幼稚，搞笑但不廉价，弱小但有戏。

## 首领风格统一

3星首领可以更强、更压迫、更有收藏感。

但仍必须属于同一个动画电影风格。

禁止把首领做成：

- 过度写实
- 暗黑怪物
- 黑神话风Boss
- 厚涂玄幻原画
- 冰冷神像

首领应当是：

- 更强的动画电影角色
- 更夸张的表情
- 更明确的动作
- 更强的轮廓
- 更明显的山头势力感

例如：

- 牛魔王应像正在号令群妖抢山头
- 白骨精应像正在算计、偷袭或布局
- 红孩儿应像正在兴奋纵火抢地
- 铁扇公主应像正在控场吹飞守军

## 场景统一原则

背景必须服务“山头争霸”。

优先出现：

- 山寨
- 洞府
- 山门
- 妖旗
- 瞭望塔
- 木栈道
- 山路
- 悬崖
- 瀑布
- 石阶
- 营寨
- 火把
- 灯笼
- 烟尘

次要出现：

- 云海
- 仙山
- 神殿
- 纯风景奇观

禁止让背景变成单纯风景壁纸。

背景应让玩家感觉：

这里是可以被抢、可以驻守、可以出兵的地盘。

## 风格禁止项

禁止混用以下方向：

- 写实暗黑风
- 黑神话风
- 炉石厚涂卡牌风
- 魔兽兵团直接仿制风
- 原神二次元风
- 国漫修仙风
- 纯水墨国风
- 页游传奇风
- 低龄Q版儿童风
- 现代机甲风
- 赛博朋克风
- 单纯概念原画风
- 玄幻小说封面风

这些风格可以作为局部灵感参考，但不能成为《占山为王》的最终美术语言。

## 统一检查标准

每生成一张新图，都必须检查：

1. 放进同一副牌里，是否像同一个游戏？
2. 角色是否正在做事？
3. 是否有山头、洞府、妖旗、栈道、山寨等占山元素？
4. 表情和动作是否一眼可读？
5. 是否偏离成暗黑、写实、二次元、厚涂或玄幻封面？
6. 缩小到手机尺寸后，是否仍能看懂角色和动作？

若不满足，应重新生成或调整提示词。

# 核心视觉原则

本章节是《占山为王》所有视觉设计方案的最高执行原则。

适用范围：

- 妖怪卡
- 法宝卡
- 地盘卡
- 卡背
- UI资产
- 图标
- Logo
- 包装盒
- 宣传图
- 官网视觉
- AI生成提示词

若专项规范与本章节冲突：

以本章节为准。

## 第一原则：一眼看懂抢地盘

玩家必须一眼看懂游戏在玩什么。

《占山为王》是：

妖怪抢地盘欢乐聚会卡牌。

因此所有视觉设计都必须围绕：

抢地盘

展开。

禁止为了世界观氛围牺牲玩法表达。

## 第二原则：争夺优先

优先表现：

- 争夺
- 冲锋
- 抢占
- 守山
- 攻山
- 群妖出动
- 妖怪混战

而非：

- 静态统治
- 孤独英雄
- 黑暗压迫
- 玄幻封印
- 灭世叙事

## 第三原则：欢乐优先于史诗

《占山为王》定位：

- 欢乐聚会卡牌
- 轻策略
- 轻代入
- 15 至 30 分钟

不是：

- 黑暗奇幻
- 史诗战争
- 灭世剧情
- 沉重叙事

视觉应当：

- 有趣
- 热闹
- 夸张
- 有动感
- 有朋友聚会桌游感

玩家看到视觉图时，应产生：

“这游戏好像很好玩。”

而不是：

“这世界观很沉重。”

## 第四原则：动感优先于装饰

视觉焦点应当来自：

- 冲锋
- 抢夺
- 对抗
- 动作
- 表情
- 山头目标

而不是：

- 花纹
- 浮雕
- 边框
- 装饰件
- 复杂暗纹

边框永远服务主体。

禁止出现：

边框比主体更吸引视线。

## 第五原则：缩略图可识别

无论是卡牌、UI、宣传图、封面还是图标，缩小到手机尺寸后，玩家都应在 1 秒内理解：

这是一个妖怪抢山头的游戏。

而不是：

- 玄幻小说
- 修仙游戏
- 暗黑传奇
- 神话壁纸
- 古代封印图

## 第六原则：部落战争式阅读体验

推荐参考：

- Clash of Clans
- Clash Royale
- Boom Beach

学习内容：

- 清晰轮廓
- 强动势
- 大主体
- 高识别度
- 明快色彩
- 欢乐竞争感
- 卡通战争节奏

禁止直接模仿具体角色、建筑或商业IP资产。

仅借鉴阅读体验、视觉节奏和欢乐竞争氛围。

## 第七原则：主体永远第一

视觉权重优先级：

1. 游戏主题
2. 主体角色 / 地盘 / 法宝
3. 动作与情绪
4. 标题与关键信息
5. 装饰元素

任何情况下：

- 花纹不能压过主体
- 边框不能压过主体
- 特效不能压过主体
- 背景不能压过主体

## 一句话原则

所有视觉设计都必须让玩家感受到：

万妖出动，抢山称王。

# 视觉层级

卡牌视觉必须一眼区分：

1. 卡牌类型：妖怪 / 法宝 / 地盘
2. 星级：1星 / 2星 / 3星
3. 卡名
4. 技能文本
5. 插画主体

妖怪卡不额外展示攻击力、防守力与阵营区。

角色表现可以强，但不能压过卡名、星级和技能区。

所有视觉设计都应优先保证“抢地盘”主题与主体动势清晰。

# 卡牌类型视觉

## 妖怪卡

展示角色主体。

强调：

- 名称
- 星级
- 身份
- 表情
- 动作
- 简洁技能文本

不展示：

- 攻击力
- 防守力
- 阵营区
- 技能名标签
- 职业
- 移动方式

妖怪卡应优先保证角色在小尺寸下可识别。

妖怪应像可上场抢山头的单位，而不是静态人物设定图。

妖怪必须符合“西游妖怪动画电影风”，不能与其它角色割裂。

## 法宝卡

展示法宝主体。

法宝视觉等级应高于普通妖怪卡。

法宝不是妖怪，不显示星级与战力。

法宝可使用更强的灵光、旋涡、吸纳、吹散、封禁等效果，但卡面信息不能混乱。

法宝应像能打乱局势的欢乐搅局牌，而不是沉重神器设定图。

法宝虽然不是角色，但材质、光照、边框和场景气质仍需与妖怪卡属于同一游戏。

## 地盘卡

展示地点与阵营。

地盘不是角色，不做人物主体。

重点是：

- 地名识别
- 阵营识别
- 可被占领的山头感
- 可被争夺的目标感

地盘卡可以保留国风山头、洞府、妖城、荒坡等场景语言，但不引入路径、相邻或距离概念。

地盘视觉应让玩家感到“这块地方值得抢”。

地盘必须与角色背景属于同一世界，优先使用山寨、山门、洞府、栈道、妖旗等可占领场景语言。

# 妖怪角色比例

妖怪角色采用动画电影式英雄化比例。

建议：

- 头部约占身高 1/3
- 身体约占身高 2/3
- 手、拳头、武器允许明显放大
- 耳朵、鼻子、角、牙、肩甲、披风、旗帜等轮廓部件要清楚
- 小妖可以更夸张、更有喜剧感
- 首领可以更厚重，但不能脱离统一动画电影风

目的：

- 小图可读
- 表情强烈
- 角色像游戏英雄单位
- 适合实体卡和网页展示
- 放在同一副牌里像同一个游戏

禁止：

- 头太小
- 身体太写实
- 武器太细
- 轮廓太碎
- 远景全身导致情绪看不清
- 首领写实化、小妖动画化造成风格割裂

# 表情与动作

表情是本视觉体系的核心。

每张妖怪卡都必须有明确情绪。

允许情绪方向：

- 愤怒
- 狂笑
- 得意
- 奸诈
- 惊怒
- 挑衅
- 暴躁
- 贪婪
- 阴险
- 机灵
- 慌张
- 讨好
- 兴奋

动作推荐：

- 冲锋
- 跳劈
- 怒吼
- 挥砸
- 施法
- 指挥群妖
- 护住法宝
- 偷袭探头
- 扑向镜头
- 巡山
- 送信
- 举旗
- 抢旗
- 守洞
- 搬运
- 逃跑

禁止默认站桩。

角色应有身体前倾、扭转、蓄力、披风飞扬、烟尘爆发等动态要素。

动作应服务抢地盘、守山头、攻山、搅局等游戏体验。

# 镜头与构图

推荐使用近景或中近景。

构图要求：

- 角色主体占据画面主要面积
- 头部、拳头、武器可向镜头方向夸张放大
- 轻微广角感，形成扑面而来的动感
- 背景简化，不能抢主体
- 保留卡名、星级和技能区安全空间
- 背景最好能暗示山头、洞府、山寨、栈道或妖旗

避免：

- 远景全身小人
- 背景大于角色
- 角色被复杂UI切碎
- 地台式展示
- 静态模型展示

对于卡背、包装和宣传视觉，允许使用群妖冲锋、山头目标、妖旗、烟尘和大标题形成动感构图，但仍必须保持干净、可读，不得堆满花纹。

# 材质与渲染

采用高质量3D卡通写实渲染。

整体渲染方向：

- 动画电影级光照
- 明快自然光
- 清晰体积感
- 不过度写实
- 不做暗黑厚涂
- 不做二次元赛璐璐

## 金属

- 圆润
- 厚重
- 高光明确
- 边缘干净
- 可有轻微磨损，但不能脏乱

## 皮肤

- 磨砂质感
- 有体积感
- 不追求真实毛孔
- 可有妖怪皮肤色彩变化，但整体风格统一

## 毛发

- 块面化
- 大方向清晰
- 不做根根分明的写实毛发

## 布料与披风

- 色块明确
- 动态飘动
- 边缘可有破损
- 纹样不要过碎

# 星级视觉

## 1星小妖

定位：基础驻军。

视觉：

- 青铜边框
- 小队感
- 巡山、看守、杂兵但不低劣
- 表情鲜明
- 背景简洁
- 轻喜剧感
- 正在执行任务

1星小妖是统一风格的重要锚点，必须体现动画电影式妖怪杂兵气质。

## 2星精英

定位：中坚驻军。

视觉：

- 银铁边框
- 更强武装
- 先锋、精怪、精英怪气质
- 动态姿态增强
- 武器或法术轮廓更明显
- 仍保持动画电影式表情与动作

## 3星首领

定位：唯一角色。

视觉：

- 暗金边框
- 专属动作
- 近景英雄构图
- 强烈表情
- 明确武器或法宝
- 技能效果可通过画面暗示
- 收藏感最强
- 更强但不脱离同一美术体系

3星首领不能变成另一套暗黑写实游戏资产。

---

# 法宝视觉

法宝统一使用紫金神兵方向。

法宝必须明显区别于妖怪卡：

- 无角色站位
- 以物件为中心
- 法力漩涡、吸纳、吹散、封禁等效果清晰
- 视觉压迫感可以高于3星首领
- 不显示攻击力、防守力、阵营区或技能名标签

法宝卡可以比妖怪卡更神秘，但仍需明快可读。

法宝视觉应体现“搅局”和“改变局势”，不能只做成静态神器展示。

法宝应与妖怪角色共享同一套光照、材质与卡牌质感。

---

# 地盘视觉

地盘卡应像一张可争夺的领地牌。

不画复杂路径，不引入相邻或距离概念。

四大阵营视觉方向：

| 阵营 | 地盘 | 视觉关键词 |
|---|---|---|
| 白骨 | 白骨岭 / 埋骨坡 / 乱葬岗 | 骨白、荒坡、鬼火、枯木 |
| 火云 | 火焰山 / 翠云山 / 芭蕉洞 | 赤红、熔岩、火云、洞府 |
| 狮驼 | 狮驼岭 / 狮驼洞 / 狮驼国 | 狮金、妖城、荒漠、巨门 |
| 盘丝 | 盘丝洞 / 黄花观 / 濯垢泉 | 毒紫、蛛网、湿雾、幽泉 |

地盘视觉必须有“能被抢、值得抢”的目标感。

地盘场景应尽量与角色背景可互相连通，形成同一个西游妖怪山头世界。

---

# 卡背视觉

卡背不应做成玄幻海报、古代封印图或复杂神器图。

卡背应体现：

- 欢乐聚会卡牌
- 抢山头
- 群妖出动
- 轻松竞争
- 明快动感
- 与角色同一套动画电影风格

公共牌库卡背允许出现标题文案。

标题层级：

1. `占山为王` 为主标题，必须最大、最醒目
2. `夕妖` 为品牌小字，尺寸明显小于主标题
3. 不建议添加其他文案

推荐构图：

- 远处山头 / 洞府 / 妖寨
- 山顶插妖旗
- 山脚群妖向上冲
- 主标题“占山为王”压在视觉中心
- 边框保持简洁，只保留细金边与必要角饰

禁止：

- 花纹太多
- 边框太重
- 像宣传海报而不像卡背
- 文字过多
- 夕妖大于占山为王
- 阴暗压抑
- 与角色卡美术风格割裂

---

# 印刷与边框

当前卡牌必须支持实体印刷。

要求：

- 63 × 88 mm
- 300 dpi
- 3 mm 出血
- 四边连续边框
- 四角完整装饰
- 背景与边框延伸到出血区
- 关键文字进入安全区
- 角色可以破框，但不能影响裁切和信息阅读

边框应服务主体和阅读。

禁止用复杂边框、过度浮雕或碎花纹削弱主体。
