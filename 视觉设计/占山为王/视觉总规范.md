# 禁止方向

禁止：

- 纯暗黑恐怖
- 过度写实血腥
- 低龄儿童Q版
- 页游廉价堆料
- 纯概念图无可读性
- 静态证件照式站桩
- 地台式模型展柜图
- 用现代科技机甲替代西游妖怪身份
- 把地盘画成复杂战棋地图而非地盘卡
- 把卡牌做成复杂竞技TCG信息板
- 在妖怪卡面显示攻击力、防守力、阵营区、职业、移动方式

# 视觉层级

卡牌视觉必须一眼区分：

1. 卡牌类型：妖怪 / 法宝 / 地盘
2. 星级：1星 / 2星 / 3星
3. 卡名
4. 技能文本
5. 插画主体

妖怪卡不额外展示攻击力、防守力与阵营区。

角色表现可以强，但不能压过卡名、星级和技能区。

---

# 卡牌类型视觉

## 妖怪卡

展示角色主体。

强调：

- 名称
- 星级
- 身份
- 表情
- 动作
- 简洁技能文本

不展示：

- 攻击力
- 防守力
- 阵营区
- 技能名标签
- 职业
- 移动方式

妖怪卡应优先保证角色在小尺寸下可识别。

## 法宝卡

展示法宝主体。

法宝视觉等级应高于普通妖怪卡。

法宝不是妖怪，不显示星级与战力。

法宝可使用更强的灵光、旋涡、吸纳、吹散、封禁等效果，但卡面信息不能混乱。

## 地盘卡

展示地点与阵营。

地盘不是角色，不做人物主体。

重点是：

- 地名识别
- 阵营识别
- 可被占领的山头感

地盘卡可以保留国风山头、洞府、妖城、荒坡等场景语言，但不引入路径、相邻或距离概念。

---

# 妖怪角色比例

妖怪角色采用英雄化比例。

建议：

- 头部约占身高 1/3
- 身体约占身高 2/3
- 手、拳头、武器允许明显放大
- 角、耳、肩甲、披风等轮廓部件要清楚

目的：

- 小图可读
- 表情强烈
- 角色像游戏英雄单位
- 适合实体卡和网页展示

禁止：

- 头太小
- 身体太写实
- 武器太细
- 轮廓太碎
- 远景全身导致情绪看不清

---

# 表情与动作

表情是本视觉体系的核心。

每张妖怪卡都必须有明确情绪。

允许情绪方向：

- 愤怒
- 狂笑
- 得意
- 奸诈
- 惊怒
- 挑衅
- 暴躁
- 贪婪
- 阴险

动作推荐：

- 冲锋
- 跳劈
- 怒吼
- 挥砸
- 施法
- 指挥群妖
- 护住法宝
- 偷袭探头
- 扑向镜头

禁止默认站桩。

角色应有身体前倾、扭转、蓄力、披风飞扬、烟尘爆发等动态要素。

---

# 镜头与构图

推荐使用近景或中近景。

构图要求：

- 角色主体占据画面主要面积
- 头部、拳头、武器可向镜头方向夸张放大
- 轻微广角感，形成扑面而来的压迫感
- 背景简化，不能抢主体
- 保留卡名、星级和技能区安全空间

避免：

- 远景全身小人
- 背景大于角色
- 角色被复杂UI切碎
- 地台式展示

---

# 材质与渲染

采用高质量3D卡通写实渲染。

## 金属

- 圆润
- 厚重
- 高光明确
- 边缘干净

## 皮肤

- 磨砂质感
- 有体积感
- 不追求真实毛孔

## 毛发

- 块面化
- 大方向清晰
- 不做根根分明的写实毛发

## 布料与披风

- 色块明确
- 动态飘动
- 边缘可有破损
- 纹样不要过碎

# 星级视觉

## 1星小妖

定位：基础驻军。

视觉：

- 青铜边框
- 小队感
- 巡山、看守、杂兵但不低劣
- 表情鲜明
- 背景简洁

## 2星精英

定位：中坚驻军。

视觉：

- 银铁边框
- 更强武装
- 先锋、精怪、精英怪气质
- 动态姿态增强
- 武器或法术轮廓更明显

## 3星首领

定位：唯一角色。

视觉：

- 暗金边框
- 专属动作
- 近景英雄构图
- 强烈表情
- 明确武器或法宝
- 技能效果可通过画面暗示
- 收藏感最强

---

# 法宝视觉

法宝统一使用紫金神兵方向。

法宝必须明显区别于妖怪卡：

- 无角色站位
- 以物件为中心
- 法力漩涡、吸纳、吹散、封禁等效果清晰
- 视觉压迫感可以高于3星首领
- 不显示攻击力、防守力、阵营区或技能名标签

法宝卡可以比妖怪卡更神秘，但仍需明快可读。

---

# 地盘视觉

地盘卡应像一张可争夺的领地牌。

不画复杂路径，不引入相邻或距离概念。

四大阵营视觉方向：

| 阵营 | 地盘 | 视觉关键词 |
|---|---|---|
| 白骨 | 白骨岭 / 埋骨坡 / 乱葬岗 | 骨白、荒坡、鬼火、枯木 |
| 火云 | 火焰山 / 翠云山 / 芭蕉洞 | 赤红、熔岩、火云、洞府 |
| 狮驼 | 狮驼岭 / 狮驼洞 / 狮驼国 | 狮金、妖城、荒漠、巨门 |
| 盘丝 | 盘丝洞 / 黄花观 / 濯垢泉 | 毒紫、蛛网、湿雾、幽泉 |

---

# 印刷与边框

当前卡牌必须支持实体印刷。

要求：

- 63 × 88 mm
- 300 dpi
- 3 mm 出血
- 四边连续边框
- 四角完整装饰
- 背景与边框延伸到出血区
- 关键文字进入安全区
- 角色可以破框，但不能影响裁切和信息阅读
