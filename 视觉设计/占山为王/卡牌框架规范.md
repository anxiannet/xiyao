# 《夕妖：占山为王》卡牌框架规范

## 文件定位

本文件定义《占山为王》实体卡与网页卡的版式框架。

本文件只定义视觉与信息层级，不定义规则内容。

卡牌名称、数量、技能、星级、战力以：

`游戏/卡牌游戏/占山为王/V1.2-基础卡表.md`

为唯一来源。

---

# 卡牌定位

《占山为王》卡牌定位为：欢乐聚会卡牌。

视觉目标：

- 一眼看懂
- 一眼认出角色
- 聚会场景快速阅读
- 收藏级但不复杂
- 适合实体打印与网页展示

参考方向：

- 三国杀式快速阅读
- 炉石式角色识别
- 爆炸猫式聚会易读性

不采用重竞技卡牌或复杂战争桌游的信息密度。

---

# 标准尺寸与印刷

实体卡建议尺寸：

- 63 × 88 mm
- 竖版
- 圆角
- 适配卡套

印刷文件必须预留：

- 3mm 出血
- 安全区内放置文字与关键图标
- 背景、花纹、边框必须延伸至出血区

禁止：

- 白边
- 断边
- 背景突然结束
- 关键文字贴近裁切线

网页展示可使用同一比例。

---

# 卡面信息层级

卡面必须清晰展示：

1. 卡名
2. 星级或法宝类型标识
3. 插画主体
4. 简洁技能文本

妖怪卡不展示：

- 攻击力
- 防守力
- 攻 / 守数值区
- 妖怪阵营区
- 技能类型标签

说明：

- 战斗规则由星级与规则结算承担，不在卡面另设攻击力、防守力。
- 妖怪来源/归属只作为卡表中的世界观说明，不在卡面作为阵营规则展示。
- 所有首领技能默认为登场触发，卡面不显示“登场技”“主动技”“被动技”等技能名。

---

# 通用卡面结构

从上到下：

1. 顶部卡名区
2. 星级 / 法宝类型徽章
3. 中央插画区
4. 下方简洁技能区
5. 四周连续装饰花纹与出血边框

新版基础结构：

```text
┌────────────────────┐
│ 四周花纹与出血区     │
│                    │
│ 金鼻白毛老鼠精       │
│                    │
│      主插画          │
│                    │
│ 简洁技能描述         │
│                    │
│ 四周花纹与出血区     │
└────────────────────┘
```

---

# 妖怪名称留位规范

妖怪名称是卡面最高优先级文字。

名称栏必须按最长首领名预留空间：

`金鼻白毛老鼠精`

要求：

- 至少可完整容纳7个中文字。
- 不得遮挡、裁切或缩到不可读。
- 不得被花纹压住。
- 名称应是卡面最大或最醒目的文字。
- 长名优先横排完整显示；若必须换行，仍需保持清晰可读。

---

# 技能文本规范

所有有技能的妖怪牌，默认都是出场时触发。

卡面不显示技能名。

不显示：

- 登场技
- 出场技
- 主动技
- 被动技
- 技能类型括号

技能区只显示效果文本。

示例：

```text
查看一名玩家手牌。
```

```text
选择1块地盘，将其中1只守军吹回手牌。
```

```text
获得其中1只守军，将其加入手牌。
```

技能文本长度控制：

- 1至2行优先
- 玩家应能在3秒内读完
- 超过两行视为过长，需要调整卡面或文案

---

# 字体方向

建议使用具备古意但清晰的中文标题字。

正文必须优先可读。

禁止使用难以辨认的书法字体作为技能正文。

名称字体可以更有风格，但必须完整可读。

---

# 边框与花纹系统

卡面四周必须填满连续花纹。

要求：

- 顶部有花纹
- 底部有花纹
- 左右有花纹
- 四角有完整花纹
- 花纹延伸至出血区
- 裁切后仍保持完整收藏卡感

边框方向：

| 卡牌类型 | 边框方向 |
|---|---|
| 1星小妖 | 青铜边框 |
| 2星精英 | 银铁边框 |
| 3星首领 | 暗金边框 |
| 法宝 | 紫金神兵边框 |
| 地盘 | 阵营色边框 |

---

# 图标位置

星级图标应固定在卡面上方或侧上方，保持统一。

法宝卡不显示星级与战力，改显示法宝类型徽章。

地盘卡不显示星级与战力，改显示阵营徽章。

妖怪卡不设置攻击力、防守力图标。

---

# 可读性要求

所有卡牌缩小到手机屏幕宽度一半时，仍需识别：

- 卡名
- 卡牌类型
- 星级或法宝类型
- 是否有技能文本

---

# 禁止项

禁止：

- 出现攻击力、防守力、攻3、守3等区域
- 在妖怪卡上展示阵营区
- 显示“登场技”“主动技”“被动技”等技能名
- 复杂装饰压住卡名
- 插画盖住技能文字
- 星级位置不统一
- 不同卡牌随意换版式
- 为了收藏感牺牲规则阅读
- 未预留3mm出血导致印刷白边
