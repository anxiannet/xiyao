# 《夕妖：占山为王》卡牌框架规范

## 文件定位

本文件定义《占山为王》实体卡与网页卡的版式框架。

本文件只定义视觉与信息层级，不定义规则内容。

卡牌名称、数量、技能、战力以：

`游戏/卡牌游戏/占山为王/V1.2-基础卡表.md`

为唯一来源。

---

# 标准尺寸

实体卡建议尺寸：

- 63 × 88 mm
- 竖版
- 圆角
- 适配卡套

网页展示可使用同一比例。

---

# 卡面信息层级

卡面必须清晰展示：

1. 卡名
2. 卡牌类型
3. 星级或法宝标识
4. 战力或无战力标识
5. 技能文本
6. 来源/归属
7. 插画主体

---

# 通用卡面结构

从上到下：

1. 顶部卡名区
2. 左上星级 / 类型徽章
3. 右上战力 / 法宝标识
4. 中央插画区
5. 下方技能区
6. 底部来源/归属与版本信息

---

# 字体方向

建议使用具备古意但清晰的中文标题字。

正文必须优先可读。

禁止使用难以辨认的书法字体作为技能正文。

---

# 文字长度控制

技能文本尽量保持一到两行。

若技能文本过长，优先调整文案排版，不得缩小到不可读。

---

# 边框系统

| 卡牌类型 | 边框方向 |
|---|---|
| 1星小妖 | 青铜边框 |
| 2星精英 | 银铁边框 |
| 3星首领 | 暗金边框 |
| 法宝 | 紫金神兵边框 |
| 地盘 | 阵营色边框 |

---

# 图标位置

星级图标应固定在左上。

战力数字应固定在右上。

法宝卡不显示战力数字，改显示法宝类型徽章。

地盘卡不显示星级与战力，改显示阵营徽章。

---

# 可读性要求

所有卡牌缩小到手机屏幕宽度一半时，仍需识别：

- 卡名
- 星级或类型
- 战力
- 是否有技能

---

# 禁止项

禁止：

- 复杂装饰压住卡名
- 插画盖住技能文字
- 星级与战力位置不统一
- 不同卡牌随意换版式
- 为了收藏感牺牲规则阅读
