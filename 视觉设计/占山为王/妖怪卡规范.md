# 《夕妖：占山为王》妖怪卡规范

## 文件定位

本文件定义1星小妖、2星精英、3星首领的视觉规范。

角色名称、数量、星级、战力、技能以：

`游戏/卡牌游戏/占山为王/V1.2-基础卡表.md`

为唯一来源。

卡面版式、名称留位、印刷出血、技能文本展示方式以：

`视觉设计/占山为王/卡牌框架规范.md`

为统一来源。

---

# 妖怪卡共同原则

妖怪卡必须体现：

- 西游妖怪身份
- 欢乐聚会卡牌的快速阅读
- 星级层级
- 收藏级卡牌质感

妖怪卡不是单纯人物立绘，必须服务聚会桌游阅读。

---

# 卡面展示原则

妖怪卡保留：

- 名称
- 星级
- 插画
- 简洁技能文本

妖怪卡不展示：

- 攻击力
- 防守力
- 攻 / 守数值区
- 阵营区
- 技能类型标签

说明：

- 战斗由星级和规则结算，不在卡面额外展示攻击力或防守力。
- 妖怪来源/归属只作为世界观说明，不作为卡面阵营规则展示。
- 有技能的妖怪默认都是登场触发，卡面不写“登场技”“主动技”“被动技”。

---

# 妖怪名称规范

妖怪名称必须完整、醒目、优先可读。

名称栏按最长名称预留：

`金鼻白毛老鼠精`

要求：

- 至少容纳7个中文字。
- 名称不得被花纹、角色、光效遮挡。
- 长名不得缩成不可读小字。
- 名称应优先横排完整显示。

---

# 技能文本规范

技能区只写效果文本。

示例：

```text
查看一名玩家手牌。
```

```text
选择1块地盘，将其中1只守军吹回手牌。
```

```text
获得其中1只守军，将其加入手牌。
```

技能文本控制在1至2行。

目标是玩家3秒内读完。

---

# 1星小妖视觉

定位：基础驻军。

视觉关键词：

- 巡山
- 守洞
- 跑腿
- 小队感
- 青铜边框

画面要求：

- 半身或三分之二身
- 背景简化
- 姿态明确
- 不做过度华丽特效
- 四周花纹完整但不压住名称

代表：

- 小钻风
- 有来有去
- 精细鬼
- 伶俐虫
- 急如火
- 快如风

---

# 2星精英视觉

定位：中坚驻军。

视觉关键词：

- 先锋
- 精怪
- 武装增强
- 银铁边框
- 比1星更有压迫感

画面要求：

- 动态姿态
- 武器或妖气更明显
- 可有轻度场景背景
- 不能强过3星首领
- 不显示攻击力与防守力

代表：

- 虎先锋
- 奔波儿灞
- 灞波儿奔
- 金角大王
- 银角大王

说明：

金角大王、银角大王虽为知名角色，但当前规则中作为2星精英，无技能。因此视觉可比普通2星更有辨识度，但不能使用3星首领边框。

---

# 3星首领视觉

定位：唯一角色。

视觉关键词：

- 暗金边框
- 专属场景
- 强身份识别
- 技能特征可视化
- 收藏级主卡

画面要求：

- 角色主体必须突出
- 背景可呼应来源/归属
- 可加入专属技能特效
- 不得改变角色核心身份
- 名称栏必须能容纳“金鼻白毛老鼠精”级别长名
- 不显示攻击力、防守力或阵营区

代表：

- 牛魔王
- 白骨精
- 红孩儿
- 铁扇公主
- 大鹏精
- 九灵元圣
- 金鼻白毛老鼠精

---

# 技能视觉化原则

技能效果可通过插画暗示，但不能替代规则文本。

示例：

| 角色 | 技能视觉方向 |
|---|---|
| 白骨精 | 骨影、伪装、窥视感 |
| 黄风怪 | 风沙、吹散低阶守军 |
| 红孩儿 | 三昧真火、火圈清场 |
| 牛魔王 | 群妖响应、号令姿态 |
| 大鹏精 | 俯冲、突袭、风压 |
| 九灵元圣 | 多首狮影、守势、威压 |

---

# 来源/归属表现

来源/归属只用于世界观说明，不限制规则。

视觉上可以通过背景表达来源，例如：

- 白骨精：白骨岭
- 红孩儿：号山 / 火云洞
- 铁扇公主：芭蕉洞
- 大鹏精：狮驼岭 / 狮驼国

但不得让玩家误以为妖怪只能驻守对应地盘。

妖怪卡面不单独展示阵营区。

---

# 印刷与边框

妖怪卡必须预留3mm出血。

四周必须有连续花纹：

- 顶部花纹
- 底部花纹
- 左右花纹
- 四角花纹

花纹必须延伸到出血区。

裁切后不得出现白边或断边。

---

# 禁止项

禁止：

- 把1星小妖画得比3星首领更华丽
- 让2星精英使用3星首领边框
- 技能视觉与规则不符
- 为了酷炫加入不存在的现代武器
- 把来源/归属画成规则限制
- 显示攻击力、防守力、攻3、守3
- 显示妖怪阵营区
- 显示“登场技”“主动技”“被动技”等技能名
- 名称栏无法完整显示“金鼻白毛老鼠精”
