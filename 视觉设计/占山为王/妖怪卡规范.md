# 《夕妖：占山为王》妖怪卡规范

## 文件定位

本文件定义1星小妖、2星精英、3星首领的视觉规范。

角色名称、数量、战力、技能以：

`游戏/卡牌游戏/占山为王/V1.2-基础卡表.md`

为唯一来源。

---

# 妖怪卡共同原则

妖怪卡必须体现：

- 西游妖怪身份
- 驻军与攻山用途
- 星级战力差异
- 收藏级卡牌质感

妖怪卡不是单纯人物立绘，必须服务桌游阅读。

---

# 1星小妖视觉

定位：基础驻军。

视觉关键词：

- 巡山
- 守洞
- 跑腿
- 小队感
- 青铜边框

画面要求：

- 半身或三分之二身
- 背景简化
- 姿态明确
- 不做过度华丽特效

代表：

- 小钻风
- 有来有去
- 精细鬼
- 伶俐虫
- 急如火
- 快如风

---

# 2星精英视觉

定位：中坚驻军。

视觉关键词：

- 先锋
- 精怪
- 武装增强
- 银铁边框
- 比1星更有压迫感

画面要求：

- 动态姿态
- 武器或妖气更明显
- 可有轻度场景背景
- 不能强过3星首领

代表：

- 虎先锋
- 奔波儿灞
- 灞波儿奔
- 金角大王
- 银角大王

说明：

金角大王、银角大王虽为知名角色，但当前规则中作为2星精英，无技能。因此视觉可比普通2星更有辨识度，但不能使用3星首领边框。

---

# 3星首领视觉

定位：唯一角色。

视觉关键词：

- 暗金边框
- 专属场景
- 强身份识别
- 技能特征可视化
- 收藏级主卡

画面要求：

- 角色主体必须突出
- 背景应与来源/归属对应
- 可加入专属技能特效
- 不得改变角色核心身份

代表：

- 牛魔王
- 白骨精
- 红孩儿
- 铁扇公主
- 大鹏精
- 九灵元圣

---

# 技能视觉化原则

技能效果可通过插画暗示，但不能替代规则文本。

示例：

| 角色 | 技能视觉方向 |
|---|---|
| 白骨精 | 骨影、伪装、窥视感 |
| 黄风怪 | 风沙、吹散低阶守军 |
| 红孩儿 | 三昧真火、火圈清场 |
| 牛魔王 | 群妖响应、号令姿态 |
| 大鹏精 | 俯冲、突袭、风压 |
| 九灵元圣 | 多首狮影、守势、威压 |

---

# 来源/归属表现

来源/归属只用于世界观说明，不限制规则。

视觉上可以通过背景表达来源，例如：

- 白骨精：白骨岭
- 红孩儿：号山 / 火云洞
- 铁扇公主：芭蕉洞
- 大鹏精：狮驼岭 / 狮驼国

但不得让玩家误以为妖怪只能驻守对应地盘。

---

# 禁止项

禁止：

- 把1星小妖画得比3星首领更华丽
- 让2星精英使用3星首领边框
- 技能视觉与规则不符
- 为了酷炫加入不存在的现代武器
- 把来源/归属画成规则限制
