# 《夕妖：占山为王》视觉设计

本目录用于维护《夕妖：占山为王》的卡牌、地盘、法宝、UI与生成规范。

本目录只负责视觉设计，不负责修改：

- 世界观设定
- 游戏规则
- 卡牌数值
- 程序实现
