# 《夕妖：占山为王》视觉设计

本目录用于维护《夕妖：占山为王》的卡牌、地盘、法宝、UI与生成规范。

本目录只负责视觉设计，不负责修改：

- 世界观设定
- 游戏规则
- 卡牌数值
- 程序实现

---

# 当前读取顺序

当前阶段以 UI 重构与商品化为主。

读取顺序：

1. `视觉设计/占山为王/当前状态.md`
2. `视觉设计/占山为王/UI修订说明-地盘横置方案.md`

7. `视觉设计/占山为王/视觉总规范.md`

# 文件结构

| 文件 | 职责 |
|---|---|
| `当前状态.md` | 当前阶段目标、边界、优先级与执行顺序 |
| `UI修订说明-地盘横置方案.md` | 地盘横置、卡背系统、卡牌类型识别的最高优先级说明 |
| `UI资产规范.md` | 可复用透明 UI 资产拆分与生成要求 |

| `卡牌框架规范.md` | 卡面结构、信息层级、版式规范；若与地盘横置方案冲突，以地盘横置方案为准 |
| `妖怪卡规范.md` | 1星小妖、2星精英、3星首领视觉规范 |

生成目录：

```text
视觉设计/占山为王/UI资产/
├── monster/
├── artifact/
├── land/
├── back/
├── common/
└── README.md
```

生成资产包括：

- 妖怪卡通用边框
- 妖怪卡星级
- 妖怪卡名称牌匾
- 妖怪卡技能卷轴
- 法宝卡边框
- 法宝卡技能铭牌
- 地盘卡横版边框
- 地盘阵营徽记
- 公共牌库卡背
- 地盘卡背

要求：

- 透明PNG
- 不含角色
- 不含文字
- 不含技能内容
- 不含名称
- 不含水印

生成一次后长期复用。

- UI元素

的统一视觉体系。
