# 《夕妖：占山为王》视觉设计

本目录用于维护《夕妖：占山为王》的卡牌、地盘、法宝、UI与生成规范。

本目录只负责视觉设计，不负责修改：

- 世界观设定
- 游戏规则
- 卡牌数值
- 程序实现

---

## 生成UI资产

读取顺序：

```text
视觉总规范
↓
UI与图标规范
```

生成：

- 1星边框
- 2星边框
- 3星边框
- 法宝边框
- 地盘边框
- 卡牌背面
- 名称栏
- 技能栏

要求：

- 透明PNG
- 不含角色
- 不含文字
- 不含技能内容

生成一次后长期复用。

---

## 生成角色立绘

只生成角色主体。

不生成：

- 边框
- 名称
- 技能区
- UI元素

最终通过固定UI进行合成。

# 当前视觉目标

打造一套：

- 西游妖怪
- 收藏级卡牌
- 夸张情绪
- 高辨识度轮廓
- 3D卡通写实
- 清晰可读
- 适合实体卡与网页展示

的统一视觉体系。
