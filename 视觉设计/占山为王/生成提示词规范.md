# 当前卡牌定位

《占山为王》定位为欢乐聚会卡牌。

生成目标：

- 一眼看懂
- 一眼认出角色
- 名称完整清晰
- 技能简洁可读
- 适合实体打印
- 四周装饰完整

生成图不得再采用复杂竞技卡牌信息框架。

# 通用提示词结构

推荐结构：

1. 卡牌类型
2. 角色 / 法宝 / 地盘名称
3. 星级或类型定位
4. 主要视觉主体
5. 背景场景
6. 色彩与光效
7. 欢乐聚会卡牌风格
8. 印刷出血与连续花纹
9. 禁止项

说明：

妖怪来源/归属可作为插画背景参考，但不得生成阵营区或阵营标签。

# 卡面文字生成限制

AI生成图容易出现乱码。

若模型无法稳定生成中文，应优先生成无文字插画底图，由后期排版加入：

- 卡名
- 星级
- 技能文本

若生成带文字卡面，必须满足：

- 卡名区域足够大
- 最长卡名“金鼻白毛老鼠精”能完整显示
- 技能区只放简短效果文本
- 不出现攻击力、防守力、阵营、技能名标签

# 通用禁止项

所有生成图禁止：

- 攻击力
- 防守力
- 攻3
- 守3
- 阵营区
- 妖怪阵营标签
- 登场技
- 出场技
- 主动技
- 被动技
- 技能类型括号
- 现代枪械
- 机甲装甲
- 赛博朋克UI
- Q版幼态
- 血腥 gore
- 文字乱码作为主体
- 与规则无关的技能图标
- 过度复杂导致卡面不可读
- 未预留3mm出血
- 四边花纹留白或断边
