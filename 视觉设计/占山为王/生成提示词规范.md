# 通用提示词结构

推荐结构：

1. 卡牌类型
2. 角色 / 法宝 / 地盘名称
3. 星级或类型定位
4. 来源/归属
5. 主要视觉主体
6. 背景场景
7. 色彩与光效
8. 卡牌风格
9. 禁止项

# 通用禁止项

所有生成图禁止：

- 现代枪械
- 机甲装甲
- 赛博朋克UI
- Q版幼态
- 血腥 gore
- 文字乱码作为主体
- 与规则无关的技能图标
- 过度复杂导致卡面不可读

---

# 输出要求

生成图应优先满足：

1. 主体清晰
2. 卡牌类型明确
3. 星级或法宝等级明确
4. 适合裁切进竖版卡面
5. 适合实体卡印刷与网页展示
