# 技能文案原则

来源文件：`游戏/卡牌游戏/占山为王/V1.2-卡面技能文案.md`

卡面文案要求：

- 一句话
- 原则上一行显示
- 统一使用句号结尾
- 不显示技能名
- 不显示“登场技”“主动技”“被动技”
- 数字统一使用阿拉伯数字，例如 `1只`、`1星`、`1块`
- 去向必须清楚，例如 `加入你的地盘`、`加入你的手牌`、`弃掉`、`吹回`
- 字少时不改变技能栏 UI，不放大成标题
- 字长时优先压缩文案，不单独改变 3 星首领卡 UI
- 保留 TRPG 代入感

示例：

| 错误写法 | 正确写法 |
|---|---|
| 获得1个守军 | 偷取1只守军，加入你的地盘。 |
| 控制目标单位 | 绑走1只守军，加入你的手牌。 |
| 发动一次攻击 | 立即发动1次抢地盘。 |

# 卡牌 UI 规格

## 妖怪卡

方向：竖版。

尺寸：63 × 88 mm。

基础牌星级显示：

- 1星小妖：只有星级、名字、插画，无技能栏
- 2星精英：只有星级、名字、插画，无技能栏
- 3星首领：显示星级、右侧竖排名字、插画、底部技能文案

3星基础模板特点：

- 左上角三星
- 红金 / 暗金边框
- 右侧圆角竖排名字牌
- 底部短技能铭牌
- 不显示“首领”文字
- 不显示“技能”文字
- 不显示技能图标
- 不显示无规则意义的右上角圆形徽章
- 视觉强度控制在基础 3星，不提前使用 4星、5星的妖王级特效

## 3星首领技能栏适配

3星首领统一使用当前底部技能铭牌。

技能文案长短不改变 UI 结构。

- 短文案：标准字号，居中，不大幅放大
- 常规文案：标准字号，居中
- 较长文案：优先压缩规则文本，必要时小幅缩字
- 超长文案：回到规则层重写，不单独加高技能栏

## 3星首领名字牌适配

3星首领统一使用右侧竖排名字牌。

名字长短不改变名字牌结构。

- 2至3字：字号略大，竖排居中
- 4至6字：标准字号
- 7至8字：字号略缩小，字距略收紧
- 9字以上：优先改短名，不建议硬塞
- 不因短名字缩短名字牌
- 不因长名字改横排
- 不添加“首领”“妖王”“精英”等身份标签

## 法宝卡

方向：竖版。

尺寸：63 × 88 mm。

特点：

- 与妖怪卡同背景体系
- 紫金神兵边框
- 法阵元素
- 法宝主体占 70% 至 80%

## 地盘卡

方向：横版。

尺寸：88 × 63 mm。

特点：

- 独立模板
- 独立背景体系
- 地图感
- 场景感
- 不显示技能区
- 不显示星级
