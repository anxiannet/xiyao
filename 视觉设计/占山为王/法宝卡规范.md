## 文件定位

本文件定义《占山为王》法宝卡的视觉规范。

法宝名称、数量、效果以：

`游戏/卡牌游戏/占山为王/V1.2-基础卡表.md`

为唯一来源。

卡面版式、印刷出血、技能文本展示方式以：

`视觉设计/占山为王/卡牌框架规范.md`

为统一来源。

---

# 法宝视觉定位

法宝是规则级压制物。

视觉等级应高于普通妖怪卡。

法宝卡必须让玩家感受到：

- 唯一性
- 神器感
- 强规则影响
- 欢乐聚会卡牌的快速阅读
- 与妖怪卡明显不同

---

# 卡面展示原则

法宝卡保留：

- 法宝名称
- 法宝类型徽章
- 法宝主体插画
- 简洁效果文本

法宝卡不展示：

- 星级
- 战力
- 攻击力
- 防守力
- 阵营区
- 技能类型标签

法宝效果文本直接显示效果，不额外写“法宝技”“主动技”等标签。

---

# 统一边框

法宝统一使用：

- 紫金边框
- 神兵纹样
- 法力光晕
- 中央器物构图
- 四周连续花纹

花纹与背景必须延伸至3mm出血区。

---

# 当前法宝

## 紫金红葫芦

核心功能：收。

视觉方向：

- 葫芦居中
- 紫红金光
- 吸纳漩涡
- 被收走的妖影可作为背景剪影

技能文本应简洁呈现，例如：

```text
选择1块地盘，收走其中1只守军。
```

## 金刚琢

核心功能：破。

视觉方向：

- 金圈悬空
- 法宝碎光
- 吸收或压制其他法器
- 清晰表现“取消法宝效果”

技能文本应简洁呈现，例如：

```text
取消1张法宝效果。
```

## 幌金绳

核心功能：抓。

视觉方向：

- 金绳盘旋
- 缠绕妖影
- 拉拽感
- 不画成普通麻绳

技能文本应简洁呈现，例如：

```text
获得1只守军，将其加入手牌。
```

## 芭蕉扇

核心功能：吹。

视觉方向：

- 巨扇压场
- 狂风横扫
- 地盘守军被吹散的剪影
- 火焰山或风云背景

技能文本应简洁呈现，例如：

```text
吹回1块地盘的所有守军，该地盘变为无主。
```

---

# 法宝强度表现

法宝可以比3星首领更有视觉压迫感。

但必须保持卡面可读：

- 法宝主体清楚
- 效果方向清楚
- 技能文本清楚
- 名称清楚

---

# 印刷与边框

法宝卡必须预留3mm出血。

四边必须填满连续花纹。

裁切后不得出现白边、断边或空边。

---

# 禁止项

禁止：

- 把法宝画成普通装备道具
- 把法宝卡做成角色卡
- 法宝主体不清晰
- 只画光效不画法宝
- 让法宝和妖怪卡边框混淆
- 显示攻击力、防守力、攻3、守3
- 显示阵营区
- 显示“法宝技”“主动技”“被动技”等技能标签
- 花纹没有延伸到出血区
