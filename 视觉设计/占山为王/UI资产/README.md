# 《夕妖：占山为王》UI资产说明

## 文件定位

本目录只存放《夕妖：占山为王》可复用卡牌 UI 资产。

只包含：

- 卡框
- 星级
- 名称牌匾
- 技能卷轴 / 铭牌
- 类型徽记
- 地盘阵营徽记
- 卡背
- 通用角饰与花纹条

不包含：

- 妖怪插画
- 法宝插画
- 地盘插画
- 完整正面卡牌
- 妖怪名称
- 技能文字
- 地盘名称
- 攻击力 / 防守力
- 编号
- 水印

## 后续合成顺序

正面卡牌合成顺序：

1. 插画底图
2. 卡框
3. 星级或类型徽记
4. 名称牌匾
5. 技能卷轴或铭牌
6. 文字

卡背不参与正面合成，单独作为背面成品使用。

---

## 生成要求

生成 UI 资产时必须使用大模型图像生成或专业设计工具导出，不允许使用本地脚本生成几何占位图冒充正式资产。

除 `back/` 下卡背成品外，其余 UI 叠加资产必须为透明背景 PNG。
