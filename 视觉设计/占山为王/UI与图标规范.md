# 《夕妖：占山为王》UI与图标规范

## 文件定位

本文件定义《占山为王》卡牌与网页界面的图标、徽章、数字与状态视觉规范。

本文件不定义程序实现。

---

# UI目标

UI必须服务：

- 快速识别卡牌类型
- 快速识别星级与战力
- 快速识别地盘归属
- 快速识别无主 / 已占领状态
- 快速识别法宝效果

---

# 核心图标

必须建立以下图标组：

| 图标 | 用途 |
|---|---|
| 1星 | 小妖星级 |
| 2星 | 精英星级 |
| 3星 | 首领星级 |
| 战力数字 | 妖怪战力 |
| 妖怪 | 妖怪卡类型 |
| 法宝 | 法宝卡类型 |
| 地盘 | 地盘卡类型 |
| 无主 | 无主地盘状态 |
| 已占领 | 已占领地盘状态 |
| 攻山 | 攻击行动 |
| 招兵 | 出牌驻守行动 |

---

# 星级图标

星级图标应保持统一形状，只通过颜色与数量区分：

- 1星：青铜
- 2星：银铁
- 3星：暗金

禁止每张卡使用不同星级图案。

---

# 战力数字

战力数字必须醒目。

建议位置：卡牌右上角。

数字风格：

- 厚重
- 清晰
- 有轻微金属质感

禁止把战力数字放在复杂背景上导致不可读。

---

# 阵营徽章

四大地盘阵营需要独立徽章：

| 阵营 | 徽章方向 |
|---|---|
| 白骨 | 骨纹 / 鬼火 |
| 火云 | 火焰 / 云纹 |
| 狮驼 | 狮首 / 妖城门 |
| 盘丝 | 蛛网 / 毒纹 |

阵营徽章只用于地盘阵营识别，不代表妖怪只能驻守该阵营。

---

# 法宝功能图标

当前法宝功能：

| 功能 | 图标方向 |
|---|---|
| 收 | 吸纳漩涡 |
| 破 | 裂环 / 破法 |
| 抓 | 缠绕绳索 |
| 吹 | 风旋 / 扇形气流 |

---

# 状态图标

当前基础版只需要：

- 无主
- 已占领
- 出兵地
- 目标地盘

暂不设计复杂持续状态。

---

# 禁止项

禁止：

- 图标风格混用现代科技UI
- 每张卡自定义一套星级图标
- 图标过小导致手机端不可读
- 阵营徽章暗示不存在的阵营加成
- 添加规则中不存在的状态图标
