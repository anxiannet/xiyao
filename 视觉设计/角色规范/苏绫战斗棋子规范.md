## 一、战斗棋子定位

苏绫的战斗棋子不是展示立绘，也不是宣传手办。

战斗棋子第一目标：

```text
战场识别
↓
阵营辨识
↓
技能辨识
↓
角色气质
```

苏绫在战斗地图中的识别定位：

```text
青丘使团队长
边境引路人
狐步使用者
黄昏裂隙感知者
风险预判者
带队寻找安全路线的人
```

视觉气质：

```text
沉静
可靠
克制
轻健
边境风尘感
黄昏旧路感
```

禁止方向：

```text
妖艳
卖萌兽耳
宫廷女官
高位祭司
火焰女帝
火焰法师
冰蓝法师
绿色鬼火妖怪
重甲法师
大型兵器角色
```

## 二、基础规格

战斗棋子统一执行：

```text
Camera Type：Orthographic（正交投影）
Pitch：45°
Yaw：按Dir方向旋转
角色高度：200px
源图背景：纯绿幕 #00FF00
源图不含地台
源图不含底座
源图不含圆盘
源图不含石台
源图不含展示座
源图不含地面
源图不含场景
源图不含地面投影
源图不含外部阴影
源图不含UI
源图不含文字
```

说明：

- 200px高度用于手机战棋地图识别。
- “无地台”不是“无地台外阴影”。
- “无地台”表示整个地台、底座、圆盘、石台、展示座都不存在。
- 角色脚下不得生成任何承托物。
- 战斗资源不得使用展示用15°~25°手办镜头。

错误示例：

```text
角色站在圆形底座上
角色站在发光法阵上
角色站在石台上
角色站在草地或六边形格上
角色脚下带展示地面
```

正确结构：

```text
绿幕背景
+
苏绫角色本体
+
随身道具
+
必要的低强度道具微光
```

## 三、三面图生产方案

苏绫战斗棋子采用：

```text
3张基础源图 + 镜像覆盖6方向
```

基础源图：

| 源图 | 用途 | 说明 |
|---|---|---|
| source_front_right | 侧前 | 能看到脸、狐灯、引路杖、斗篷轮廓 |
| source_right | 侧面 | 强化移动感、旅者感、狐灯悬挂关系 |
| source_back_right | 侧后 | 能看到长发、旧旅行斗篷、备用绳索、狐灯背侧轮廓 |

镜像派生：

```text
source_front_left = source_front_right 水平镜像
source_left = source_right 水平镜像
source_back_left = source_back_right 水平镜像
```

禁止：

- 使用正前 / 侧面 / 正后三视图作为战斗方向资源。
- 使用8方向图作为六边形地图默认资源。
- 每个方向单独重绘导致道具位置、服装结构不一致。

## 四、轮廓识别

苏绫在200px高度下必须依靠轮廓识别。

识别优先级：

```text
旧旅行斗篷
↓
引路狐灯
↓
偏长狐耳
↓
引路杖
↓
备用绳索 / 地图袋
↓
左眼角浅伤痕
```

### 旧旅行斗篷

- 形成苏绫的主要外轮廓。
- 青灰白色、风沙灰、旧布白方向。
- 边缘有轻微磨损。
- 下摆有行路感，不拖地，不遮挡脚部移动方向。
- 不做宫廷披帛，不做华丽法袍。

### 偏长狐耳

- 作为头部最高识别点。
- 自然、克制，不做可爱兽耳卖点。
- 三个基础方向中均可辨识。

### 引路狐灯

- 苏绫最重要的战场识别道具之一。
- 体积小而亮，内部使用黄昏色狐火。
- 光色以夕照红、琥珀橙、暮金为主，可有极少暮紫边缘。
- 放在身体一侧，形成清晰侧轮廓。
- 不做大灯笼，不遮挡身体主体。
- 不使用冰蓝、鬼火绿或科技青作为默认灯光。

### 引路杖

- 本质是旅行杖 / 探路杖，不是法杖。
- 可以支撑行路、标记方向、点地指路。
- 禁止做成法师权杖、长枪、巨杖或重型武器。

### 备用绳索与地图袋

- 腰部或侧身挂载。
- 作为边境引路人与生存习惯的视觉记忆点。
- 体积不能大到影响主轮廓。

## 五、服装与配色

苏绫战斗棋子配色必须服从：

```text
视觉设计/小队视觉/青丘使团视觉规范.md
视觉设计/特效规范/狐火与夕子视觉规范.md
```

不再使用“狐火青 / 狐火蓝”作为默认狐火方向。

战斗棋子执行色：

| 部位 | 色彩 / 材质方向 | 说明 |
|---|---|---|
| 斗篷 | 旧布白、风沙灰、青灰白 | 主轮廓，体现旧旅行斗篷 |
| 内层旅装 | 低饱和夜青、旧布白、风沙灰 | 轻便，便于移动 |
| 狐灯 | 夕照红、琥珀橙、暮金，少量暮紫 | 黄昏残痕，不是青蓝鬼火 |
| 装备件 | 旧木、皮革、绳索 | 引路人与旅者感 |
| 金属点缀 | 暮金、旧铜 | 少量，不可宫廷化 |

禁止：

- 大面积红金宫廷化。
- 高饱和爆炎橙红。
- 冰蓝法力光。
- 绿色鬼火光。
- 科技青光。
- 纯黑暗系。
- 复杂宫廷饰品堆砌。

## 六、黄昏视觉语言

苏绫不是火焰角色。

苏绫的力量视觉链路是：

```text
夕子
↓
狐火残痕
↓
黄昏裂隙感知
↓
引路
```

因此视觉重点是：

```text
方向
路径
余晖
残痕
黄昏交界
```

而不是：

```text
燃烧
爆炸
火球
火墙
法术阵
```

苏绫的狐火应像：

```text
夕阳余烬
落日残光
风中残痕
路径微光
```

## 七、动作设计

### 待机动作

推荐：

```text
身体微侧
一手持引路杖
一侧悬浮或提起引路狐灯
重心轻健
斗篷自然展开
视线像在判断前方路线
```

动作语义：

```text
随时准备带队继续前进
```

禁止：

- 正面站桩摆拍。
- 双手张开大法师施法。
- 夸张攻击姿势。
- 过度柔媚或卖萌姿势。

### 动态细节

允许轻量待机动效：

- 狐灯轻微摆动。
- 狐灯内部夕照色微光呼吸。
- 苏绫轻触狐耳尖，表现思考方向。

注意：

- 足下安全路线、选中圈、状态圈由程序层表现。
- 源图不得把足下路线画成地台或地面。

禁止：

- 满屏粒子。
- 大面积火焰。
- 爆炸式位移特效。
- 遮挡棋盘路线的光带。

## 八、狐火与夕子表现

苏绫的狐火不是爆炸火焰，也不是青蓝鬼火。

战斗棋子源图中狐火只允许表现为：

```text
引路狐灯内部夕照色微光
低强度黄昏余烬粒子
角色道具边缘极轻微暮金残光
```

以下内容由程序特效层表现，不进入角色源图：

```text
足下安全路线
裂隙感知路线
状态圈
选中框
地块状态
```

视觉关键词：

```text
路径
引路
残痕
余晖
流动
克制
```

禁止：

```text
青色狐火
蓝色狐火
绿色鬼火
火球
火柱
火焰翅膀
大面积红色爆炎
战斗法师式爆炎
脚下发光地台
脚下法阵
```

## 九、底座与地台规则

苏绫战斗棋子源图禁止包含任何地台或底座。

禁止包含：

```text
地台
底座
圆盘
石台
展示座
发光法阵
地面
草地
六边形格
场景碎片
地面投影
外部阴影
```

游戏内如需底座、站位圈、选中圈或状态圈，必须由程序或统一资源层挂载。

程序底座参考：

```text
底座直径：约80px
```

程序层可使用青丘风格底座皮肤，但不得进入苏绫角色源图。

推荐程序底座视觉：

```text
旧路石纹
裂隙路线纹
暮金微光
夕照色狐火残痕
小型路标石
风蚀纹
```

禁止：

- 火焰女帝式大火地台。
- 宫殿法阵式地台。
- 墓地、十字架、暗黑废墟。
- 底座占用相邻格。
- 将展示底座画进角色源图。

## 十、200px可读性检查

苏绫战斗棋子必须通过以下检查：

| 显示高度 | 必须识别 |
|---:|---|
| 200px | 苏绫、狐耳、狐灯、斗篷、引路杖 |
| 100px | 青丘单位、引路/辅助定位、狐灯轮廓 |
| 50px | 青丘小队气质与主轮廓 |

若狐灯、斗篷、狐耳在100px下不可读，则退回修改。

如果源图出现底座、地台、圆盘或地面，则无论可读性如何，直接退回。

如果狐火默认画成青蓝鬼火、绿色鬼火、冰蓝法术光，也直接退回。
