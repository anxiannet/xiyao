# 夕妖（XIYAO）

> 黄昏不是结束，黎明也不是答案。
> 当昼夜的秩序开始松动，被称为“妖”的人，才终于看见世界的真相。

《夕妖》是一款原创东方神话战棋项目。

当前开发目标是 MVP：

- 3 人小队战棋
- 15 分钟一局
- 单人对 AI 对弈
- 两支初始小队：青丘使团、天门执法队
- 小型六角格地图
- 规则轻量、可快速验证、可长期扩展

# 一句话设定

世界并非只有天、人、妖三界。

真正支撑世界运转的是四种本源：

- 昼曜：白昼、秩序、规则、神权
- 夜魄：黑夜、混沌、欲望、本能
- 晨辉：黎明、新生、修复、重建
- 夕子：黄昏、终结、变化、自由

天门宣称夕子是不祥之力，将拥有夕火的人统称为“夕妖”。

但真相是：

> 夕妖未必是妖。
> 神也未必是善。
