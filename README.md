# 一句话设定

世界并非只有天、人、妖三界。

真正支撑世界运转的是四种本源：

- 昼曜：白昼、秩序、规则、神权
- 夜魄：黑夜、混沌、欲望、本能
- 晨辉：黎明、新生、修复、重建
- 夕子：黄昏、终结、变化、自由

天门宣称夕子是不祥之力，将拥有夕火的人统称为“夕妖”。

但真相是：

> 夕妖未必是妖。
> 神也未必是善。
