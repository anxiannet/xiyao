# 夕妖（XIYAO）世界观圣经 V1

## 核心概念
夕妖是一款东方神话战棋游戏。

世界由天界、人界、妖界构成。
三界共同依赖一种名为“夕火”的本源力量维持平衡。

当太阳落下、月亮升起的短暂时刻，被称为“夕时”。
在夕时诞生或受到夕火眷顾的生灵，被称为“夕妖”。

## 历史
### 太初纪
夕火诞生，三界形成。

### 神妖纪
神族与妖族共同统治世界。

### 镇妖纪
三千年前爆发镇妖之战。
妖族战败，暮渊封印形成。

### 余烬纪（当前时代）
夕火衰弱。
封印松动。
三界重新争夺夕火本源。

## 势力
### 天巡司
天界执法力量。

### 镇妖军
人界精锐军团。

### 青丘狐族
幻术与情报大师。

### 苍狼部族
崇尚力量与狩猎。

### 暮渊遗民
镇妖之战幸存者。

### 星宫
观测天命与未来。

## 核心资源
夕火：用于觉醒、施法与胜利。

## 游戏核心体验
3人小队战棋。
15分钟一局。
占点、任务、击败敌人获得夕火。
角色通过化形成长。
