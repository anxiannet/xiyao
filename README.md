# 夕妖（XIYAO）

> 黄昏不是结束，黎明也不是答案。
> 当昼夜的秩序开始松动，被称为“妖”的人，才终于看见世界的真相。

《夕妖》是一款原创东方神话战棋项目。

---

# 当前 MVP

当前 MVP 只验证可玩闭环，不追求完整商业版本。

MVP 目标：

- 验证核心战棋规则是否成立。
- 验证两支初始小队是否有清晰差异。
- 验证玩家 vs AI 是否可以完整跑通。
- 验证 AI vs AI 是否可以用于平衡测试。

MVP 模式：

- 自由对战。
- 玩家 vs AI。
- AI vs AI。

MVP 内容：

- 3 人小队战棋。
- 15 分钟一局。
- 两支初始小队：青丘使团、天门执法队。
- 每名单位只保留 1 个核心技能。
- 小型六角格地图。
- 7 个基础地形。
- 状态系统：单位状态、地块状态、小队状态、全局状态。
- 地块状态：狐火残留、勘验区。
- 资源：气运、狐火、AP。
- 律令系统。
- 战斗骰子系统。
- 占点系统。

MVP 不包含：

- 单人战役。
- 教学关卡。
- 剧情推进。
- 解锁系统。
- 奖励系统。
- 成长系统。
- 经济系统。

# 一句话设定

世界并非只有天、人、妖三界。

真正支撑世界运转的是四种本源：

- 昼曜：白昼、秩序、规则、神权
- 夜魄：黑夜、混沌、欲望、本能
- 晨辉：黎明、新生、修复、重建
- 夕子：黄昏、终结、变化、自由

天门宣称夕子是不祥之力，将拥有夕火的人统称为“夕妖”。

但真相是：

> 夕妖未必是妖。
> 神也未必是善。

# 文档边界

- `世界观/` 存放世界观总设定与长期世界设定。
- `小说/` 存放小说大纲、人物小传与章节正文。
- `游戏/游戏设定/` 存放当前 MVP 游戏规则、地形、小队、单位、技能、地图与审计文档。
- `游戏/程序代码/` 存放前端、后端、共享代码与开发文档。
- `游戏/数据库/` 存放表结构说明、SQL 脚本与初始化数据。
- `游戏/测试/` 存放规则测试、AI 测试、地图测试与平衡测试。
- `归档/` 存放废弃内容、历史版本与 MVP 范围外内容。

# MVP 裁剪原则

当前版本只验证可玩闭环。

百宝锦囊、宝物、妙计、阵型、法术、化形、天门威仪、命运系统等未来扩展，统一归档到 `归档/mvp_out_of_scope/`，不作为 MVP 规则来源。
