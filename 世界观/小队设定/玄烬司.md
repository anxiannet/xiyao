# 玄烬司

玄烬司是狐族内部少数仍保留禁术传承的小队。

他们不属于青丘使团，也不沿用青丘使团的使节体系。青丘使团擅长机动、幻术与边缘路线，玄烬司则专注于蓝焰禁术、区域压制与持续侵蚀。

本文件只维护玄烬司的小队世界观设定，不维护单个角色详细人物志，不维护视觉执行规则，不维护战斗数值。

---

## 设定定位

玄烬司是一支以狐族法师为核心的禁术小队。

他们不追求瞬间击杀，而是通过蚀魂蓝焰长期压迫敌人的行动空间，使敌人在持续消耗中失去反击能力。

核心关键词：

```text
狐族禁术
蓝焰
持续伤害
侵蚀
区域压制
诅咒扩散
减疗
消耗战
边界守护
不被信任的守护者
```

---

## 小队气质

玄烬司的气质不是热血、华丽或正统英雄感。

他们更接近狐族体系中被隐藏起来的一面。

小队气质：

```text
神秘
克制
压迫
危险
冷静
禁忌
长期消耗
被需要但不被完全信任
```

玄烬司成员应给人一种“站在边界上守夜”的感觉。

他们不是主动展示力量的人，而是必须在灾厄到来前维持封印、控制蔓延、承受代价的人。

---

## 世界观背景

狐族常见狐火多用于迷惑、引路、幻化与扰乱感知。

但玄烬司掌握的蓝焰不同。

这种火焰传说来自界潮深处，被称为“蚀魂蓝焰”。

它不会立刻焚毁肉身，而是持续灼烧灵力、意志与生命气息。

正因如此，许多妖族将其视为禁术。

玄烬司因此长期处于狐族主流秩序之外。

他们守护狐族边界，却也被视为可能带来灾厄的异端。

---

## 核心术式

### 蚀魂蓝焰

蚀魂蓝焰是玄烬司的核心术式。

它的危险之处不在于一瞬间的破坏力，而在于持续附着和逐步侵蚀。

被蓝焰缠上的目标，初期只会感到灵力流动迟滞。

随着蓝焰持续燃烧，目标的恢复、护体与意志都会受到影响。

### 蓝焰印记

玄烬司的术式会在目标身上留下蓝焰印记。

印记并非普通伤口，而是一种附着在灵息上的燃烧痕迹。

当印记持续累积，蓝焰会从单点灼烧转为范围蔓延。

### 术式领域

玄烬司擅长把战场变成缓慢收缩的术式领域。

他们通过蓝焰、诅咒、束缚与灵力稳定，将敌人限制在持续侵蚀范围内。

这个领域不应表现为明亮圣域，而应表现为幽暗、冷色、压低亮度的禁术空间。

以下内容为玄烬司视觉设计的世界观来源。


### 色彩来源

```text
主色意象：深蓝黑
识别色意象：幽蓝
辅助色意象：银灰、冷白、紫蓝
禁用色意象：红焰、橙焰、圣金强光
```

蓝色在玄烬司中不是热烈火光，而是冷、静、深、持续侵蚀的禁术光。

### 材质来源

```text
黑曜石
旧银
蓝晶
暗色布料
符纸
狐毛
冷色灯火
封印石
```

材质应体现“禁术组织”与“边界守护者”，避免华丽宫廷感、现代科技感与机械感。

### 图形语言

```text
狐尾弧线
灯笼轮廓
铃形轮廓
符咒竖线
封印圆环
细长火舌
棘刺状束缚线
```

图形应偏收束、细长、低亮度，不应大面积爆发。

### 建筑与场景意象

```text
封印室
禁术祭坛
黑曜石石阶
藏书阁
界潮边界
冷雾石门
蓝焰灯廊
```

这些场景只作为玄烬司相关视觉背景来源，不应直接进入战斗棋子源图。

### 装备与法器意象

```text
灯类法器
咒符
灵铃
束缚咒链
祭司法带

```

装备以法术媒介为主，不以刀剑、枪械、机甲或现代装备为主。

## 小队视觉共性

玄烬司成员应共享以下共性：

```text
狐族特征
黑蓝体系
幽蓝禁术光
低饱和冷色气质
符咒或法器识别
压抑、克制、危险的姿态
```

但每名成员必须拥有独立视觉锚点：

```text
沧离：烬魂灯
灰瞳：灰蓝眼瞳与咒符
夜棘：影棘与束缚咒链
玄铃：玄铃法器与祭司法带
```

## 与青丘使团的关系

玄烬司与青丘使团同属狐族体系，但路线完全不同。

青丘使团代表狐族的灵巧、幻术、外交与机动性。

玄烬司代表狐族被隐藏起来的一面：禁术、牺牲、长期消耗与危险边界。

二者并非上下级关系。

视觉上，玄烬司不应沿用青丘使团的轻盈外交感，而应更冷、更暗、更具禁术压迫感。
