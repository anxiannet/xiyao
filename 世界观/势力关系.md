# 势力关系

本文档用于补充《夕妖》世界观中的势力关系。

本文件只维护世界观层面的立场、矛盾、合作边界与叙事关系，不维护游戏规则、数值、技能、棋子属性或美术规范。

---

# 1. 使用边界

## 1.1 唯一来源

- 世界观正文以 `世界观/` 为准。
- 人物事实以 `世界观/人物志/` 为准。
- 当前 MVP 小队状态以 `游戏/游戏设定/当前状态.md` 为准。
- 视觉方向以 `视觉设计/` 为准。

## 1.2 本文档用途

本文档解决三个问题：

1. 各势力为什么冲突。
2. 各势力之间是否有合作空间。
3. 机关城、涂山等扩展势力如何接入现有世界观。

本文档不把机关城、涂山写入当前 MVP 规则小队。它们属于世界观扩展势力与未来小队来源。

# 2. 核心冲突轴

《夕妖》的势力关系不是简单的神、妖、人三方对立，而是围绕四象本源与历史叙事展开。

## 2.1 秩序与变化

- 天门、天巡司、部分人界王朝偏向秩序。
- 青丘、暮渊遗民、部分流亡夕妖偏向变化。
- 机关城追求可计算、可控制的技术秩序。
- 涂山追求在秩序缝隙中的权力、交易与自保。

## 2.2 官方历史与被抹除的旧史

天门叙事将三千年前的镇妖之战描述为平定妖乱。

青丘、暮渊遗民、部分星宫记录者认为镇妖之战本质上是对夕火种、暮君遗力和旧妖族历史的清洗。

## 2.3 四象失衡

- 昼曜失控会走向永昼与绝对审判。
- 晨辉失控会走向强制净化与记忆重置。
- 夜魄失控会走向梦魇实体化与侵蚀。
- 夕火失控会走向形态崩坏与无序变化。

各势力并不天然代表善恶，而是在失衡时代选择不同的生存方式。

# 4. 天门系关系

## 4.1 天门

天门是维持天界秩序、执行律令与审判的统治体系。

天门的公开叙事是维护三界安定，实际核心目标是稳定天命结构，阻止夕火复燃导致旧秩序崩塌。

天门内部并非完全铁板一块：

- 正统派相信所有夕火异象都必须封禁。
- 执法派强调程序、证据与律令。
- 神殿派倾向借晨辉净化异常。
- 少数异见者开始怀疑镇妖之战的历史记录。

## 4.2 天巡司

天巡司是天门的执法机关。

天巡司与天门执法队关系密切，但两者不完全等同：

- 天巡司是组织。
- 天门执法队是可派出的精英执行小队。

天巡司对青丘、暮渊遗民和流亡夕妖采取高压策略，对机关城则更倾向征召、监管和技术备案。

## 4.3 镇妖军

镇妖军是人界九州的镇妖武装。

镇妖军接受天门合法性背书，但其士兵与将领多来自人界，因此更容易与普通百姓、边境村落、受灾城镇产生真实联系。

镇妖军的叙事张力在于：

- 他们以为自己在保护人间。
- 他们逐渐发现某些被追捕者并不是妖乱源头。
- 当军令与良知冲突时，镇妖军可能分裂。

# 5. 妖族与夕火系关系

## 5.1 青丘

青丘是古老狐族势力。

青丘不等于夕妖，但青丘中有许多成员能感知夕子，并将夕火技巧化为狐火、狐印、假身与迷踪等表现。

青丘与天门的核心矛盾不是单纯的神妖仇恨，而是：

- 天门要没收暮君残片。
- 青丘认为残片记录着被抹去的旧史。
- 天门要定义青丘为潜在妖乱源。
- 青丘拒绝接受这种命名。

## 5.2 苍狼部族

苍狼部族位于北境荒原与雪原地带。

他们对天门和人界边军长期不信任。相比青丘的情报、幻术和旧路，苍狼更强调血脉、猎场、部族战斗与失地记忆。

苍狼可与青丘、暮渊遗民短期合作，但并不天然接受青丘的谨慎路线。

## 5.3 暮渊遗民

暮渊遗民是镇妖之战幸存者后裔。

他们与青丘都关注旧史，但方式不同：

- 青丘更重视护送、保存和解读。
- 暮渊遗民更重视揭露、释放和复仇。

暮渊遗民是最容易把夕火推向暴走的一方。

# 6. 中立与技术系关系

## 6.1 星宫

星宫观测天命，但并不完全服从天门。

星宫知道“未来早已写下，但可以被读错”。这使他们天然怀疑唯一叙事，也因此容易被天门视为危险旁观者。

星宫与机关城存在知识合作空间：

- 星宫提供星轨、天象、命运偏差记录。
- 机关城提供仪器、测绘、灵能装置与保存技术。

## 6.2 机关城

机关城不是现代科幻城市，也不是纯粹军工势力。

它是以灵能、机关、创造、技术秩序为核心的自治城邦。

机关城与天门的关系复杂：

- 天门需要机关城的工造、测绘、封印器具和灵能装置。
- 机关城需要天门承认其工造合法性与商路安全。
- 但机关城反对天门垄断知识、禁毁旧史和强征工匠。

机关城与青丘并非天然同盟。青丘不信任可记录、可追踪、可归档的机关术；机关城则认为青丘过度依赖直觉、旧路和隐秘经验。

但在黄昏裂隙测绘、暮君残片保存、旧史复原等问题上，双方有合作空间。

---

# 7. 涂山关系

涂山是妖族贵族、夜宴组织与幻术权谋势力。

涂山与青丘同属狐族相关文化圈，但不是同一政治实体。

## 7.1 涂山与青丘

两者关系接近“同源异路”。

- 青丘重视旧路、生存、护送和真相。
- 涂山重视宴席、盟约、交易和权势平衡。

青丘认为涂山太会妥协。

涂山认为青丘太容易把自己推到明处。

双方可以互相帮助，也会互相试探。

## 7.2 涂山与天门

涂山不会公开反天门。

涂山擅长通过贡礼、宴请、婚盟、情报交换和替罪边界维持自身安全。

天门不信任涂山，但也不愿轻易撕破脸，因为涂山掌握大量妖族贵族、人界商路与夜宴密约。

## 7.3 涂山与暮渊遗民

涂山同情旧史被抹除，但不轻易支持暮渊遗民的激烈路线。

对涂山而言，真相本身不是目的。真相能否成为筹码，才决定它是否值得公开。

# 8. 关系张力写作原则

1. 不把天门写成绝对正义，也不写成纯粹邪恶。
2. 不把青丘写成单纯受害者或单纯反派。
3. 不把机关城写成现代科技阵营，它必须保留东方机关与灵能幻想质感。
4. 不把涂山写成低俗魅惑势力，它的核心是夜宴、权谋、贵族礼法与危险优雅。
5. 不让任何势力直接掌握完整真相。
6. 各势力之间可以短期合作，但合作必须有代价、条件与互不信任。
7. 势力关系服务世界观与剧情，不替代规则文档。
