# 沧离

## 身份

| 项目 | 内容 |
|---|---|
| 姓名 | 沧离 |
| 种族 | 狐族 |
| 性别 | 男 |
| 年龄 | 12岁 |
| 阵营 | 玄烬司 |
| 身份 | 玄烬司队长、首席焚咒师 |
| 战斗定位 | 持续伤害法师 |
| 核心能力 | 蓝焰、侵蚀、持续伤害、区域压制 |
| 武器 / 法器 | 烬魂灯 |

## 性格

沧离性格冷静、克制、理智。

相比同龄人，他极少表现出情绪波动。

长期与蚀魂蓝焰共存，使他比同龄人更加早熟。

他并不热衷权力，却愿意承担责任。

## 外貌

十二岁狐族少年。

银白短发。

深蓝狐耳。

肤色偏白。

身形瘦削。

常佩戴圆框眼镜。

## 装束

黑蓝法袍。

高领、多层结构。

深蓝黑主色。

幽蓝纹路。

腰间悬挂铃饰与法术挂件。

## 武器

### 烬魂灯

灯内封存蚀魂蓝焰。

沧离通过烬魂灯引导蓝焰、施展诅咒、扩散印记以及维持大型术式。

## 行为习惯

说话速度较慢。

回答问题前习惯思考。

阅读术式时会下意识推扶眼镜。

思考时经常观察烬魂灯中的火焰变化。

## 人物称号

蓝焰继承者

玄烬司少主

界潮幸存者

## 人物语录

“火焰并不总意味着毁灭。”

“秩序并非束缚，而是代价。”

## 喜好

研究术式。

整理古籍。

观察蓝焰变化。

## 厌恶

失控的力量。

无意义的牺牲。

## 恐惧

蚀魂蓝焰彻底失控。

玄烬司重蹈上一代覆灭的命运。

## 执念

守住玄烬司传承。

查明自己与界潮的真正联系。
