# 世界观

本目录是《夕妖》世界观设定的唯一来源。

任何世界观正文、势力说明、历史、地理、时间线、术语和长期世界设定总表，都必须以本目录文件为准。

其他目录只允许引用本目录内容，不允许复制或另写一份世界观正文。

---

# 允许

- 世界观
- 势力
- 历史
- 地理
- 时间线
- 术语
- 长期世界设定总表

# 禁止

- 游戏规则
- 技能数值
- 数据库
- 程序实现
- 小说正文

---

# 文件规则

本目录下所有 Markdown 文件均属于世界观来源文件。

README 不维护手写文件清单，避免新增文件后遗漏更新。

---

# 引用规则

游戏设定、小说、程序文档如需使用世界观内容，只能通过链接引用本目录文件。

如果世界观内容需要修改，必须直接修改本目录中的对应文件。
