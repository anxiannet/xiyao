# xiyao-mvp

《夕妖》MVP自由对战前端。

目录：

```text
xiyao-mvp/
├─ src/
├─ docs/
├─ package.json
├─ tsconfig.json
├─ vite.config.ts
└─ index.html
```

用途：

- 玩家 vs AI
- AI vs AI
- 地图调试
- 状态展示
- 规则验证UI

后续新的前端版本请独立建立：

```text
frontend/
├─ xiyao-mvp/
├─ xiyao-admin/
├─ xiyao-tabletop/
├─ xiyao-spectator/
└─ shared-ui/
```
