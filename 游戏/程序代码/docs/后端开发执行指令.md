# 后端开发执行指令

当前阶段暂不开发实时对战后端。

当前阶段不做：

```text
创建对局接口
实时提交动作接口
后端规则引擎
后端AI
后端对局状态推进
```

当前阶段优先级：

```text
前端
前端本地规则引擎
前端本地AI
LocalStorage本地存储
离线运行
```

---

# 后端定位

后端是未来阶段服务。

未来后端只负责：

```text
对战结束后结果上传
对局日志归档
AI统计汇总
排行榜或统计展示
配置数据管理
```

不负责当前阶段的实时对局执行。

---

# 当前禁止

禁止现在实现：

```text
POST /matches
POST /matches/:id/actions
POST /matches/:id/ai-action
POST /matches/:id/auto-run
实时规则结算API
实时AI决策API
```

禁止把规则执行从前端移回后端。

禁止让当前网页版依赖服务器才能运行。

---
