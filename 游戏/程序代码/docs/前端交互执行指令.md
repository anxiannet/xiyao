# 前端交互执行指令

只做前端交互壳，不做规则、不做后端、不做数据库。

读取：当前状态.md、README.md、规则总表.md、地图格子设置规则.md、程序代码/docs/README.md。

目标：制作《夕妖》MVP自由对战前端交互原型。

模式：玩家 vs AI、AI vs AI。

首屏：模式选择、地图选择、队伍选择、开始对战。

对战界面：地图区域、回合信息、行动方、AP、单位信息面板、动作按钮区、日志区。

地图交互：选中单位、移动、攻击、查看据点。

允许使用 mockMaps、mockTiles、mockUnits、mockActions、mockBattleLog。

禁止连接 Supabase、后端、AI逻辑、规则引擎。

输出：页面结构、组件树、mock数据结构、状态流、后端接入点。
