# 前端交互执行指令

当前阶段前端不只是交互壳。

当前阶段前端需要承载：

```text
前端交互
本地规则引擎
本地AI
本地对局状态
本地日志
LocalStorage存储
离线运行
```

---

# 职责

只负责：

- 网页版自由对战
- 前端内置规则引擎执行
- 前端本地 AI 决策
- 前端本地对局状态
- 前端本地日志与回放数据
- LocalStorage 保存对局结果

禁止：

- 连接 Supabase
- 调用后端接口
- 上传服务器
- 设计新规则
- 修改数据库
- 新增玩法

# 目标

制作《夕妖》MVP网页版自由对战。

支持：

```text
玩家 vs AI
AI vs AI
```

要求：

```text
可以离线运行
不依赖后端
不依赖 Supabase
对局数据保存到 LocalStorage
```

---

# 架构要求

前端内部必须包含：

```text
engine/
ai/
storage/
state/
ui/
data/
```

## engine

负责：

```text
合法动作判断
规则执行
伤害结算
状态结算
据点结算
回合推进
胜负判断
```

## ai

负责：

```text
动作生成
动作评分
AI行动选择
AI vs AI 自动运行
```

## storage

负责：

```text
LocalStorage读写
对局快照保存
对局日志保存
AI统计保存
```

## state

负责：

```text
当前对局状态
当前选中单位
当前行动方
当前可行动作
```

## ui

负责：

```text
首屏
战斗界面
地图
单位面板
动作按钮
日志
```

## data

负责：

```text
本地配置数据
测试地图
单位配置
技能配置
状态配置
```
