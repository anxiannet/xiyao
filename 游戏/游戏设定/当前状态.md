# 当前最终律令方案

玄照技能：

```text
颁令
```

消耗：

```text
1 AP
```

限制：

```text
每大回合最多1次
同一时间仅1条律令
```

原则：

```text
律令从颁布时开始生效
不追溯过去行为
只影响颁布后的事件
```

## 禁行令

效果：

```text
敌方进入：
边缘据点
黄昏裂隙

获得迷踪
```

额外：

```text
天门单位位于黄昏裂隙相邻格
支援面自动成功
```

## 戒严令

效果：

```text
天门单位位于中央据点
DEF_DICE +1
```

## 追捕令

效果：

```text
敌方移动后
获得逃逸
```

额外：

```text
天门攻击带逃逸目标
支援面自动成功
```
