## 小队定位

4 人机动小队。

MVP 规则定位：

```text
狐火资源
狐印
狐火残留
假身
狐火充盈
边缘路线
机动拉扯
```

与天门执法队的规则对照：

- 天门：3 人精英，正面推进，依靠律令、勘验区和护阵压缩路线。
- 青丘：4 人机动，绕后拉扯，依靠狐火残留与假身制造路线优势。

## 小队核心能力

## 狐印围猎

苏绫是狐印的唯一稳定来源。

青丘攻击带狐印目标时，支援面自动成功。

## 狐火路线

青丘通过狐火残留和黄昏裂隙收集狐火。

狐火在规则层是小队资源，主要来自路线收益与攻击收益。

## 假身支援

假身是青丘召唤物。

假身阻挡移动，不阻挡视线，可吸收一次攻击或技能，并参与支援。

## 不正面对抗

青丘使团不适合硬打中央据点。

他们通过边缘据点、假身、狐火残留和裂隙逼迫天门分兵。
