# 小队目录

本目录用于收口《夕妖》游戏设定中的小队规则入口。

当前正式 MVP 小队状态以：

```text
游戏/游戏设定/当前状态.md
```

为准。

世界观正文、势力关系、地理与人物事实以：

```text
世界观/
```

为准。

# 维护规则

1. 当前 MVP 小队不得在未更新 `当前状态.md` 前擅自扩容。
2. 扩展小队可以保留世界观定位，但不得在本目录直接补数值。
3. 新小队进入规则设计前，必须先确认世界观来源。
4. 人物背景不得写在本目录，必须写入 `世界观/人物志/`。
5. 视觉方向不得写在本目录，必须写入 `视觉设计/`。
