# 小队目录

本目录用于收口《夕妖》游戏设定中的小队规则入口。

当前游戏实现状态以：

```text
游戏/游戏设定/当前状态.md
```

为准。

世界观中的小队、势力行动组织、地理与人物事实以：

```text
世界观/
```

为准。

# 维护规则

1. 本目录只维护游戏规则入口与索引。
2. 人物背景不得写在本目录，必须写入 `世界观/人物志/`。
3. 势力、小队身份、地理、历史不得写在本目录，必须写入 `世界观/`。
4. 视觉方向不得写在本目录，必须写入 `视觉设计/`。
5. 行动组织进入游戏规则实现时，再在本目录建立对应规则文件。
