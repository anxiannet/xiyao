# 天门执法队

## 小队定位

3人精英小队。

天门执法队代表昼曜、律令、勘验、护阵与正面推进。

在 MVP 中，它是青丘使团的主要对照小队：

- 青丘依靠狐火残留、掩影、裂隙与假身制造路线和目标变化。
- 天门依靠律令、勘验区和护阵压缩敌方行动空间。

---

# 天门执法链

天门执法队并非传统战斗小队。

他们的行动遵循：

律
↓
勘
↓
罚

玄照负责颁布律令。

白烬负责勘验异常区域。

赤霄负责执行与压制。

天门的核心不是消灭敌人。

而是恢复秩序。

# 队长机制：律令

玄照通过核心技能“颁令”主动选择律令与颁布时机。

颁令消耗 1AP。

每个大回合最多颁布 1 次律令。

律令从颁布时开始生效。

律令不追溯已经发生的行为。

当前 MVP 律令：

## 禁行令

本回合严查边界。

生效期间：

敌方单位进入：
- 边缘据点
- 黄昏裂隙

获得迷踪。

天门单位位于黄昏裂隙相邻格时：

支援面自动成功。

## 戒严令

中央区域进入戒严。

生效期间：

天门单位位于中央据点时：

DEF_DICE +1。

## 追捕令

本回合优先追捕逃逸目标。

生效期间：

敌方单位移动后，获得逃逸。

天门单位攻击带逃逸的目标时：

支援面自动成功。

逃逸持续至本大回合结束。
