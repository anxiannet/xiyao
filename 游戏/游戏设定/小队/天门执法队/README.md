## 队长机制：律令

玄照通过核心技能“颁令”主动选择律令与颁布时机。

颁令消耗 1AP。

每个大回合最多颁布 1 次律令。

律令从颁布时开始生效。

律令不追溯已经发生的行为。

当前 MVP 律令：

| 律令 | 规则效果 |
|---|---|
| 禁行令 | 生效期间，敌方进入边缘据点或黄昏裂隙时获得迷踪；天门单位位于黄昏裂隙相邻格时支援面自动成功 |
| 戒严令 | 生效期间，天门单位位于中央据点时 DEF_DICE +1 |
| 追捕令 | 生效期间，敌方单位移动后获得逃逸；天门攻击带逃逸的目标时支援面自动成功 |

## 核心循环

```text
玄照选择颁令时机与律令
↓
白烬让关键地块获得勘验区状态
↓
赤霄护阵、占点、承伤
↓
天门通过规则与站位压缩对方行动空间
```
