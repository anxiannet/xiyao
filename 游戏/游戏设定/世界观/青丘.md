# 青丘

青丘是《夕妖》中的古老狐族势力。

MVP 中，青丘主要通过“青丘使团”呈现。

---

# 核心定位

青丘不是单纯的妖族反派。

它代表：

- 幻术
- 情报
- 伪装
- 换位
- 狐火
- 生存智慧
- 对天门叙事的不信任

青丘擅长在强权规则中寻找缝隙。

---

# 与夕子的关系

青丘并不等于夕妖，但青丘中有许多成员能感知夕子或使用狐火。

狐火是青丘将夕火技巧化、仪式化、战术化后的表现。

在 MVP 中，青丘的夕子相关能力不写成大规模化形系统，而压缩为：

- 狐火资源
- 狐印状态
- 狐火残留覆盖层
- 假身
- 换位
- 迷踪

---

# MVP 战术表现

青丘使团的玩法关键词：

- 机动
- 误导
- 假身
- 标记
- 换位
- 边缘路线
- 快速抢点

青丘不靠正面压制取胜，而靠让敌人“判断失误”。

---

# 写作原则

1. 青丘角色要聪明，但不能全知。
2. 青丘台词可以带试探、反讽、留白。
3. 不把青丘写成纯粹受害者。
4. 不把青丘写成邪恶妖族。
5. 青丘的魅力来自“知道真相，却不轻易说破”。
