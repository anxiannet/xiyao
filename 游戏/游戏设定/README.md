# 文档读取规则

- `当前状态.md` 是当前项目状态入口。
- `规则总表.md` 是当前规则唯一来源。
- 世界观正文以 `世界观/` 内文档为准，本目录只引用世界观。
- 当前玩法规则以 `游戏/游戏设定/` 内文档为准。
- 程序实现以 `游戏/程序代码/` 内文档为准。
- 数据库设计以 `游戏/数据库/` 内文档为准。
- 若开发文档与规则文档冲突，以 `游戏/游戏设定/规则总表.md` 为准。
- `归档/mvp_out_of_scope/` 只保存未来方向，不作为当前规则来源。
- 新设定进入 MVP 前，必须先更新 `当前状态.md`、`规则总表.md`、`设定审计总表.md` 与 `命名规范.md`。
