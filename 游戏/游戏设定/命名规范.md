# 《夕妖》命名规范

本文件只维护当前 MVP 规则层命名、废弃名称与术语边界。

世界观名词定义唯一来源：

```text
世界观/世界观词条.md
```

本文件不重复维护四象本源、夕子、夕灵、夕妖、青丘、天门等世界观定义。

---

# 世界观名词引用

以下名词可以在规则文档中引用，但定义以世界观文档为准：

```text
昼曜
夜魄
晨辉
夕子
夕灵
夕妖
青丘
天门
暮君
黄昏裂隙
```

规则文档中只说明这些名词对应的规则效果，不解释世界观来源。

---

# 青丘规则命名

| 名词 | 规则层定义 |
|---|---|
| 狐火 | 青丘小队资源 |
| 狐火残留 | 地块状态 |
| 狐印 | 单位状态，标记类状态 |
| 假身 | 青丘召唤物 |
| 狐火充盈 | 小队状态 |

说明：

- 狐火是规则资源名，不等同于世界观中的夕子定义。
- 狐火残留在规则层统一称为地块状态，不再称为覆盖层。
- 狐印是单位状态，不作为人物事实或世界观标签使用。

---

# 天门规则命名

| 名词 | 规则层定义 |
|---|---|
| 律令 | 玄照颁布的临时战场规则，只影响颁布后的事件 |
| 禁行令 | 律令之一，生成禁行状态 |
| 戒严令 | 律令之一，生成戒严状态 |
| 追捕令 | 律令之一，生成追捕状态 |
| 勘验区 | 地块状态，使指定地块及相邻格上的部分地形 / 地块状态效果暂时失效 |
| 迷踪 | 单位状态，表示下一次移动受限 |
| 逃逸 | 单位状态，表示目标在追捕状态生效后完成过移动 |

说明：

- 律令不是单位状态，也不是标记。
- 勘验区在规则层统一称为地块状态，不再称为临时区域。
- 禁行状态、戒严状态、追捕状态属于全局状态。

# 废弃名称

不得继续使用：

- 暮炁
- 暮炁值
- 暮炁囊
- 暮灯
- 夕火作为资源名
- 狐火状态
- 狐火残留覆盖层
- 勘验区区域
- 律令领域
- 夕灵作为四象本源名
- 封印作为当前 MVP 状态名
