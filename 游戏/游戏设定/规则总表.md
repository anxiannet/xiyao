# 基础资源

| 资源 | 类型 | 规则 |
|---|---|---|
| AP | 行动点 | 每名单位每次激活拥有2AP |
| 气运 | 胜利分 | 3个大回合后气运高者获胜 |
| 狐火 | 青丘资源 | 青丘共享资源，0/5；达到5点后进入狐火充盈 |

## 狐火

狐火是青丘共享小队资源。

获得方式：

- 青丘单位经过带有狐火残留状态的地块：+1狐火。
- 青丘单位经过黄昏裂隙：+1狐火。
- 青丘单位命中带狐印目标：+1狐火。
- 假身参与青丘攻击支援，且本次攻击命中：+1狐火。

当狐火达到5点时，青丘获得狐火充盈。

狐火充盈是小队状态。

狐火充盈期间，青丘不再获得狐火。

青丘下一次主动发起攻击时，在掷骰前自动触发狐火充盈：

- 本次攻击 ATK_DICE +1。
- 本次攻击 DMG +1。
- 无论命中与否，攻击结算后狐火归零。

狐火充盈不可选择保留。

# 单位数值原则

当前 MVP 中，正式英雄单位 MV 最低为 2。

迷踪会降低下一次移动距离，但不能让正式英雄单位完全无法移动。

# 天门律令

玄照的核心技能是颁令。

颁令消耗1AP。

玄照每个大回合最多颁布1次律令。

同一时间只能有1条律令。

律令从颁布时开始生效。

律令不追溯已经发生的行为。

律令只影响颁布之后发生的事件与状态。

律令持续至本大回合结束，或玄照阵亡后的本大回合结束。

律令颁布后，在对局中形成对应的全局规则状态。

| 律令 | 生效期间触发 | 效果 |
|---|---|---|
| 禁行令 | 敌方单位进入边缘据点或黄昏裂隙 | 该单位获得迷踪；天门单位位于黄昏裂隙相邻格时支援面自动成功 |
| 戒严令 | 天门单位位于中央据点 | DEF_DICE +1 |
| 追捕令 | 敌方单位移动后 | 该单位获得逃逸；天门攻击带逃逸的目标时支援面自动成功 |

# 状态

规则层统一使用四类状态：

```text
单位状态
地块状态
小队状态
全局状态
```

| 状态 | 类型 | 效果 |
|---|---|---|
| 狐印 | 单位状态 | 青丘攻击该目标时支援面自动成功；命中狐印目标时青丘+1狐火 |
| 迷踪 | 单位状态 | 下一次移动距离 -1，最低保留1格移动能力 |
| 逃逸 | 单位状态 | 追捕令生效期间，敌方移动后获得；天门攻击该目标时支援面自动成功 |
| 狐火充盈 | 小队状态 | 青丘下一次主动发起攻击时，掷骰前 ATK_DICE +1、DMG +1；结算后狐火归零 |
| 狐火残留 | 地块状态 | 青丘进入该地块时 +1狐火；敌方进入该地块时获得迷踪 |
| 勘验区 | 地块状态 | 使该地块上的狐火残留、黄昏裂隙、掩影效果暂时失效 |
| 禁行状态 | 全局状态 | 禁行令生效期间的全局规则状态 |
| 戒严状态 | 全局状态 | 戒严令生效期间的全局规则状态 |
| 追捕状态 | 全局状态 | 追捕令生效期间的全局规则状态 |

状态不记录绝对好坏。

同一个状态对不同小队、不同目标和不同局面可能价值不同。

## 狐印刷新

狐印默认持续1回合。

重复施加狐印时，刷新为1回合。

狐印不叠加层数，不延长为多回合。

## 迷踪移动下限

受到迷踪影响时，下一次移动距离 -1。

最低保留1格移动能力。

迷踪触发后移除。

# 地形

| 地形 | 规则 |
|---|---|
| plain 普通格 | 无特殊规则 |
| central_objective 中央据点 | 据点范围为中心格及其相邻1圈格；被明确占领后归属保持，直到被对方明确占领；结算时归属方 +2 气运 |
| edge_objective 边缘据点 | 据点范围为中心格；被明确占领后归属保持，直到被对方明确占领；结算时归属方 +1 气运 |
| high_ground 高台 | 远程攻击从高台发起时 RNG +1 |
| cover_shadow 掩影 | 防御方 DEF_DICE +1；假身在此生成持续 +1回合 |
| dusk_rift 黄昏裂隙 | 青丘经过时收集1点狐火 |
| obstacle 障碍 | 不可进入，阻挡移动和直线视野 |

据点也是地形。

# 据点控制权

据点拥有归属状态。

据点初始归属可以是：

- 无归属
- 青丘
- 天门

单位离开据点后，不改变据点归属。

只有被对方明确占领时，据点归属才会改变。

## 据点范围

中央据点范围：

- 中央据点中心格。
- 中心格相邻1圈格。

边缘据点范围：

- 边缘据点中心格。

## 明确占领

单位执行占领动作时，检查该据点范围内正式单位数量。

只统计正式英雄单位。

不统计：

- 假身。
- 召唤物。
- 单位状态。
- 地块状态。
- 小队状态。
- 全局状态。

若执行占领方正式单位数量大于敌方正式单位数量，则该据点归属改为执行占领方。

若双方正式单位数量相同，或执行占领方数量不大于敌方，则占领失败，据点归属不变。

# 地块状态与召唤物

## 狐火残留

狐火残留是地块状态。

- 青丘单位进入带有狐火残留状态的地块：收集1点狐火。
- 敌方单位进入带有狐火残留状态的地块：获得迷踪。
- 不阻挡移动。
- 不阻挡视线。

## 勘验区

勘验区是地块状态。

来源：白烬。

持续：回合结束。

带有勘验区状态的地块：

- 狐火残留状态失效。
- 黄昏裂隙效果失效。
- 掩影效果失效。

说明：地形与地块状态仍然存在，仅对应规则效果暂时失效。

## 假身

假身是青丘召唤物，不是地块状态。

- 占格。
- 阻挡移动。
- 不阻挡视线。
- 可吸收1次攻击或技能。
- 可参与攻击支援和防御支援。
- 不占领据点。
- 假身参与青丘攻击支援且本次攻击命中时，青丘+1狐火。

# 战斗骰子

行动骰共有6面：

```text
暴击 ×1
攻击 ×2
防御 ×2
支援 ×1
```

攻击时：暴击、攻击、满足条件的支援算成功。

防御时：暴击、防御、满足条件的支援算成功。

攻击者本人不提供攻击支援。

---

# 护阵

赤霄选择1名相邻友方作为护阵目标。

护阵持续到本大回合结束。

护阵目标第一次受到攻击时，由赤霄承受本次攻击。

护阵触发后立即结束。

同一时间，赤霄只能维持1个护阵目标。

---

# 胜负

一局共3个大回合。

气运来源：

- 中央据点归属方：+2
- 边缘据点归属方：+1
- 击败敌方成员：+1
- 击败敌方队长：+2

同分时：

1. 存活单位更多者胜。
2. 队长存活者胜。
3. 控制中央据点者胜。
