# 《夕妖：占山为王》

# V0.7.5 妖怪卡表（地盘主循环 + 手牌战术循环）

原则：

- 基础卡池优先采用《西游记》原著高辨识度妖怪
- 《黑神话：悟空》角色作为替补与后续扩展来源
- 每张牌一句效果
- 不出现数值
- 不出现生命值
- 不出现费用
- 地盘必须通过卡牌效果获得，不能免费获得
- 地盘动作是主循环，手牌动作是战术循环

---

# 技能闭环原则

## 主循环：地盘

胜利条件是拥有3块地盘，因此以下动作为主循环动作：

| 动作 | 含义 |
|---|---|
| 获得 | 从中央地盘区获得地盘 |
| 抢夺 | 从其他玩家处抢夺地盘 |
| 保护 | 保护自己的地盘不被抢夺 |
| 反制 | 取消一次获得、抢夺或保护效果 |

## 战术循环：手牌

手牌决定玩家能否获得、抢夺、保护、反制地盘，因此以下动作保留：

| 动作 | 含义 |
|---|---|
| 偷看 | 查看目标玩家手牌，获得情报 |
| 偷牌 | 从目标玩家手中拿走1张牌 |
| 弃牌 | 令目标玩家失去手牌资源 |
| 多抓牌 | 增加自己的后续选择 |
| 交换牌 | 与目标玩家交换手牌资源 |

手牌动作本身不直接获胜，但能影响地盘争夺能力，属于有效战术动作。

# 反制规则草案

反制类牌可以在其他玩家打出对应效果时立即打出。

反制不占用自己的回合出牌次数。

一次效果最多被反制1次。

# 审查结果

V0.7.5 修正了 V0.7.4 过度收敛的问题。

当前版本承认两层闭环：

1. 地盘主循环：获得、抢夺、保护、反制。
2. 手牌战术循环：偷看、偷牌、弃牌、多抓牌、交换牌。

手牌动作不直接获胜，但会影响玩家是否有能力完成地盘争夺，因此保留。
