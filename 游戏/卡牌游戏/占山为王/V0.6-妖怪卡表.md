# 《夕妖：占山为王》

# V0.7.4 妖怪卡表（技能闭环重构版）

原则：

- 基础卡池优先采用《西游记》原著高辨识度妖怪
- 《黑神话：悟空》角色作为替补与后续扩展来源
- 每张牌一句效果
- 不出现数值
- 不出现生命值
- 不出现费用
- 地盘必须通过卡牌效果获得，不能免费获得
- 所有技能必须直接影响地盘争夺闭环

---

# 技能闭环原则

保留四种核心动作：

| 动作 | 含义 |
|---|---|
| 获得 | 从中央地盘区获得地盘 |
| 抢夺 | 从其他玩家处抢夺地盘 |
| 保护 | 保护自己的地盘不被抢夺 |
| 反制 | 取消一次获得、抢夺或保护效果 |

删除以下伪闭环动作：

- 弃牌
- 偷牌
- 看牌
- 抽牌
- 随机抽牌
- 回收弃牌堆
- 夺取法宝
- 交换地盘

原因：

这些动作不直接改变地盘数量，不服务“拥有3块地盘获胜”的主循环。

# 审查结果

V0.7.4 已将妖怪技能收敛到四种核心动作：

- 获得
- 抢夺
- 保护
- 反制

所有妖怪效果均直接服务于：

中央地盘 → 玩家地盘 → 玩家互抢 → 3地盘胜利

这一核心循环。

# 待进一步测试

当前版本为闭环优先版本，技能差异较少。

后续试玩时重点观察：

- 获得牌是否过多
- 抢夺牌是否足够压制领先玩家
- 保护牌是否导致拖局
- 反制牌是否造成连锁争议
- 是否需要允许少量“强条件变体”，但不得脱离四大核心动作
