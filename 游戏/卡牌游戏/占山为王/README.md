# 《夕妖：占山为王》

原概念名：妖王大会

状态：概念设计（V0.5.1）

## 设计目标

- 西游背景
- 妖怪势力争夺地盘
- 规则5分钟学会
- 一局10~20分钟
- 无生命值
- 无妖气
- 无骰子
- 无棋盘
- 无复杂结算

## 核心概念

玩家扮演一方妖王势力。

通过派遣妖怪、施展法术、使用法宝，争夺西游世界中的地盘。

率先获得3块地盘的玩家成为妖王并立即获胜。

## 一句话规则

抽1张牌 → 出1张牌 → 结束。

## 胜利条件

拥有任意3块地盘立即获胜。

## 核心规则确认

### 地盘没有特殊能力

地盘仅代表胜利点。

所有地盘价值相同。

所有特殊效果均来自妖怪、法术与法宝。

## 原型牌库结构

| 类型 | 数量 |
|---|---:|
| 地盘牌 | 12 |
| 妖怪牌 | 28 |
| 法术牌 | 20 |
| 法宝牌 | 12 |
| 合计 | 72 |

## 名称来源优先级

优先级如下：

1. 《西游记》原著
2. 西游经典影视与民间形象
3. 《黑神话：悟空》
4. 原创补充

基础卡池优先采用原著内容。

黑神话相关角色优先作为扩展内容补充。

## 地盘设计原则

优先使用妖怪势力盘踞地点。

推荐：

- 白骨岭
- 火焰山
- 狮驼岭
- 黄风岭
- 黑风山
- 平顶山
- 盘丝洞
- 流沙河
- 通天河

尽量减少国家与城池类地点。

## 妖怪设计原则

### 唯一来源

优先采用西游记原著妖怪。

黑神话角色作为补充来源。

### 妖怪职责体系

- 抢夺型
- 破坏型
- 盗窃型
- 防御型
- 特殊型

## 核心体验

占山头
抢地盘
夺法宝
当妖王
