# 《夕妖：占山为王》

原概念名：妖王大会

状态：概念设计（V0.3）

## 设计目标

- 西游背景
- 妖怪势力争夺地盘
- 规则5分钟学会
- 一局10~20分钟
- 不绑定 UNO 固定牌数，按玩法实际需要估算
- 无生命值
- 无妖气
- 无骰子
- 无棋盘
- 无复杂结算

## 核心概念

玩家扮演一方妖王势力。

通过派遣妖怪、施展法术、使用法宝，争夺西游世界中的地盘。

率先获得3块地盘的玩家成为妖王并立即获胜。

## 一句话规则

抽1张牌 → 出1张牌 → 结束。

## 胜利条件

拥有任意3块地盘。

例如：

- 白骨岭
- 火焰山
- 盘丝洞

达到3块地盘立即获胜。

## 卡牌类型

### 地盘牌

胜利条件来源。

示例：

- 白骨岭
- 火焰山
- 黄风岭
- 狮驼岭
- 流沙河
- 通天河
- 盘丝洞

### 妖怪牌

用于获得或抢夺地盘。

示例：

- 虎先锋（抢夺一块地盘）
- 狼妖（获得中央地盘区一块地盘）
- 蜘蛛精（偷取手牌）

### 法术牌

用于干扰其他玩家。

示例：

- 妖风
- 摄魂术
- 遁地术

### 法宝牌

用于防御和反制。

示例：

- 芭蕉扇
- 紫金葫芦
- 替身草人

## 回合流程

1. 抽1张牌
2. 打出1张牌
3. 结束回合

## 原型牌库结构

V0.3 原型版暂定总牌数为 72 张。

该数量不是为了匹配 UNO 规格，而是按以下需求估算：

- 2~6 人可玩
- 4 人局体验稳定
- 洗牌不累
- 关键牌不会过度稀释
- 一局控制在 10~20 分钟

| 类型 | 数量 | 设计目的 |
|---|---:|---|
| 地盘牌 | 12 | 胜利条件与争夺核心 |
| 妖怪牌 | 28 | 获得地盘、抢夺地盘、偷牌、制造互动 |
| 法术牌 | 20 | 干扰、反制、改变局势 |
| 法宝牌 | 12 | 防御、保护地盘、反弹效果 |
| 合计 | 72 | 原型测试牌量 |

## 地盘区规则草案

推荐采用中央地盘区。

开局从牌库中翻出 3 张地盘牌，放在桌面中央，形成中央地盘区。

当中央地盘区的地盘被玩家获得后，立即从牌库继续翻开地盘牌补充，尽量保持中央地盘区有 3 张地盘。

这样可以保证玩家始终有明确争夺目标，避免没有地盘可抢。

## 设计原则

优先保证：

- 规则简单
- 互动频繁
- 西游味浓
- 适合聚会
- 非卡牌玩家也能快速上手

避免：

- 生命值系统
- 资源系统
- 残血记录
- 复杂数值计算
- 长时间对局

## 核心体验

占山头
抢地盘
夺法宝
当妖王
