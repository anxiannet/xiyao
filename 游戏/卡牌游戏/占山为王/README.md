# 《夕妖：占山为王》

原概念名：妖王大会

状态：概念设计（V0.5）

## 设计目标

- 西游背景
- 妖怪势力争夺地盘
- 规则5分钟学会
- 一局10~20分钟
- 不绑定 UNO 固定牌数，按玩法实际需要估算
- 无生命值
- 无妖气
- 无骰子
- 无棋盘
- 无复杂结算

## 核心概念

玩家扮演一方妖王势力。

通过派遣妖怪、施展法术、使用法宝，争夺西游世界中的地盘。

率先获得3块地盘的玩家成为妖王并立即获胜。

## 一句话规则

抽1张牌 → 出1张牌 → 结束。

## 胜利条件

拥有任意3块地盘立即获胜。

## 核心规则确认

### 地盘没有特殊能力

地盘仅代表胜利点。

所有地盘价值相同。

所有特殊效果均来自妖怪、法术与法宝。

## 妖怪设计原则

### 唯一来源

妖怪名称优先采用《黑神话：悟空》已出现妖怪、头目、妖王。

尽量避免原创妖怪名称。

### 技能来源

技能优先参考《黑神话：悟空》中对应角色的战斗特点。

不是照搬动作。

而是抽象为简单卡牌效果。

例如：

- 善于突袭 → 抢夺地盘
- 善于控制 → 令目标弃牌
- 善于隐匿 → 防止被指定
- 善于召唤 → 获得额外收益
- 善于夺宝 → 抢夺法宝

### 妖怪职责体系

28张妖怪分为五类。

#### 抢夺型

核心职责：抢地盘。

示例：

- 虎先锋
- 石先锋
- 百目真人

#### 破坏型

核心职责：弃牌、拆除保护。

示例：

- 黄风大圣
- 虫总兵

#### 盗窃型

核心职责：偷牌、夺宝。

示例：

- 狐妖
- 蜘蛛精

#### 防御型

核心职责：保护地盘。

示例：

- 黑熊精
- 靡道人

#### 特殊型

核心职责：交换、复制、反制。

示例：

- 红孩儿
- 金角
- 银角

## 设计原则

优先保证：

- 规则简单
- 互动频繁
- 西游味浓
- 非卡牌玩家可快速上手

避免：

- 生命值系统
- 资源系统
- 残血记录
- 长连锁结算
- 大量随机判定

## 核心体验

占山头
抢地盘
夺法宝
当妖王
