# 《夕妖：占山为王》

原概念名：妖王大会

状态：概念设计（V0.8）

## 设计目标

- 西游背景
- 妖怪势力争夺地盘
- 规则5分钟学会
- 一局10~20分钟
- 无生命值
- 无妖气
- 无骰子
- 无棋盘
- 无复杂结算

## 核心概念

玩家扮演一方妖王势力。

通过派遣妖怪、施展法术、使用法宝，争夺西游世界中的地盘。

率先凑够3点地盘势力的玩家成为妖王并立即获胜。

## 一句话规则

抽1张牌 → 出1张牌 → 结算地盘效果 → 结束。

## 胜利条件

拥有地盘势力合计达到3点，立即获胜。

## 核心规则确认

### 地盘有轻量价值

地盘不再完全等价。

每张地盘拥有：

- 地盘势力值
- 一句简单效果

地盘效果必须非常轻，不引入复杂资源系统。

### 地盘势力值

基础版只使用两种势力值：

| 势力值 | 含义 |
|---|---|
| 1 | 普通地盘 |
| 2 | 大地盘 / 难守但价值高 |

胜利条件为合计3点地盘势力。

例如：

- 3张普通地盘
- 1张大地盘 + 1张普通地盘

都可以获胜。

### 地盘效果原则

地盘效果只允许围绕：

- 抽牌
- 看牌
- 防抢
- 反制
- 限制抢夺
- 地盘保护

禁止引入：

- 生命值
- 妖气
- 攻击力
- 长期计数器
- 多段复杂连锁

## 名称来源优先级

优先级如下：

1. 《西游记》原著
2. 西游经典影视与民间形象
3. 《黑神话：悟空》
4. 原创补充

基础卡池优先采用原著内容。

黑神话相关角色优先作为扩展内容补充。

## 地盘设计原则

优先使用妖怪势力盘踞地点。

推荐：

- 白骨岭
- 火焰山
- 狮驼岭
- 黄风岭
- 黑风山
- 平顶山
- 盘丝洞
- 流沙河
- 通天河

尽量减少国家与城池类地点。

## 妖怪设计原则

优先采用西游记原著妖怪。

黑神话角色作为补充来源。

小妖可以重复。

不同卡名技能必须不同。

头目与妖王在基础卡池中唯一。

## 核心体验

占山头
抢地盘
夺法宝
当妖王
