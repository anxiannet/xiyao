## 文件定位

本文件用于维护《占山为王》实体卡与网页卡正面显示的技能短文案。

本文件不替代规则。

完整规则与卡牌效果仍以：

`游戏/卡牌游戏/占山为王/V1.2-基础卡表.md`

为唯一来源。

本文件只负责把已确认技能压缩为适合卡面排版的一句话。

---

# 卡面文案原则

卡面技能文案用于欢乐聚会卡牌场景。

要求：

- 一句话
- 原则上一行显示
- 统一使用句号结尾
- 不显示技能名
- 不显示“登场技”“主动技”“被动技”
- 数字统一使用阿拉伯数字，例如 `1只`、`1星`、`1块`
- 妖怪、守军默认使用 `只`，避免与西游妖怪题材割裂
- 去向必须写清楚，例如 `加入你的地盘`、`加入你的手牌`、`弃掉`、`吹回`
- 字少时不放大成标题，仍居中显示在统一技能栏内
- 字长时优先压缩文案，不单独改变 3 星首领卡 UI
- 尽量保留西游与TRPG代入感
- 同时保留规则结果

卡面允许使用更有代入感的动词，例如：

- 偷取
- 掳走
- 吃掉
- 烧掉
- 绑走
- 收走
- 吹回

但必须避免规则结果不清楚。

# 五、卡面排版适配规则

## 3星首领技能栏

3星首领统一使用当前红金边框与底部短技能铭牌。

技能文案长短不改变 UI 结构。

要求：

- 技能栏高度固定
- 技能文字水平居中
- 技能文字垂直居中，允许略微偏上
- 短文案不放大成标题
- 长文案优先精简规则文字
- 原则上不拆成两行
- 若实体印刷测试发现过小，可微调字号，但不改变技能栏比例

推荐字号策略：

| 文案长度 | 处理方式 |
|---|---|
| 8字以内 | 标准字号，居中，不大幅放大 |
| 9至16字 | 标准字号，居中 |
| 17至24字 | 字号略缩小，仍尽量一行 |
| 24字以上 | 优先回到规则层精简文案 |

# 六、名字 UI 适配规则

本文件不定义名字牌美术，但卡面技能文案排版需要与名字 UI 保持一致。

3星首领统一使用右侧竖排名字牌。

名字长短不改变名字牌结构。

要求：

- 右侧红底金边竖牌固定
- 金色竖排文字固定
- 不因短名字缩短名字牌
- 不因长名字改成横排
- 不添加“首领”“妖王”“精英”等身份标签
- 超长名字优先在卡牌正面改用短名，完整称号放入规则书或图鉴

推荐适配：

| 名字字数 | 处理方式 |
|---|---|
| 2至3字 | 字号略大，竖排居中 |
| 4至6字 | 标准字号 |
| 7至8字 | 字号略缩小，字距略收紧 |
| 9至10字 | 优先改短名；必要时才考虑双列竖排 |
| 12字以上 | 不建议出现在卡牌正面 |

# 七、用词说明

## 偷取

用于金鼻白毛老鼠精。

卡面写作：

`偷取1只1星守军，加入你的地盘。`

原因：

“偷取”比“偷”更适合正式卡面文案，同时保留老鼠精偷盗的角色感。

必须写清楚被偷守军的去向，否则玩家会误解为加入手牌、弃牌或暂时控制。

## 收走

用于紫金红葫芦。

卡面写作：

`收走1只守军并弃掉。`

原因：

只写“收走1只守军”会产生去向歧义。

当前规则中，被紫金红葫芦收走的守军进入弃牌堆，因此卡面必须明确“弃掉”。

## 取消

用于金刚琢。

卡面写作：

`取消1次法宝效果。`

原因：

“取消1张法宝牌效果”更偏卡牌规则语言。

“取消1次法宝效果”更符合西游法宝对抗与TRPG代入感，同时规则结果清楚。

## 绑走

用于幌金绳。

卡面写作：

`绑走1只守军，加入你的手牌。`

原因：

体现幌金绳“捆人、抓人”的西游味，同时明确加入手牌。

## 抢地盘

卡面文案统一使用“抢地盘”，不使用“攻山”。

原因：

“抢地盘”更符合欢乐聚会卡牌的口语体验，玩家更容易理解和表达。

规则文档中若仍使用“攻山”，以规则文档为准；卡面展示时使用“抢地盘”。

# 禁止写法

卡面禁止出现：

- 攻击力
- 防守力
- 阵营区
- 职业
- 技能名
- 登场技
- 主动技
- 被动技
- 长段规则说明
- 多句解释
- “技能”二字
- 技能图标
- “首领”标签
- “精英”标签
