# 《夕妖》命名规范

## 四象本源

```text
昼曜
夜魄
晨辉
夕子
```

## 夕子

夕子是黄昏本源的基础单位。

定位类似：

```text
量子
原子
分子
夕子
```

夕子代表变化、过渡、感知和可能性。

夕子不是小队资源。

## 夕灵

夕灵是能够感知、接触或运用夕子的生灵。

## 夕妖

夕妖是天庭对夕灵的统称。

青丘狐族属于夕妖的一类。

# 天门命名

| 名词 | 定义 |
|---|---|
| 律令 | 玄照颁布的临时战场规则，只影响颁布后的事件 |
| 勘验区 | 白烬制造的临时区域，使异常环境失效 |
| 迷踪 | 负面状态，表示下一次移动受限 |
| 逃逸 | 追捕令产生的负面状态，表示目标在律令生效后移动过 |
