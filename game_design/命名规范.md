# 《夕妖》命名规范

## 四象本源

```text
昼曜
夜魄
晨辉
夕子
```

## 夕子

夕子是黄昏本源的基础单位。

定位类似：

```text
量子
原子
分子
夕子
```

夕子代表变化、过渡、感知和可能性。

夕子不是战帮资源。

## 夕灵

夕灵是能够感知、接触或运用夕子的生灵。

## 夕妖

夕妖是天庭对夕灵的统称。

青丘狐族属于夕妖的一类。

# 废弃名称

不得继续使用：

- 暮炁
- 暮炁值
- 暮炁囊
- 暮灯
- 夕火作为资源名
- 狐火状态
- 夕灵作为四象本源名
