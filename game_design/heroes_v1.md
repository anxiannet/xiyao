# 《夕妖》英雄设计 V1

## 目标

Alpha 阶段不追求完整72英雄。

目标是先完成12名可开发英雄。

用于验证：

- 战帮差异
- 四象系统
- AI对战
- 化形机制
- 15分钟单局节奏

---

# Alpha 首批英雄数量

12名英雄。

覆盖6大势力。

每个势力2名。

形成最小可玩角色池。

---

# 英雄字段标准

每个英雄需要具备：

- 名称
- 势力
- 定位
- 四象倾向
- 初始形态
- 化形形态
- 普通技能
- 化形技能
- 触发条件
- 美术关键词

---

# 天巡司

## 玄照

势力：天巡司

定位：前排控制

四象：昼曜

初始形态：天门执法官

# Alpha英雄设计原则

1. 每个势力至少体现一种战术特色。
2. 每名英雄必须有明确化形条件。
3. 化形条件要鼓励玩家主动操作。
4. 不做复杂长文本技能。
5. 每名英雄最多2个主动技能。
6. AI必须能理解英雄用途。
