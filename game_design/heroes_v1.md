# 《夕妖》单位设计 V1

## 重要修正

《夕妖》不是单英雄战棋。

玩家选择并操控的是完整战帮（Warband），不是单个英雄。

因此 Alpha 阶段不再采用“每势力2名英雄”的设计。

正确结构为：

- 6个势力
- 每个势力1个完整战帮
- 每个战帮3名单位
- 共6个战帮
- 共18名单位

每个战帮包含：

- 1名队长
- 2名成员

# 单位字段标准

每名单位应包含：

- 名称
- 所属战帮
- 队内身份：队长 / 成员
- 定位
- 四象倾向
- 初始形态
- 化形形态
- 普通能力
- 化形能力
- 化形条件
- 与队友协同
- 美术关键词

---

# 队长与成员的区别

## 队长

队长是战帮的核心。

特点：

- 能力更强
- 化形后提供全队增益
- 死亡会影响战帮节奏
- 通常决定战帮打法

## 成员

成员不是杂兵。

成员承担明确战术职责：

- 控制
- 治疗
- 位移
- 追击
- 占点
- 远程
- 诅咒

---

# Alpha设计目标

Alpha 不追求单位数量多。

只追求6个战帮都能完整成立。

玩家进入游戏时，应选择：

- 天门执法队
- 北境守军
- 青丘使团
- 啸月猎团
- 观星者
- 余烬小队

而不是选择单个英雄。
