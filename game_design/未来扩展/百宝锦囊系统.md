# 《夕妖》百宝锦囊系统（未来扩展）

## 当前状态

百宝囊、宝物、妙计、阵型、法术不进入 MVP。

MVP 只验证：

- 英雄核心技能
- 地形
- 覆盖层
- 状态
- 战帮资源
- 律令
- 战斗骰子
- 占点与气运
- AI 对战

---

# 未来定位

百宝锦囊是战帮战术模块的高级系统。

未来可包含：

- 宝物
- 妙计
- 阵型
- 法术
- 令牌
- 事件
- 机关
- 盟友支援

---

# 未来设计原则

1. 百宝锦囊不能替代英雄核心技能。
2. 百宝锦囊应改变战术风格，而不是制造额外规则负担。
3. 每套锦囊应围绕一个清晰主题。
4. 新增宝物前必须先检查设定审计总表。
5. MVP 完成数值验证后，再决定是否恢复宝物系统。

# 未来内容类型

## 宝物

偏一次性物件效果。

## 妙计

偏临场反应与时机操作。

## 阵型

偏开局部署、站位奖励或持续阵地风格。

## 法术

偏消耗资源、制造状态或改变节奏。
