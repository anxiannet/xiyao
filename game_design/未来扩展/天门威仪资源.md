# 天门威仪资源（未来扩展）

## 当前状态

威仪不进入当前 MVP。

当前 MVP 的天门执法队只保留：

- 律令
- 律令领域
- 晨辉净化
- 护阵
- 封印
- 迷踪

---

# 归档原因

战帮不是必须拥有战帮资源。

青丘保留狐火，是因为狐火来自地图路线、狐火残留和黄昏裂隙，直接构成青丘玩法。

天门的威仪在当前版本中更像中间计数器：

```text
律令触发
↓
获得威仪
↓
消耗威仪
↓
强化律令或攻击
```

这会增加记账复杂度，但没有明显增加 MVP 决策深度。

因此移出 MVP，留待未来扩展。

---

# 未来可回归条件

若未来恢复威仪，需要满足：

1. 获得威仪有明确成本。
2. 消耗威仪有多个互斥选择。
3. 储存威仪有风险。
4. 玩家需要在“现在花”与“以后花”之间做判断。

---

# 未来方向示例

威仪上限可设为 3。

用途示例：

- 1威仪：本次防御 DEF_DICE +1。
- 1威仪：立即触发一次友方净化。
- 2威仪：生成律令领域。
- 3威仪：本大回合额外维持一条律令。

以上仅为未来方向，不进入当前 MVP。
