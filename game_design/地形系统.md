# 《夕妖》地形系统

## 地图原则

地图由六角格、基础地形、地块状态和据点地形组成。

地图背景只负责氛围，不参与规则计算。

覆盖层、区域、领域等词只属于程序实现或视觉表现，不作为规则本体。

规则层统一使用：

```text
单位状态
地块状态
小队状态
全局状态
```

---

# 基础地形

| 地形 | 规则 |
|---|---|
| plain 普通格 | 无特殊规则 |
| central_objective 中央据点 | 据点范围为中心格及相邻1圈格；归属方在回合结算时 +2 气运 |
| edge_objective 边缘据点 | 据点范围为中心格；归属方在回合结算时 +1 气运 |
| high_ground 高台 | 远程攻击从高台发起时 RNG +1 |
| cover_shadow 掩影 | 防御方 DEF_DICE +1；假身在此生成持续 +1回合 |
| dusk_rift 黄昏裂隙 | 青丘单位经过时收集1点狐火 |
| obstacle 障碍 | 不可进入；阻挡移动和直线视野 |

---

# 据点规则

据点是地形。

据点拥有归属状态。

据点初始归属可以是：

- 无归属
- 青丘
- 天门

单位离开据点后，不改变据点归属。

只有被对方明确占领时，据点归属才会改变。

## 据点范围

中央据点范围：

- 中央据点中心格。
- 中心格相邻1圈格。

边缘据点范围：

- 边缘据点中心格。

## 明确占领

单位执行占领动作时，检查该据点范围内正式单位数量。

只统计正式英雄单位。

不统计：

- 假身。
- 召唤物。
- 地块状态。
- 小队状态。
- 全局状态。

若执行占领方正式单位数量大于敌方正式单位数量，则该据点归属改为执行占领方。

若双方正式单位数量相同，或执行占领方数量不大于敌方，则占领失败，据点归属不变。

---

# 视线规则

阻挡直线视野：

- 障碍

不阻挡直线视野：

- 普通单位
- 假身
- 狐火残留状态
- 据点
- 掩影
- 黄昏裂隙
- 勘验区状态

---

# 地块状态

地块状态是附着在地图格子上的持续效果。

它可以由技能产生，也可以由规则结算产生。

地块状态不是基础地形。

地块状态不改变地形本身，只改变该地块在持续时间内的规则表现。

---

## 狐火残留

狐火残留是地块状态。

来源：青萝。

规则：

- 青丘单位进入带有狐火残留状态的地块：收集1点狐火。
- 敌方单位进入带有狐火残留状态的地块：获得迷踪。
- 不阻挡移动。
- 不阻挡视线。
- 不能生成在障碍格。

---

## 勘验区

勘验区是地块状态。

来源：白烬。

规则：

- 指定1格及相邻格。
- 这些地块获得勘验区状态。
- 持续到回合结束。
- 带有勘验区状态的地块上，狐火残留状态失效。
- 带有勘验区状态的黄昏裂隙，黄昏裂隙效果失效。
- 带有勘验区状态的掩影，掩影效果失效。
- 不阻挡移动。
- 不阻挡视线。

失效不代表移除。

地形与地块状态仍然存在，只是本回合不产生对应规则效果。

---

# 召唤物：假身

假身是场上实体，不是地块状态。

规则：

- 占格。
- 阻挡移动。
- 不阻挡视线。
- 可被攻击或技能指定。
- 承受1次攻击或技能后消失。
- 不占领据点。
- 可提供支援。

---

# 状态关系

狐火残留和勘验区都属于地块状态。

勘验区不会移除狐火残留，只会让带有勘验区状态的地块上，狐火残留在本回合失效。
