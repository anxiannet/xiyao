# 《夕妖》地形系统

## 地图原则

地图由程序生成六角网格、基础地形、覆盖层和据点。

地图背景只负责氛围，不参与规则计算。

---

# 基础地形

| 地形 | 规则 |
|---|---|
| plain 普通格 | 无特殊规则 |
| central_objective 中央据点 | 可占领；回合结束控制方 +2 气运 |
| edge_objective 边缘据点 | 可占领；回合结束控制方 +1 气运 |
| high_ground 高台 | 远程攻击从高台发起时 RNG +1 |
| cover_shadow 掩影 | 防御方 DEF_DICE +1；假身在此生成持续 +1回合 |
| dusk_rift 黄昏裂隙 | 青丘单位经过时收集1点狐火 |
| obstacle 障碍 | 不可进入；阻挡移动和直线视野 |

---

# 视线规则

阻挡直线视野：

- 障碍

不阻挡直线视野：

- 普通单位
- 假身
- 狐火残留
- 据点
- 掩影
- 黄昏裂隙
- 勘验区

---

# 覆盖层与区域

## 狐火残留

青丘覆盖层。

来源：青萝。

规则：

- 青丘单位经过：收集1点狐火。
- 敌方单位经过：获得迷踪。
- 不阻挡移动。
- 不阻挡视线。
- 不能生成在障碍格。

## 勘验区

天门区域。

来源：白烬。

规则：

- 指定1格及相邻格。
- 持续到回合结束。
- 区域内狐火残留失效。
- 区域内黄昏裂隙失效。
- 区域内掩影失效。
- 不阻挡移动。
- 不阻挡视线。

失效不代表移除。

地形与覆盖层仍然存在，只是本回合不产生规则效果。

## 假身

青丘召唤物。

规则：

- 占格。
- 阻挡移动。
- 不阻挡视线。
- 可被攻击或技能指定。
- 承受1次攻击或技能后消失。
- 不占领据点。
- 可提供支援。

---

# 覆盖与区域关系

狐火残留是覆盖层。

勘验区是区域。

勘验区不会移除狐火残留，只会让区域内的狐火残留本回合失效。
