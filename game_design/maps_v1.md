# 《夕妖》地图设计 V1

## 目标

Alpha 阶段制作5张地图。

每张地图承担不同测试目标。

用于验证：

- 移动
- 占点
- 地形
- 四象共鸣
- AI路径决策

---

# 地图通用结构

每张地图包含：

- 地图名称
- 格子规模
- 中央目标
- 地形机制
- 四象倾向
- 推荐战帮
- AI测试重点

---

# 地图1：天门关

规模：19格

定位：教学地图

四象倾向：昼曜

中央目标：天门令台

机制：

占领中央据点获得气运。

地形简单。

无复杂障碍。

测试目的：

验证基础移动、占点、攻击、结算。

适合：

天巡司
镇妖军

AI重点：

是否会主动争夺中央据点。

---

# 地图2：青丘谷

规模：37格

定位：机动地图

四象倾向：夕火

中央目标：灵狐圣树

机制：

黄昏回合中，部分边缘格可连通。

角色可以绕后。

测试目的：

验证位移、偷点、假身、机动战术。

适合：

青丘狐族
苍狼部族

AI重点：

是否能识别绕后路线。

---

# 地图3：暮渊裂谷

规模：37格

定位：风险地图

四象倾向：夕火 / 夜魄

中央目标：暮渊裂缝

机制：

站在裂缝附近获得额外夕火。

但每回合可能受到伤害。

测试目的：

验证高风险高收益目标。

适合：

暮渊遗民
青丘狐族

AI重点：

是否能判断收益与风险。

---

# 地图4：苍狼雪原

规模：37格

定位：冲锋地图

四象倾向：夜魄

中央目标：狼王猎场

机制：

直线路径更长。

高地提供攻击加成。

残血单位更容易被追击。

测试目的：

验证冲锋、追击、远近距离压迫。

适合：

苍狼部族
镇妖军

AI重点：

是否优先攻击残血单位。

---

# 地图5：晨辉神殿

规模：37格

定位：回复地图

四象倾向：晨辉

中央目标：晨辉圣坛

机制：

占领圣坛可回复生命。

但连续占领会触发净化代价。

净化代价：随机移除一个增益效果。

测试目的：

验证治疗、净化、防守反击。

适合：

天巡司
星宫
镇妖军

AI重点：

是否会利用回复点维持战力。

---

# 地图设计原则

1. Alpha地图不要太大。
2. 每张地图只测试一个核心变量。
3. 地图机制必须与四象系统绑定。
4. AI必须能理解地图目标。
5. 地图视觉要强烈区分。
