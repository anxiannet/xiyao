# 《夕妖》MVP 可玩闭环规格 V1.3

## 核心目标

MVP 目标不是做完整内容库，而是做一套真正能玩的核心盒。

MVP 只包含：

- 2支主战帮：天门执法队、青丘使团。
- 程序生成地图网格。
- 按模板摆放地形与覆盖层。
- 6套百宝囊。
- 骰子攻防系统。
- 5个教学/战役关卡。
- 1个自由对战模式。

所有技能、百宝囊、状态、地形、骰子必须形成闭环。

---

# 两支战帮的核心闭环

## 青丘使团

核心机制：

```text
掩影 / 裂隙 / 边缘据点
↓
狐火残留
↓
假身 / 迷踪 / 换位
↓
偷点或集火
```

资源：暮炁。

优势：机动、诱骗、残留地形、边缘控制。

弱点：正面抗压弱，怕追缉、封印和律令领域。

## 天门执法队

核心机制：

```text
昼曜令台 / 中央据点 / 高台
↓
律令领域
↓
威仪
↓
追缉 / 封印 / 净化
↓
压缩青丘行动空间
```

资源：威仪。

优势：阵地、净化、追缉、压制特殊位移。

弱点：单位少、移动慢、怕多点拉扯和假身诱骗。

# MVP 状态与覆盖层

## 核心状态

- 封印：禁止化形和部分特殊技能。
- 迷踪：行动方向受干扰或移动受限。
- 易伤：受到攻击时防御骰-1，最低1。
- 追缉：受到天门攻击时防御骰-1，最低1。

## 核心覆盖层

- 狐火残留：青丘生成，提供暮炁和干扰。
- 律令领域：天门生成，提供威仪和压制。
- 假身：青丘生成，诱导攻击和封印。
- 关闭裂隙：天门临时封锁裂隙。

覆盖层互斥：

```text
狐火残留 ↔ 律令领域
```

后生成者覆盖前者。

# 地形闭环

```text
青丘利用掩影、裂隙、边缘据点生成狐火残留
↓
天门占据高台、中央据点、昼曜令台生成律令领域
↓
律令领域覆盖狐火残留并积累威仪
↓
青丘绕开正面，用边缘点和假身分散天门
↓
天门用追缉锁定核心单位
```

# 骰子闭环

攻击和防御使用同一套骰子规则。

骰面：

- 命中
- 暴击
- 专注
- 空白

攻击方掷攻击骰。

防御方掷防御骰。

成功数比较：

```text
攻击成功数 > 防御成功数 = 命中
攻击成功数 <= 防御成功数 = 未命中
暴击可触发额外伤害或特殊效果
```

地形、状态、宝物影响骰子数量或骰面解释。

例如：

- 高台：距离+1或攻击骰+1，二选一。
- 掩影：防御骰+1。
- 律令领域：天门防御骰+1。
- 追缉：目标防御骰-1。
- 天罚令：攻击获得专注。

# 教学关卡围绕两支战帮展开

## 关卡1：封界初令

教学：移动、攻击、占点、骰子。

使用：天门基础单位 vs 青丘基础单位。

## 关卡2：狐火留痕

教学：狐火残留、暮炁、覆盖层。

## 关卡3：律令压境

教学：律令领域、威仪、覆盖互斥。

## 关卡4：追缉与假身

教学：追缉、假身、掩影、侦测。

## 关卡5：青丘谷决战

教学：完整对局，双方百宝囊各选1套。

# MVP 可玩标准

完成 MVP 时，玩家应该能够：

1. 选择天门或青丘。
2. 为战帮选择一套百宝囊。
3. 在程序生成的六角地图上部署。
4. 使用骰子进行攻击和防御。
5. 使用地形、覆盖层、状态、宝物进行战术决策。
6. 在15分钟内完成一局。
7. 明显感受到两支战帮打法不同。

# 暂不进入 MVP 的系统

- 命运系统。
- 联网天梯。
- 大量势力扩展。
- 抽卡。
- 战帮养成。
- 超大地图。

这些内容保留为后续版本。
