# 《夕妖》MVP 可玩闭环规格 V1.7

## 核心目标

MVP 目标不是做完整内容库，而是做一套真正能玩的核心盒。

MVP 只包含：

- 2支主战帮：天门执法队、青丘使团。
- 程序生成地图网格。
- 按模板摆放地形与覆盖层。
- 7个基础地形标签。
- 6套百宝囊。
- 战斗骰子系统。
- 5个教学/战役关卡。
- 1个自由对战模式。

所有技能、百宝囊、状态、地形、骰子必须形成闭环。

# 两支战帮的核心闭环

## 青丘使团

核心机制：

```text
掩影 / 裂隙 / 边缘据点
↓
狐火残留
↓
收集狐火
↓
狐印 / 假身 / 迷踪 / 换位
↓
偷点或集火
```

资源：狐火。

优势：机动、狐火残留、假身支援、边缘控制。

弱点：正面抗压弱，怕追缉、封印和律令领域。

## 天门执法队

核心机制：

```text
中央据点 / 高台
↓
律令领域
↓
威仪
↓
追缉 / 封印 / 净化
↓
压缩青丘行动空间
```

资源：威仪。

优势：阵地、净化、追缉、防御支援、压制特殊位移。

弱点：单位少、移动慢、怕多点拉扯和假身阻挡。

# MVP 状态与覆盖层

## 核心状态

- 封印：禁止主动技能，且不可被晨辉净化移除。
- 狐印：青丘攻击该目标时，支援面自动成功。
- 迷踪：下一次移动距离 -1。
- 易伤：受到攻击时 DEF_DICE -1，最低1。
- 追缉：受到天门攻击时 DEF_DICE -1，最低1。
- 护盾：抵消 1 点伤害。

## 核心覆盖层

- 狐火残留：青丘生成，青丘经过时收集狐火，天门经过时获得迷踪。
- 律令领域：天门生成，提供威仪和防御支援。
- 假身：青丘生成，占格、阻挡、吸收一次攻击或技能，并参与支援判定。

覆盖层互斥：

```text
狐火残留 ↔ 律令领域
```

后生成者覆盖前者。

# 地形闭环

```text
青丘利用掩影、裂隙、边缘据点生成狐火残留
↓
青丘通过狐火残留收集狐火，并用假身参与支援
↓
天门占据高台、中央据点，生成律令领域
↓
律令领域覆盖狐火残留，并让天门防御支援自动成功
↓
障碍限制双方路线，使站位和支援更重要
↓
青丘绕开正面，用边缘点和假身分散天门
↓
天门用追缉和封印锁定核心单位
```

# 战斗骰子闭环

攻击和防御使用同一套六面骰。

骰面：

```text
暴击 ×1
攻击 ×2
防御 ×2
支援 ×1
```

攻击时：

- 暴击 = 成功。
- 攻击 = 成功。
- 防御 = 失败。
- 支援 = 满足攻击支援条件则成功。

防御时：

- 暴击 = 成功。
- 防御 = 成功。
- 攻击 = 失败。
- 支援 = 满足防御支援条件则成功。

攻击支援条件：

被攻击目标周围 1 格内存在至少 1 个攻击方单位，且该单位不是本次攻击者。

防御支援条件：

防御者周围 1 格内存在至少 1 个防御方友军。

特殊自动支援：

- 青丘攻击带狐印目标时，支援面自动成功。
- 天门单位位于律令领域内防御时，支援面自动成功。
- 假身可以参与攻击支援和防御支援。

完整规则见：

```text

```

# 教学关卡围绕两支战帮展开

## 关卡1：封界初令

教学：移动、攻击、占点、战斗骰子。

使用：天门基础单位 vs 青丘基础单位。

## 关卡2：狐火留痕

教学：狐火残留、狐火收集、覆盖层。

## 关卡3：律令压境

教学：律令领域、威仪、覆盖互斥。

## 关卡4：追缉与假身

教学：追缉、假身、掩影、支援判定。

## 关卡5：青丘谷决战

教学：完整对局，双方百宝囊各选1套。

# MVP 可玩标准

完成 MVP 时，玩家应该能够：

1. 选择天门或青丘。
2. 为战帮选择一套百宝囊。
3. 在程序生成的六角地图上部署。
4. 使用战斗骰子进行攻击和防御。
5. 使用地形、覆盖层、状态、宝物进行战术决策。
6. 在15分钟内完成一局。
7. 明显感受到两支战帮打法不同。

# 暂不进入 MVP 的系统

- 命运系统。
- 联网天梯。
- 大量势力扩展。
- 抽卡。
- 战帮养成。
- 超大地图。
- 临时黄昏裂隙。
- 直接改变基础地形的技能或宝物。

这些内容保留为后续版本。
