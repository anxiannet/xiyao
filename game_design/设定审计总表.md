# 《夕妖》设定审计总表

## 用途

本文档用于集中审查 MVP 设定是否存在：

- 名词冲突
- 状态无来源
- 状态无用途
- 宝物重复
- 地形无人联动
- 资源无消耗
- 技能与战帮背景不一致
- 后续战帮扩展时的机制冲突

本文档是审计入口，不替代各规则原文。

|---|---|---|---|
| 昼曜 | 四象本源 | 与白昼、秩序、照见相关的本源 | 天门可关联 |
| 夜魄 | 四象本源 | 与黑夜、魂魄、隐秘相关的本源 | 后续战帮预留 |

| 夕灵 | 生灵类别 | 能感知、接触或运用夕子的生灵 | 天庭称其为夕妖 |

| 夕妖定义 | 天庭对夕灵的统称 |
