# 《夕妖》设定审计总表

## 用途

本文档用于集中审查 MVP 设定是否存在名词冲突、状态无来源、宝物重复、地形无人联动、资源无消耗、机制同质化等问题。

# 10. 关键闭环审计

## 狐印闭环

| 项目 | 内容 |
|---|---|
| 来源 | 苏绫狐步 |
| 用途 | 支援自动成功、琉尾增伤、燃尾令、镇妖钉识别敌方标记 |
| 解法 | 友方净化、持续结束 |
| 结论 | 闭环成立 |

## 律令闭环

| 项目 | 内容 |
|---|---|
| 来源 | 玄照颁令 |
| 用途 | 改变本大回合规则，制造威仪、领域和攻击窗口 |
| 解法 | 玄照阵亡后无法颁布新律令；律令持续到大回合结束 |
| 结论 | 闭环成立，且区别于狐印标记体系 |

## 狐火闭环

| 项目 | 内容 |
|---|---|
| 来源 | 狐火残留、黄昏裂隙 |
| 用途 | 燃尾令、灵狐羽、狐踪符的移动、追击、增伤 |
| 解法 | 天门压缩路线、覆盖狐火残留 |
| 结论 | 已补强消耗 |

## 假身闭环

| 项目 | 内容 |
|---|---|
| 来源 | 阿照、狐影铃 |
| 用途 | 阻挡移动、吸收攻击/技能、支援、迷光面、镜尾符、断影香 |
| 解法 | 攻击、技能指定、破影令、断影香 |
| 结论 | 已统一为同一种假身规则 |

# 12. 下一步建议

1. 测试禁行令、戒严令、缉妖令三条律令的触发频率。
2. 测试燃尾令、灵狐羽、狐踪符的狐火消耗强度。
3. 若天门太强，减少律令获得威仪的频率。
4. 若青丘太强，提高狐火爆发的消耗或限制次数。
