# 《夕妖》MVP 规则总表

## 文档目的

本文档是《夕妖》MVP 当前规则的统一审计入口。

用途：

- 汇总全部地形、覆盖层、状态、战斗骰子、资源、角色技能、百宝囊。
- 检查规则是否闭环。
- 暴露当前文档中的矛盾、缺失和待确认设定。
- 作为后续 Codex / Cursor / 开发实现时的优先读取文档。

---

# 1. MVP 范围总览

## 当前 MVP 包含

- 2 支战帮：天门执法队、青丘使团。
- 6 套百宝囊。
- 7 个基础地形标签。
- 3 种覆盖层。
- 战斗骰子系统。
- 状态分类与友方净化规则。
- 3 个大回合。
- 气运胜利分。
- AI 对战。
- 教学战役与自由对战。

## 当前 MVP 不包含

- 完整四象天气系统。
- 圣坛。
- 昼曜令台。
- 窄路。
- 临时障碍。
- 临时黄昏裂隙。
- 关闭裂隙。
- 直接改变基础地形的技能或宝物。
- 联网天梯。
- 抽卡。
- 长期战帮养成。

# 3. 通用状态规则

## 3.1 正面状态

正面状态通常只能施加给友方单位，除非效果明确写明可以给敌方。

MVP 示例：

- 护盾。

## 3.2 负面状态

负面状态通常只能施加给敌方单位，除非效果明确写明可以给友方。

MVP 示例：

- 狐印。
- 封印。
- 迷踪。
- 易伤。
- 追缉。

## 3.3 友方净化

当前 MVP 的“净化”默认指友方净化。

友方净化只能选择友方单位，并移除该友方单位身上的 1 个负面状态。

因此白烬的晨辉净化不能净化敌方单位身上的封印。

这不是“封印不可被任何净化移除”，而是当前 MVP 没有能选择敌方单位并移除负面状态的能力。

## 3.4 后续扩展

后续战帮可以拥有“敌方净化”“驱散”“解除控制”等能力。

如果新能力明确允许选择敌方单位并移除其负面状态，则可以移除封印。

## 3.5 默认持续时间

所有未注明持续时间的单位状态，默认持续 1 回合。

持续 1 回合的含义：

- 生成或施加后立即生效。
- 在下一个结算阶段处理持续状态倒计时时移除。

若文档特别写明“下一次移动前”“下一次受到攻击前”等触发式持续时间，则按具体文本执行。

# 15. 当前待确认清单

当前 A 级与 B 级问题均已清零。

新增裁定已写入规则：

1. 封界石移出 MVP，天律囊替换为肃行令。
2. 肃行令不改变地形，不点名具体战帮技能，只基于“敌方单位在律令领域内开始激活”。
3. 状态分为正面状态与负面状态。
4. 负面状态默认只能施加给敌方单位。
5. 当前 MVP 的净化为友方净化，只能净化友方身上的负面状态。
6. 封印是负面状态；当前 MVP 中不会被白烬净化，但未来敌方净化/驱散能力可以移除封印。

# 17. 结论

当前 MVP 的核心已经统一为：

```text
青丘：狐火残留 → 收集狐火 → 狐印 / 假身 / 机动 → 边缘偷点或集火
天门：律令领域 → 威仪 → 追缉 / 封印 / 友方净化 → 中央压制和防御支援
```

夕灵保留为四象本源，不再作为青丘 MVP 战帮资源。

临时黄昏裂隙、直接改变基础地形的技能/宝物、封界石不进入 MVP。
