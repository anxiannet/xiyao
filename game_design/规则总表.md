# 基础资源

| 资源 | 类型 | 规则 |
|---|---|---|
| AP | 行动点 | 每名单位每次激活拥有2AP |
| 气运 | 胜利分 | 3个大回合后气运高者获胜 |
| 狐火 | 青丘资源 | 青丘共享资源，0/5；达到5点后进入狐火充盈 |

## 狐火

狐火是青丘共享战帮资源。

```text
狐火：0 / 5
```

获得方式：

- 青丘单位经过狐火残留：+1狐火。
- 青丘单位经过黄昏裂隙：+1狐火。
- 青丘单位命中带狐印目标：+1狐火。
- 假身参与青丘攻击支援，且本次攻击命中：+1狐火。

当狐火达到5点时，青丘获得狐火充盈。

狐火充盈期间，青丘不再获得狐火。

青丘下一次主动发起攻击时，在掷骰前自动触发狐火充盈：

- 本次攻击 ATK_DICE +1。
- 本次攻击 DMG +1。
- 无论命中与否，攻击结算后狐火归零。

狐火充盈不可选择保留。

设计目的：狐火参考连击点节奏，但不采用任意点数消费，改为攻击前自动终结，降低 AI 决策复杂度，并避免“成功后才增加攻击骰”的时序矛盾。

# 天门律令

玄照存活时，每个大回合开始时选择1条律令生效。

同一时间只能有1条律令。

| 律令 | 触发条件 | 效果 |
|---|---|---|
| 禁行令 | 敌方单位第一次进入边缘据点或黄昏裂隙 | 该单位获得迷踪 |
| 戒严令 | 敌方单位第一次占领中央据点 | 占领后在中央据点生成律令领域 |
| 缉妖令 | 天门单位本大回合第一次协同攻击同一个敌方单位 | 本次攻击 ATK_DICE +1 |

# 状态

| 状态 | 类型 | 效果 |
|---|---|---|
| 狐印 | 负面标记 | 青丘攻击该目标时支援面自动成功；命中狐印目标时青丘+1狐火 |
| 封印 | 负面控制 | 目标无法使用主动技能 |
| 迷踪 | 负面状态 | 下一次移动距离 -1 |
| 狐火充盈 | 资源状态 | 青丘下一次主动发起攻击时，掷骰前 ATK_DICE +1、DMG +1；结算后狐火归零 |

负面状态通常施加给敌方。

友方净化只能移除友方身上的负面状态。

狐火充盈不是负面状态，不能被净化移除。

# 地形

| 地形 | 规则 |
|---|---|
| plain 普通格 | 无特殊规则 |
| central_objective 中央据点 | 回合结束控制方 +2 气运 |
| edge_objective 边缘据点 | 回合结束控制方 +1 气运 |

| obstacle 障碍 | 不可进入，阻挡移动和直线视野 |

# 覆盖层

## 狐火残留

青丘覆盖层。

- 青丘经过：收集1点狐火。
- 敌方经过：获得迷踪。
- 可被敌方覆盖层覆盖。

## 律令领域

天门覆盖层。

- 天门单位在其中 DEF_DICE +1。
- 天门单位在其中防御时，支援面自动成功。
- 敌方经过时，获得迷踪。
- 可覆盖敌方有害覆盖层。

## 假身

青丘召唤物。

- 占格。
- 阻挡移动。
- 不阻挡视线。
- 可吸收1次攻击或技能。
- 可参与攻击支援和防御支援。
- 不占领据点。
- 假身参与青丘攻击支援且本次攻击命中时，青丘+1狐火。

# 战斗骰子

行动骰共有6面：

```text
暴击 ×1
攻击 ×2
防御 ×2
支援 ×1
```

攻击时：暴击、攻击、满足条件的支援算成功。

防御时：暴击、防御、满足条件的支援算成功。

攻击者本人不提供攻击支援。

---

# 胜负

一局共3个大回合。

气运来源：

- 控制中央据点：+2
- 控制边缘据点：+1
- 击败敌方成员：+1
- 击败敌方队长：+2

同分时：

1. 存活单位更多者胜。
2. 队长存活者胜。
3. 控制中央据点者胜。
