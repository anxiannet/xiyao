# 《夕妖》MVP 规则总表

## 文档目的

本文档是《夕妖》MVP 当前规则的统一审计入口。

用途：

- 汇总全部地形、覆盖层、状态、战斗骰子、资源、角色技能、百宝囊。
- 检查规则是否闭环。
- 暴露当前文档中的矛盾、缺失和待确认设定。
- 作为后续 Codex / Cursor / 开发实现时的优先读取文档。

---

# 1. MVP 范围总览

## 当前 MVP 包含

- 2 支战帮：天门执法队、青丘使团。
- 6 套百宝囊。
- 7 个基础地形标签。
- 3 种覆盖层。
- 战斗骰子系统。
- 3 个大回合。
- 气运胜利分。
- AI 对战。
- 教学战役与自由对战。

## 当前 MVP 不包含

- 完整四象天气系统。
- 圣坛。
- 昼曜令台。
- 窄路。
- 临时障碍。
- 临时黄昏裂隙。
- 关闭裂隙。
- 直接改变基础地形的技能或宝物。
- 联网天梯。
- 抽卡。
- 长期战帮养成。

# 2. 四象与命名总表

## 四象本源

```text
昼曜
夜魄
晨辉
夕灵
```

对应关系：

```text
昼 ↔ 夜
晨 ↔ 夕
曜 ↔ 魄
辉 ↔ 灵
```

## 命名层级

| 名词 | 类型 | 当前定义 | 是否 MVP 规则资源 |
|---|---|---|---|
| 夕灵 | 四象本源 | 黄昏、变化、灵性、化形与过渡的本源 | 否 |
| 狐火 | 青丘战帮资源 | 青丘通过收集狐火残留获得的资源 | 是 |
| 狐火残留 | 地格覆盖层 | 地面上的狐火痕迹，青丘经过时收集狐火 | 是 |
| 狐印 | 单位状态 | 青丘施加的标记，攻击带狐印目标时支援自动成功 | 是 |
| 威仪 | 天门战帮资源 | 天门通过律令、封印、清理残留、中央据点获得的资源 | 是 |
| 律令领域 | 地格覆盖层 | 天门法度形成的阵地覆盖层 | 是 |

不得再使用：

- 暮炁。
- 夕火作为资源名。
- 夕灵值作为 MVP 青丘资源。
- 狐火状态。

# 7. 战斗骰子总表

## 7.1 骰面

| 骰面 | 数量 | 攻击时 | 防御时 |
|---|---:|---|---|
| 暴击 | 1 | 成功 | 成功 |
| 攻击 | 2 | 成功 | 失败 |
| 防御 | 2 | 失败 | 成功 |
| 支援 | 1 | 满足攻击支援则成功 | 满足防御支援则成功 |

## 7.2 攻击支援

| 项目 | 规则 |
|---|---|
| 检查位置 | 被攻击目标周围 1 格 |
| 成功条件 | 至少 1 个攻击方单位在目标周围，且该单位不是本次攻击者 |
| 攻击者本人 | 不计入攻击支援 |
| 假身 | 可以计入攻击支援 |
| 狐印自动支援 | 青丘攻击带狐印目标时，支援面自动成功 |

## 7.3 防御支援

| 项目 | 规则 |
|---|---|
| 检查位置 | 防御者周围 1 格 |
| 成功条件 | 至少 1 个防御方友军在防御者周围 |
| 防御者本人 | 不计入防御支援 |
| 假身 | 可以计入防御支援 |
| 律令领域自动支援 | 天门单位位于律令领域内防御时，支援面自动成功 |

## 7.4 结算

| 阶段 | 规则 |
|---|---|
| 暴击抵消 | 攻击暴击只能由防御暴击抵消 |
| 普通成功抵消 | 暴击抵消后比较普通成功数 |
| 命中条件 | 攻击方剩余成功数大于防御方剩余成功数 |
| 防住条件 | 攻击方剩余成功数小于或等于防御方剩余成功数 |
| 默认伤害 | 命中后造成基础伤害 |
| 额外成功伤害 | MVP 不默认采用“每多 1 成功 +1 伤害” |

# 16. 结论

当前 MVP 的核心已经统一为：

```text
青丘：狐火残留 → 收集狐火 → 狐印 / 假身 / 机动 → 边缘偷点或集火
天门：律令领域 → 威仪 → 追缉 / 封印 / 净化 → 中央压制和防御支援
```

夕灵保留为四象本源，不再作为青丘 MVP 战帮资源。

临时黄昏裂隙与直接改变基础地形的技能/宝物不进入 MVP。
