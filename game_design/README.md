# 当前 MVP 范围

- 2 支小队：青丘使团、天门执法队
- 每方 3 名单位
- 每名单位只保留 1 个核心技能
- 小型六角格地图
- 7 个基础地形
- 覆盖层与区域：狐火残留、假身、勘验区
- 状态：狐印、封印、迷踪
- 资源：气运、狐火、AP
- 律令系统
- 战斗骰子系统
- 占点系统
- 5 个教学关卡
- 自由对战
- AI 对战

---

# MVP 裁剪原则

1. 先保证 15 分钟一局成立。
2. 先验证 2 支小队是否有清晰差异。
3. 先让 AI 能完整跑完一局。
4. 每名英雄只保留一个核心技能。
5. 地形、状态、资源、律令和骰子负责战术变化。
6. 百宝锦囊、宝物、妙计、阵型、法术、化形、天门威仪、命运系统暂不进入 MVP。
