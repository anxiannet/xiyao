# MVP英雄原则

每名英雄只保留一个核心技能。

核心技能必须体现：

- 英雄身份
- 小队职责
- 战帮中的不可替代性

英雄负责身份识别。

百宝囊、地形、状态、资源和骰子负责战术变化。

# 人物写作原则

人物志要先解释新名词，再写人物经历。

涉及“裂隙暴走”“黄昏裂隙”“夕子感知”等新设定时，必须先有词条或在正文中即时解释。

人物小传优先写：

- 他小时候发生过什么
- 为什么变成现在这样
- 别人怎么看他
- 他怎么学会核心能力
- 他和队友是什么关系

避免只写“获得某能力”“掌握某技能”。

# 命名规范

- 夕子：四象本源之一
- 夕灵：能感知、接触或运用夕子的生灵
- 夕妖：天庭对夕灵的统称
- 狐火：青丘战帮资源
- 狐印：青丘负面标记状态
- 狐火残留：青丘覆盖层
- 威仪：天门战帮资源
- 律令：天门队长玄照颁布的战场规则
- 封印：负面控制状态
- 律令领域：天门覆盖层
