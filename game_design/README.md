# MVP英雄原则

每名英雄只保留一个核心技能。

核心技能必须体现：

- 英雄身份
- 小队职责
- 战帮中的不可替代性

英雄负责身份识别。

百宝囊、地形、状态、资源和骰子负责战术变化。

# 命名规范

- 夕子：四象本源之一
- 夕灵：能感知、接触或运用夕子的生灵
- 夕妖：天庭对夕灵的统称
- 狐火：青丘战帮资源
- 狐印：青丘负面标记状态
- 狐火残留：青丘覆盖层
- 威仪：天门战帮资源
- 追缉：天门负面标记状态
- 封印：负面控制状态
- 律令领域：天门覆盖层
