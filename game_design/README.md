# MVP 范围

当前 MVP 锁定为：

- 2支战帮：天门执法队、青丘使团
- 每名英雄 1 个核心技能
- 6套百宝囊
- 7个基础地形标签
- 程序生成六角网格
- 地形标签与覆盖资源层
- 核心标签系统
- 狐火残留 / 律令领域 / 假身等覆盖层
- 狐火资源与狐印状态
- 追缉、封印、友方净化、护盾等状态机制
- 战斗骰子系统
- 5个教学关卡
- 自由对战
- AI 对战

不包含：

- 化形
- 化形条件
- 化形技能
- 百宝锦囊
- 妙计
- 阵型
- 法术
- 六势力完整扩展
- 长期经济系统
- 赛季系统
- 命运系统
- 联网天梯
- 抽卡
- 大规模战帮养成
- 临时黄昏裂隙
- 直接改变基础地形的技能或宝物

# MVP 英雄原则

MVP 中，每名英雄只保留一个核心技能。

核心技能必须体现：

- 英雄身份。
- 小队背景故事中的职责。
- 该英雄在战帮中的不可替代性。

MVP 不通过多技能和化形塑造英雄。

战术变化主要来自：

- 地形。
- 覆盖层。
- 状态。
- 战斗骰子。
- 百宝囊。
- 战帮资源。

# 百宝囊与未来百宝锦囊

MVP 使用“百宝囊”。

百宝囊只包含宝物。

未来版本可升级为“百宝锦囊”。

百宝锦囊可包含：

- 宝物
- 妙计
- 阵型
- 法术
- 令牌
- 事件
- 机关
- 盟友支援

# 命名与扩展规范

当前统一使用：

- 夕灵：四象本源之一，不作为 MVP 青丘战帮资源
- 狐火：青丘战帮资源
- 狐印：青丘负面标记状态
- 狐火残留：地格覆盖层，可被青丘收集为狐火
- 威仪：天门战帮资源
- 追缉：天门负面标记状态
- 封印：负面控制状态
- 律令领域：天门覆盖层

扩展规则：

- 新战帮必须先定义队长机制、战帮资源、核心状态或覆盖层。
- 新状态、覆盖层、召唤物必须优先配置标签。
- 新技能和宝物应优先引用标签，例如负面状态、标记状态、召唤物、有害覆盖层。
- 不得在通用技能中硬编码狐印、追缉、假身、狐火残留等具体名词，除非该技能属于对应战帮主题。

不得再使用：

- 暮炁
- 夕火（作为资源名）
- 暮炁囊
- 暮灯
- 狐火状态
