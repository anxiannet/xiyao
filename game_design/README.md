# MVP 范围

当前 MVP 锁定为：

- 2支战帮：天门执法队、青丘使团
- 6套百宝囊
- 程序生成六角网格
- 地形标签与覆盖资源层
- 狐火残留 / 律令领域等覆盖层
- 骰子攻防系统
- 5个教学关卡
- 自由对战
- AI 对战

不包含：

- 六势力完整扩展
- 长期经济系统
- 赛季系统
- 命运系统
- 联网天梯
- 抽卡
- 大规模战帮养成

# 命名规范

当前统一使用：

- 夕灵：黄昏本源资源
- 狐火：青丘状态
- 狐火残留：地格覆盖层
- 威仪：天门资源
- 律令领域：天门覆盖层

不得再使用：

- 暮炁
- 夕火（作为资源名）
- 暮炁囊
- 暮灯
