# 青丘使团

## 小队定位

4 人机动小队。

青丘使团代表夕灵余火、变化、狐火残留、假身、边缘偷点与黄昏路线。

MVP 中与天门执法队形成强对比：

- 天门：3 人精英，正面推进，依靠律令、勘验区和护阵压缩路线。
- 青丘：4 人机动，绕后拉扯，依靠狐火残留与假身制造路线优势。

---

# 小队背景

青丘使团原本不是战斗小队。

他们是为了护送一枚“暮君残片”离开青丘谷而临时组成的逃亡队伍。

天庭称他们为妖乱源头。

青丘内部知道：

暮君残片不是邪物。

它是一段被封印的历史。

苏绫带队离开青丘，不是为了作乱，而是为了把真相送到暮渊裂谷。

# 小队成员

## 苏绫

队长。

核心技能：狐步。

职责：带领队伍穿过黄昏、掩影与边界，制造狐印并发动围猎。

特别之处：青丘唯一稳定狐印来源。

## 阿照

成员。

核心技能：假身。

职责：制造替身，阻挡路线，吸收攻击或封印。

特别之处：青丘唯一稳定假身来源。

## 青萝

成员。

核心技能：狐火引路。

职责：为同伴指路，在路径上留下狐火残留。

特别之处：青丘唯一稳定狐火残留铺场者。

## 琉尾

成员。

核心技能：扰弦。

职责：远程断后，压制据点，利用狐印与狐火充盈制造输出窗口。

特别之处：青丘唯一远程压制单位。

---

# 小队核心能力

## 狐印围猎

苏绫是狐印的唯一稳定来源。

青丘攻击带狐印目标时，支援面自动成功。

## 狐火路线

青丘通过狐火残留和黄昏裂隙收集狐火。

狐火不是按钮收益，而是地图路线收益。

## 假身支援

假身是青丘召唤物。

假身阻挡移动，不阻挡视线，可吸收一次攻击或技能，并参与支援。

## 不正面对抗

青丘使团不适合硬打中央据点。

他们通过边缘据点、假身、狐火残留和裂隙逼迫天门分兵。
