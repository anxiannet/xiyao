# 天门执法队

## 小队定位

3人精英小队。

天门执法队代表昼曜、律令、勘验、护阵与正面推进。

在 MVP 中，它是青丘使团的主要对照小队：

- 青丘依靠狐火残留、掩影、裂隙与假身制造路线和目标变化。
- 天门依靠律令、勘验区、律令领域和护阵压缩敌方行动空间。

---

# 天门执法链

天门执法队并非传统战斗小队。

他们的行动遵循：

律
↓
勘
↓
罚

玄照负责颁布律令。

白烬负责勘验异常区域。

赤霄负责执行与压制。

天门的核心不是消灭敌人。

而是恢复秩序。

# 队长机制：律令

当前 MVP 律令：

## 禁行令

本回合严查边界。

效果：

敌方单位第一次进入：
- 边缘据点
- 黄昏裂隙

获得迷踪。

天门单位位于黄昏裂隙相邻格时：

支援面自动成功。

## 戒严令

中央区域进入戒严。

效果：

中央据点生成律令领域。

天门单位位于中央据点时：

DEF_DICE +1。

## 追捕令

本回合优先追捕逃逸目标。

效果：

天门单位攻击本回合移动过的敌人时：

支援面自动成功。
