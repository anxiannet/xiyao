# 《夕妖》Alpha 战帮详表 V1

## 核心原则

Alpha 阶段玩家选择的是完整战帮。

不是选择单个英雄。

每个战帮都是一个独立玩法包，包含：

- 队长
- 成员
- 战帮被动
- 化形方向
- 战术定位
- AI使用逻辑

Alpha 目标：

6个势力，6个战帮，18个单位。

---

# 1. 天巡司：天门执法队

## 战帮定位

3人精英控制战帮。

特点：

- 封印
- 防御
- 净化
- 正面占点

四象倾向：昼曜 / 晨辉

适合玩家：新手、中速控制玩家。

## 战帮被动：天门律令

当天象为永昼或昼曜值最高时，天门执法队第一次封印效果额外持续1回合。

## 单位1：玄照（队长）

定位：前排控制

初始形态：天门执法官

化形形态：昼曜神将

普通能力：律令封禁

效果：封印相邻敌方单位1个主动技能1回合。

化形条件：成功封印2次，或占领昼曜据点1次。

化形能力：天门审判

效果：对目标造成伤害，并附加封印。

队长光环：玄照存活时，友方单位受到的第一次控制效果无效。

## 单位2：白烬（成员）

定位：辅助净化

初始形态：晨辉神官

化形形态：净世圣使

普通能力：晨辉净化

效果：移除友方1个负面状态，并回复1点生命。

化形条件：累计治疗或净化3次。

化形能力：圣辉环护

效果：为相邻友方提供护盾。

## 单位3：赤霄（成员）

定位：近战护卫

初始形态：天门卫士

化形形态：赤霄天卫

普通能力：护阵

效果：替相邻友方承受一次攻击。

化形条件：成功护阵2次，或生命低于一半仍存活到回合结束。

化形能力：不退之门

效果：本回合无法被击退，且占点权重+1。

# 2. 镇妖军：北境守军

## 战帮定位

3人均衡战帮。

特点：

- 近战
- 远程
- 防守
- 新手友好

四象倾向：昼曜

适合玩家：第一局体验。

## 战帮被动：军阵

当三名单位彼此距离不超过2格时，全队防御提升。

## 单位1：沈戈（队长）

定位：均衡战士

初始形态：镇妖校尉

化形形态：护国战将

普通能力：镇妖斩

效果：普通攻击后，可推动目标1格。

化形条件：击败1名敌人，或完成1次据点占领。

化形能力：破阵斩

效果：攻击一条直线上的首个敌人，并击退。

队长光环：沈戈存活时，北境守军占领据点时额外获得1点军阵值。

## 单位2：柳知微（成员）

定位：远程方士

初始形态：随军方士

化形形态：灵阵术师

普通能力：符阵

效果：在目标格放置符阵，敌人进入时受到减速。

化形条件：技能命中或触发2次。

化形能力：三才阵

效果：指定区域内友方获得防御，敌方移动降低。

## 单位3：石山（成员）

定位：前排盾卫

初始形态：北境盾兵

化形形态：铁壁卫

普通能力：举盾

效果：本回合受到伤害降低。

化形条件：累计承受3点伤害后仍存活。

化形能力：铁壁推进

效果：向前移动并推开路径上的敌人。

# 3. 青丘狐族：青丘使团

## 战帮定位

3人机动幻术战帮。

特点：

- 位移
- 假身
- 偷点
- 黄昏爆发

四象倾向：夕火 / 夜魄

适合玩家：喜欢操作、绕后、骗技能的玩家。

## 战帮被动：狐影

黄昏天象下，青丘单位第一次移动后可额外移动1格。

## 单位1：苏绫（队长）

定位：机动刺客

初始形态：青丘狐女

化形形态：九尾灵狐

普通能力：狐步

效果：移动后可攻击相邻敌人。

化形条件：在黄昏天象中移动2次，或击败1名敌人。

化形能力：九尾掠影

效果：连续穿越2格并攻击路径终点敌人。

队长光环：苏绫存活时，青丘单位占领边缘据点时额外获得夕火。

## 单位2：阿照（成员）

定位：幻术干扰

初始形态：幻形少年

化形形态：千面狐影

普通能力：假身

效果：在相邻格生成一个假身。

化形条件：假身承受1次攻击。

化形能力：千面迷阵

效果：交换一个友方单位与假身的位置。

## 单位3：青萝（成员）

定位：占点支援

初始形态：青丘灵侍

化形形态：狐火灵使

普通能力：狐火引路

效果：为友方单位提供移动加成。

化形条件：友方单位因其能力完成占点或击败。

化形能力：夕火花径

效果：连接两格，友方可视为相邻移动。

# 4. 苍狼部族：啸月猎团

## 战帮定位

3人追击爆发战帮。

特点：

- 猎印
- 残血收割
- 高机动追杀
- 夜魄强化

四象倾向：夜魄 / 夕火

适合玩家：进攻型玩家。

## 战帮被动：猎印

攻击敌人后为目标添加猎印。

苍狼单位攻击带猎印目标时获得加成。

## 单位1：骨牙（队长）

定位：近战爆发

初始形态：苍狼猎手

化形形态：啸月狼将

普通能力：裂牙斩

效果：对带猎印目标造成额外伤害。

化形条件：攻击生命低于一半的敌人2次。

化形能力：啸月追杀

效果：攻击后若目标未死亡，可再移动1格追击。

队长光环：骨牙存活时，猎印不会在回合结束自动消失。

## 单位2：乌岚（成员）

定位：萨满辅助

初始形态：狼族萨满

化形形态：夜嚎祭司

普通能力：狼魂鼓舞

效果：为友方添加攻击加成。

化形条件：为队友施加2次增益。

化形能力：夜嚎

效果：全体带猎印敌人移动降低。

## 单位3：灰尾（成员）

定位：游击追踪

初始形态：雪原斥候

化形形态：灰影狼卫

普通能力：追踪

效果：移动至带猎印敌人附近时不消耗额外移动。

化形条件：参与击败1名带猎印敌人。

化形能力：断后

效果：攻击后使目标无法远离自己。

# 5. 星宫：观星者

## 战帮定位

3人战术控制战帮。

特点：

- 预判
- 远程
- 天象调整
- 控制节奏

四象倾向：均衡

适合玩家：喜欢算牌、控局、慢速运营的玩家。

## 战帮被动：星轨

每个大回合开始时，可预览下一次天象倾向。

## 单位1：辰无咎（队长）

定位：预判控制

初始形态：观星官

化形形态：执命星君

普通能力：观命

效果：标记一个敌方单位。若其下次行动为移动，则受到减速。

化形条件：成功触发1次预判效果。

化形能力：改命

效果：改变一个单位的行动顺序。

队长光环：辰无咎存活时，星宫每回合第一次预判失败不消耗资源。

## 单位2：洛微尘（成员）

定位：远程术法

初始形态：星术师

化形形态：星河行者

普通能力：星落

效果：远程攻击目标格。

化形条件：非相邻距离命中2次。

化形能力：星河断线

效果：攻击并切断目标与最近友方的协同效果1回合。

## 单位3：司南（成员）

定位：天象辅助

初始形态：星盘童子

化形形态：天象司仪

普通能力：拨星

效果：小幅调整四象能量。

化形条件：成功触发2次四象共鸣。

化形能力：转象

效果：强制将当前天象向指定四象偏移。

# 6. 暮渊遗民：余烬小队

## 战帮定位

3人高风险夕火战帮。

特点：

- 自损
- 诅咒
- 低血爆发
- 夕火暴走

四象倾向：夕火 / 夜魄

适合玩家：高风险高收益玩家。

## 战帮被动：余烬

当单位生命低于一半时，获得额外夕火。

## 单位1：灰阎（队长）

定位：高风险战士

初始形态：余烬战士

化形形态：暮火狂徒

普通能力：燃血刃

效果：自损1点，对敌人造成额外伤害。

化形条件：生命低于一半。

化形能力：暮火斩

效果：根据已损失生命提高伤害。

队长光环：灰阎存活时，余烬单位低血时不会因第一次自损效果阵亡。

## 单位2：小满（成员）

定位：诅咒支援

初始形态：遗民少女

化形形态：暮渊咒女

普通能力：裂痕咒

效果：给敌人施加易伤。

化形条件：敌方身上存在2个负面状态。

化形能力：咒火回响

效果：引爆目标身上的负面状态造成伤害。

## 单位3：残灯（成员）

定位：生存辅助

初始形态：残灯守夜人

化形形态：余火引路者

普通能力：守灯

效果：为低血友方提供护盾。

化形条件：成功保护低血友方1次。

化形能力：余火不灭

效果：让一名友方在本回合受到致命伤害时保留1点生命。

# Alpha 平衡目标

每个战帮都必须拥有：

- 1个明确得分方式
- 1个主动进攻手段
- 1个防守或保命工具
- 1个化形节奏
- 1个AI可理解的核心策略

---

# AI使用简表

天门执法队：优先占点与封印。

北境守军：保持阵型，稳定推进。

青丘使团：优先边缘据点与绕后。

啸月猎团：集中攻击带猎印目标。

观星者：预判敌方高价值单位。

余烬小队：低血时主动换伤。
