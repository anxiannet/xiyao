# 《夕妖》未来扩展规划 V1

## 文档目的

本文档保存不进入 MVP、但已经确认方向的未来开发内容。

当前原则：

- MVP 先保证可玩闭环。
- 未来系统先记录概念，不进入当前实现。
- 不在 MVP 文档中保留会影响程序实现的半成品规则。

---

# 1. 化形系统

## 1.1 当前决议

化形不进入 MVP。

MVP 中：

- 没有化形条件。
- 没有化形形态。
- 没有化形技能。
- 没有化形数值成长。
- 没有化形动画需求。

## 1.2 未来方向

未来化形系统可以作为中后期版本内容。

设计原则：

- 化形不应让英雄获得一套完全无关的新技能。
- 化形应强化英雄原本的核心身份。
- 每名英雄仍应围绕一个核心机制展开。

示例方向：

- 苏绫：强化狐步与狐印。
- 阿照：强化假身数量、持续或支援价值。
- 青萝：强化狐火残留与路线经营。
- 琉尾：强化远程压制与断后能力。
- 玄照：强化追缉与执法压迫。
- 白烬：强化友方净化、治疗、护盾。
- 赤霄：强化护阵、占点与承伤。

## 1.3 未来触发方式

未来可探索：

- 战帮资源触发。
- 地形条件触发。
- 剧情关卡触发。
- 限时活动测试。
- 百宝锦囊中的特殊妙计或法术触发。

当前不定具体规则。

# 2. 百宝锦囊系统

## 2.1 当前决议

MVP 阶段只使用“百宝囊”。

百宝囊内容只包含宝物。

未来版本可升级为“百宝锦囊”。

## 2.2 百宝锦囊定位

百宝锦囊是战帮战术模块的高级版本。

它不只是装宝物，而是包含多种战术组件。

未来可包含：

- 宝物
- 妙计
- 阵型
- 法术
- 令牌
- 事件
- 机关
- 盟友支援

## 2.3 分类建议

### 宝物

偏一次性物件效果。

示例：狐火珠、昼曜令、破影令。

### 妙计

偏战术反应、时机操作、临场变招。

示例方向：

- 被攻击前换位。
- 对方占点时反制。
- 回合结束前追加移动。

### 阵型

偏开局部署、站位奖励或持续阵地风格。

示例方向：

- 天门三才阵。
- 青丘散影阵。
- 高台压制阵。
- 边缘游猎阵。

### 法术

偏消耗资源、制造状态或改变战斗节奏。

示例方向：

- 晨辉护佑。
- 夕灵幻幕。
- 夜魄梦咒。
- 昼曜裁令。

## 2.4 与 MVP 的关系

MVP 当前百宝囊仍然保留：

- 天律囊
- 镇妖囊
- 巡天囊
- 狐影囊
- 狐火囊
- 迷踪囊

但这些只作为“宝物集合”。

未来升级百宝锦囊时，可以把当前宝物重新分层：

```text
百宝锦囊
├─ 宝物
├─ 妙计
├─ 阵型
└─ 法术
```

# 3. 未来开发原则

未来新增系统必须遵守：

1. 不破坏 MVP 的核心可玩闭环。
2. 不让单个英雄拥有过多技能。
3. 不让百宝锦囊完全替代英雄职责。
4. 优先使用 `core_tags.md` 中的标签系统。
5. 新战帮先定义队长机制、战帮资源、核心状态或覆盖层。
6. 新系统先通过限时活动或实验模式测试。

---

# 4. 暂不实现清单

- 化形。
- 百宝锦囊。

- 阵型。
- 法术。
- 命运系统。
- 长期养成。
- 联网天梯。
- 抽卡。
- 大型势力扩展。
