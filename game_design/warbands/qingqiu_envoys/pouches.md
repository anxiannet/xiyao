# 青丘使团：百宝囊

## 核心原则

百宝囊决定战术风格。

青丘使团依靠狐火残留收集狐火，再用狐火打出快速移动、连击和爆发。

苏绫负责制造狐印，百宝囊负责把狐火与地形优势转化为行动窗口。

---

# 百宝囊一：狐影囊

## 战术定位

假身协同流。

目标：用假身阻挡路线、吸收攻击或封印，并提供支援协同。

## 宝物列表

### 1. 狐影铃

类型：召唤物宝物

消耗：1 AP

效果：在相邻格生成 1 个使用者的假身。

假身复制使用者的外观和名称，不复制技能、数值、资源或状态。

阿照生成的假身与狐影铃生成的假身使用同一套规则，只是外观来源不同。

地形联动：若在掩影使用，假身持续 +1 回合。

### 2. 镜尾符

类型：伪装宝物

消耗：0 AP

效果：选择一个假身，使其显示为相邻友方单位的外观，直到被攻击、被技能指定或回合结束。

该效果只改变外观，不复制技能、状态、数值或资源。

### 3. 迷光面

类型：反指定宝物

消耗：0 AP，每大回合限 1 次

效果：本回合第一个成为单体指定目标的青丘单位，可立即与相邻假身交换位置。

可响应：单体攻击、单体技能、单体宝物。

不可响应：范围效果、地格效果、全体效果。

### 4. 断影香

类型：覆盖层宝物

消耗：1 AP

效果：清除一个友方假身，并在原地留下狐火残留。

# 百宝囊二：狐火囊

## 战术定位

狐火资源爆发流。

目标：通过狐火残留和黄昏裂隙收集狐火，再用狐火完成快速移动、连击或爆发。

## 宝物列表

### 1. 狐火灯

类型：覆盖层宝物

消耗：1 AP

距离：1

效果：选择一个非障碍格，生成狐火残留。

若该格在黄昏裂隙旁，狐火残留持续 +1 回合。

### 2. 暮君残片

类型：风险收益宝物

消耗：0 AP，每局限 1 次

效果：选择一个友方单位。该单位受到 1 点伤害，然后可立即移动 1 格。

若这次移动经过狐火残留或黄昏裂隙，正常收集狐火。

若该伤害导致目标被击败，不计为敌方击败，敌方不获得气运。

### 3. 狐火珠

类型：覆盖层宝物

消耗：1 AP

距离：2

效果：选择一个非障碍格，在该格生成狐火残留。

可选择空格、己方单位所在格、敌方单位所在格。

若该格已有敌方覆盖层，移除敌方覆盖层后生成狐火残留。

若该格已有狐火残留，刷新持续时间。

若放在边缘格、掩影格或黄昏裂隙旁，持续时间 +1 回合。

### 4. 燃尾令

类型：狐火爆发宝物

消耗：0 AP，每大回合限 1 次

效果：选择一个青丘单位。本回合该单位可以消耗狐火获得以下效果之一：

- 消耗 1 狐火：本次移动 MV +1。
- 消耗 1 狐火：攻击后移动 1 格。
- 消耗 2 狐火：本次攻击 ATK_DICE +1。

限制：每次使用燃尾令只能选择一种效果。

# 百宝囊三：迷踪囊

## 战术定位

边缘偷点流。

目标：绕过敌方中央防线，利用边缘据点、掩影和迷踪持续拉扯。

## 宝物列表

### 1. 影遁符

类型：移动宝物

消耗：1 AP

效果：移动到相邻掩影格。

### 2. 灵狐羽

类型：移动宝物

消耗：0 AP，每大回合限 1 次

效果：移动后额外移动 1 格。

若额外消耗 1 狐火，可再额外移动 1 格。

### 3. 迷途香

类型：负面状态宝物

消耗：1 AP

距离：2

效果：指定一个敌方目标，施加迷踪。

迷踪：目标下一次移动距离 -1。

### 4. 狐踪符

类型：支援强化宝物

消耗：0 AP，每大回合限 1 次

效果：选择一个本回合带有迷踪的敌方单位。青丘单位攻击该目标时，支援面自动成功，直到本大回合结束。

若额外消耗 1 狐火，本次攻击命中后造成 +1 伤害。

# 宝物背景

| 宝物 | 背景 |
|---|---|
| 狐影铃 | 青丘幻狐祭器，原本用于训练幼狐辨别真身与幻身。 |
| 镜尾符 | 以狐尾影纹写成的幻符，可让假身披上同伴外观。 |
| 迷光面 | 青丘逃亡者常戴的半面狐面，可在被锁定瞬间借假身换位。 |
| 断影香 | 燃烧后能让幻身散成狐火残留的香。 |
| 狐火灯 | 青丘夜行灯，用来照见夕子残痕。 |
| 暮君残片 | 记录旧史的残片，接触者会受伤，但也能看见隐秘路径。 |
| 狐火珠 | 封存狐火残留的珠器，投出后可在地面留下狐火痕迹。 |
| 燃尾令 | 青丘古令，允许狐族短时燃起尾火换取速度或爆发。 |
| 影遁符 | 青丘斥候使用的短符，可借掩影一步脱离追捕。 |
| 灵狐羽 | 取自灵狐尾羽的轻符，能让脚步短暂变轻。 |
| 迷途香 | 使追兵误判路径的香。 |
| 狐踪符 | 用来锁定迷路目标脚印的追踪符。 |
}
