# 青丘使团（Qingqiu Envoys）

## 战帮定位

4人机动战帮。

代表夕火、变化、狐火、假身、边缘偷点与黄昏爆发。

MVP 中与天门执法队形成强对比：

- 天门：3人精英，正面推进。
- 青丘：4人机动，绕后拉扯。

---

# 战帮背景

青丘使团原本不是战斗小队。

他们是为了护送一枚“暮君残片”离开青丘谷而临时组成的逃亡队伍。

天庭称他们为妖乱源头。

青丘内部知道：

暮君残片不是邪物。

它是一段被封印的历史。

苏绫带队离开青丘，不是为了作乱，而是为了把真相送到暮渊裂谷。

---

# 战帮成员

- 苏绫：队长，机动刺客，黄昏爆发核心。
- 阿照：成员，假身干扰，诱骗封印。
- 青萝：成员，移动支援，狐火连通。
- 琉尾：成员，远程骚扰，高台击退。

---

# 战帮核心能力

## 狐影

青丘单位擅长利用林地、边缘格、黄昏裂隙制造位移与假目标。

## 不正面对抗

青丘使团不适合硬打中央据点。

他们通过边缘据点、假身、狐火和裂隙逼迫天门分兵。

---

# 推荐百宝囊

- 狐影囊：假身扰阵。
- 夕火囊：化形爆发。
- 迷踪囊：边缘偷点。
