# 天门执法队：百宝囊

## 核心原则

百宝囊决定战术风格。

天门执法队的核心是律令、威仪、封印、阵地和执法推进。

玄照负责颁布律令，改变本大回合的战场规则。

百宝囊负责把律令优势转化为控图、封印、移动和攻击窗口。

---

# 百宝囊一：天律囊

## 战术定位

律令领域控图流。

目标：用律令领域覆盖敌方有害覆盖层，压缩敌方路线，并让天门在阵地中获得稳定防守收益。

## 宝物列表

### 1. 昼曜令

类型：覆盖层宝物

消耗：1 AP

距离：2

效果：指定一个非障碍格，生成律令领域，持续 1 回合。

若该格存在敌方有害覆盖层，先清除该覆盖层，再生成律令领域。

### 2. 律令旗

类型：阵地宝物

消耗：1 AP

效果：选择己方控制的中央据点。该格和相邻 1 格生成律令领域，持续 1 回合。

限制：只能以中央据点为中心使用。

### 3. 肃行令

类型：控场宝物

消耗：0 AP，每大回合限 1 次

触发：敌方单位在律令领域内开始激活时。

效果：对该敌方单位施加迷踪。

迷踪：目标下一次移动距离 -1。

### 4. 裁界印

类型：阵地压制宝物

消耗：1 AP

效果：选择一个位于律令领域内的敌方单位。本大回合天门第一次攻击该单位时，ATK_DICE +1。

说明：裁界印不施加易伤，只制造一次阵地攻击窗口。

# 百宝囊二：镇妖囊

## 战术定位

封印抓核心流。

目标：利用律令、中央据点和威仪制造封印窗口，压制敌方核心技能。

## 宝物列表

### 1. 镇妖钉

类型：控制宝物

消耗：1 AP

距离：2

效果：指定一个敌方单位。若满足以下任一条件，施加封印，持续 1 回合：

- 目标位于律令领域内。
- 本大回合已触发过天门律令。
- 目标带有敌方标记状态。

地形联动：从高台使用时，距离 +1。

说明：狐印属于标记状态，因此青丘被自己狐印体系暴露时，也可能被镇妖钉抓住。

### 2. 破影令

类型：反召唤物宝物

消耗：1 AP

距离：2

效果：指定一个敌方召唤物，将其清除。

清除后，天门获得 1 威仪。

若该召唤物位于掩影，额外在该格生成律令领域，持续 1 回合。

### 3. 天罚令

类型：攻击强化宝物

消耗：1 威仪

效果：本次天门攻击 ATK_DICE +1。

若本大回合已触发过天门律令，本次攻击额外 ATK_DICE +1。

若本次攻击掷出暴击，暴击额外造成 +1 伤害。

### 4. 锁妖令

类型：控制宝物

消耗：1 AP

距离：1

效果：指定相邻敌方单位。

若目标位于律令领域，施加封印，持续 1 回合，并使其下一次移动距离 -1。

若目标不在律令领域，本宝物不能使用。

# 百宝囊三：巡天囊

## 战术定位

追捕机动流。

目标：补足天门移动慢的问题，使天门能回应边缘据点、裂隙绕后和敌方拉扯。

## 宝物列表

### 1. 巡天符

类型：移动宝物

消耗：1 AP

效果：选择一个天门单位。本回合该单位 MV +1。

若目标移动终点为律令领域、高台或中央据点，天门获得 1 威仪。

### 2. 天门索

类型：拉拽宝物

消耗：1 AP

距离：2

效果：将一个敌方单位朝最近的天门单位方向拉动 1 格。

限制：

- 目标必须位于律令领域、黄昏裂隙旁、边缘据点或掩影。
- 不能把目标拉入被占据格。
- 不能把目标拉出地图。
- 若存在多个最近的天门单位，由使用者选择其中一个作为拉动方向。

### 3. 踏令步

类型：移动宝物

消耗：0 AP，每大回合限 1 次

效果：选择一个位于律令领域上的天门单位。

该单位可移动 1 格。

若移动后仍位于律令领域或中央据点，获得 1 威仪。

### 4. 巡边令

类型：反偷点宝物

消耗：1 AP

效果：选择一个边缘据点。

本大回合该边缘据点被敌方占领时，天门获得 1 威仪，并在该据点生成律令领域，持续 1 回合。

# 三套百宝囊对局差异

| 百宝囊 | 核心风格 | 强项 | 弱点 |
|---|---|---|---|
| 天律囊 | 控图 | 覆盖有害残留、压缩路线、阵地防守 | 依赖据点和领域位置 |
| 镇妖囊 | 封印 | 抓核心技能、制造爆发窗口 | 依赖律令触发或领域站位 |
| 巡天囊 | 追捕 | 机动补足、反边缘偷点 | 中央压制较弱 |

# 宝物背景

| 宝物 | 背景 |
|---|---|
| 昼曜令 | 巡界司制式令牌，刻有昼曜纹，用于在混乱地带临时展开天门法度。 |
| 律令旗 | 天门小队立阵时使用的法旗，插下后即代表此地由天规接管。 |
| 肃行令 | 用于约束越界者行动的短令，常在封锁线内使用。 |
| 裁界印 | 巡界司审讯前封锁目标退路的法印，可将领域优势转化为攻击窗口。 |
| 镇妖钉 | 用于钉住失控夕灵的制式法器，能短暂封住其主动能力。 |
| 破影令 | 专门破除幻身、纸人、影卫等召唤物的执法令。 |
| 天罚令 | 巡界司批准的短时攻击授权，只有积累威仪的小队才能发动。 |
| 锁妖令 | 近距离拘束令，用于在律令领域内强行压制目标。 |
| 巡天符 | 巡界司赶赴异常现场的行军符。 |
| 天门索 | 天门捕缚法器，用于把越界者拖回执法范围。 |
| 踏令步 | 执法队在律令领域内移动的步法。 |
| 巡边令 | 用于反制边缘偷渡与据点偷占的巡界命令。 |
}
