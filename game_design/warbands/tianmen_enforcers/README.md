# 天门执法队（Tianmen Enforcers）

## 战帮定位

3人精英战帮。

天门执法队代表昼曜、律令、封界、追缉、威仪与正面推进。

在 MVP 中，它是青丘使团的主要对照战帮：

- 青丘依靠狐火残留、掩影、裂隙与假身创造不确定性。
- 天门依靠律令领域、昼曜令台、威仪与追缉压缩青丘的行动空间。

玩家选择天门时，不是单纯控制一名强力英雄，而是控制一个小型执法小队：玄照负责下令封界，白烬负责净化稳场，赤霄负责占点与护阵。

---

# 战帮背景

天门执法队隶属于天巡司，是天庭派往人界处理黄昏异常的前线小队。

天庭官方说法称：

- 暮炁异常会导致妖变。
- 青丘狐族会制造幻觉并诱导凡人堕入夕界。
- 黄昏裂隙会吞噬村镇。
- 所有不受天命登记的夕妖，都必须被封印或押回天门审讯。

玄照奉命进入青丘谷，追捕被记录为“暮炁源头”的青丘使团。

起初，天门执法队认为自己是在维护秩序。

但随着战役推进，玄照逐渐发现：

天庭封印的不只是妖。

还有历史、名字与真相。

---

# 核心对立

## 天门要做什么

天门的目标不是追求爆发，而是通过律令和地形控制，把青丘逼入可预测的正面对抗。

天门的核心循环：

```text
占据关键地形
↓
生成律令领域
↓
积累威仪
↓
追缉被标记目标
↓
封印、净化、占点
```

## 青丘要做什么

青丘的目标是打破秩序，通过狐火残留、假身和裂隙不断制造路径与目标的不确定。

青丘的核心循环：

```text
利用掩影 / 裂隙 / 边缘格
↓
生成狐火残留
↓
迷踪与换位
↓
偷点或集火
↓
迫使天门离开律令区域
```

# 战帮成员

- 玄照：队长，封界核心，生成律令领域，积累与消耗威仪。
- 白烬：成员，净化治疗，清除狐火、迷踪与狐火残留。
- 赤霄：成员，护卫占点，守住窄路、中央据点与律令领域。

# 战帮专属机制

## 威仪 authority

威仪是天门执法队的战帮资源。

它不是世界本源，而是天门律令被执行后形成的战场压迫感。

获得方式：

- 天门单位在律令领域内开始行动：+1威仪，每回合最多2。
- 成功对敌方施加封印：+1威仪。
- 清除狐火残留：+1威仪。
- 控制昼曜令台或中央据点到回合结束：+1威仪。

使用方式：

- 增强封印距离。
- 触发追缉。
- 强化防御骰。
- 施放天门审判。

## 律令领域 decree_zone

律令领域是天门的动态覆盖层，对应青丘的狐火残留。

生成来源：玄照技能、天律囊宝物、昼曜令台互动。

基础效果：

- 青丘单位在律令领域中进行特殊位移时，额外消耗1 AP或受到封印判定。
- 假身进入律令领域后，持续时间-1。
- 狐火残留与律令领域不能共存。后生成者覆盖前者。

## 追缉 marked_pursuit

追缉是天门对高机动目标的锁定机制。

被追缉的敌方单位：

- 被天门攻击时，防御骰-1，最低1。
- 从律令领域离开时，天门可获得1威仪。
- 若同时带有狐火，白烬净化该目标时额外清除狐火。

# 战术风格由百宝囊决定

天门本体只是基础战帮。

真正的战术风格由携带的百宝囊决定：

## 天律囊

制造律令领域，封锁狐火路线，适合控图。

## 镇妖囊

追缉、封印、压制化形，适合抓青丘核心单位。

## 巡天囊

提升天门机动与追击能力，减少被青丘拉扯的弱点。

同一支天门执法队携带不同百宝囊时，打法会明显变化。

# 地形关系

天门强地形：

- 昼曜令台
- 中央据点
- 高台
- 窄路
- 律令领域

天门弱地形：

- 掩影
- 边缘据点
- 黄昏裂隙
- 狐火残留密集区域

地形闭环：

```text
青丘用狐火残留污染关键格
↓
天门用律令领域覆盖狐火残留
↓
青丘用裂隙和掩影绕开律令领域
↓
天门用追缉与巡天囊锁定目标
```
