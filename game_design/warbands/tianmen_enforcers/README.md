# 天门执法队（Tianmen Enforcers）

## 战帮定位

3 人精英战帮。

天门执法队代表昼曜、律令、追缉、封印、威仪与正面推进。

在 MVP 中，它是青丘使团的主要对照战帮：

- 青丘依靠狐火残留、掩影、裂隙与假身制造路线和目标变化。
- 天门依靠律令领域、威仪、追缉与封印压缩敌方行动空间。

玩家选择天门时，不是单纯控制一名强力英雄，而是控制一个小型执法小队：

- 玄照负责追缉、封印和律令核心。
- 白烬负责友方净化、治疗和护盾。
- 赤霄负责占点、护阵和阵地承压。

---

# 战帮背景

天门执法队隶属于天巡司，是天庭派往人界处理黄昏异常的前线小队。

天庭官方说法称：

- 夕灵异常会导致妖变。
- 青丘狐族会制造幻术并诱导凡人堕入夕界。
- 黄昏裂隙会吞噬村镇。
- 所有不受天命登记的夕妖，都必须被封印或押回天门审讯。

玄照奉命进入青丘谷，追捕被记录为“夕灵异常源头”的青丘使团。

起初，天门执法队认为自己是在维护秩序。

但随着战役推进，玄照逐渐发现：

天庭封印的不只是妖。

还有历史、名字与真相。

# 队长机制：追缉

追缉是天门的核心标记状态。

MVP 原则：

- 玄照是追缉的唯一稳定来源。
- 其他天门单位和百宝囊主要利用追缉，不再稳定生产追缉。
- 玄照阵亡后，天门仍能依靠律令领域、防御和封印作战，但抓核心能力明显下降。

追缉效果：

- 类型：负面标记状态。
- 持续 1 回合。
- 被追缉目标受到天门攻击时，DEF_DICE -1，最低 1。
- 被追缉目标离开律令领域时，天门获得 1 威仪。

---

# 核心循环

天门的目标不是追求爆发，而是通过律令和地形控制，把敌方逼入可预测的正面对抗。

```text
占据中央 / 高台
↓
生成律令领域
↓
积累威仪
↓
玄照施加追缉
↓
封印、护阵、友方净化、占点
```

青丘的对抗方式是：

```text
利用掩影 / 裂隙 / 边缘格
↓
生成狐火残留
↓
收集狐火
↓
用假身与狐印制造围猎窗口
↓
偷点或集火
```

# 战帮成员

## 玄照

队长。

核心职责：追缉、封印、律令领域、威仪引擎。

玄照存活时，天门追缉体系稳定运转。

玄照阵亡后，天门仍能防守和控图，但对敌方核心单位的锁定能力明显下降。

## 白烬

支援成员。

核心职责：友方净化、治疗、护盾。

白烬不再承担覆盖层战争职责。

清除敌方有害覆盖层主要交给天律囊和律令领域系统。

## 赤霄

精英成员。

核心职责：护阵、占点、承伤、守中央。

赤霄是天门维持阵地的前排。

# 战帮专属机制

## 威仪 authority

威仪是天门执法队的战帮资源。

获得方式：

- 天门单位在律令领域内开始行动：+1 威仪，每大回合最多 2。
- 成功对敌方施加封印：+1 威仪。
- 友方净化成功：+1 威仪，每大回合最多 1。
- 回合结束时控制中央据点：+1 威仪。

使用方式：

- 强化天门攻击。
- 攻击追缉目标时提高压制力。
- 生成律令领域。
- 强化天门审判。

## 律令领域 decree_zone

律令领域是天门的动态覆盖层。

基础效果：

- 与敌方有害覆盖层互斥，后生成者覆盖前者。
- 敌方单位经过律令领域时，下一次移动距离 -1。
- 敌方召唤物进入律令领域后，持续时间 -1。
- 天门单位在律令领域内 DEF_DICE +1。
- 天门单位在律令领域内防御时，支援面自动成功。

MVP 示例：

- 狐火残留对天门而言是敌方有害覆盖层。
- 假身属于敌方召唤物。

## 封印 seal

封印是负面控制状态。

效果：

- 目标无法使用主动技能。

当前 MVP 中，白烬只能进行友方净化，因此不能净化敌方单位身上的封印。

后续若出现可选择敌方单位的净化、驱散或解除控制能力，可以移除封印。

# 战术风格由百宝囊决定

天门本体只是基础战帮。

真正的战术风格由携带的百宝囊决定：

## 天律囊

制造和扩展律令领域，适合控图。

## 镇妖囊

利用追缉和封印抓敌方核心单位。

## 巡天囊

提升天门机动与追击能力，减少被拉扯的弱点。

同一支天门执法队携带不同百宝囊时，打法会明显变化。

# 地形关系

天门强地形：

- 中央据点
- 高台
- 律令领域
- 障碍形成的防守通道

天门弱地形：

- 掩影
- 边缘据点
- 黄昏裂隙
- 狐火残留密集区域

地形闭环：

```text
青丘用狐火残留占用关键格
↓
天门用律令领域覆盖敌方有害覆盖层
↓
青丘用裂隙和掩影绕开律令领域
↓
天门用追缉与巡天囊锁定目标
```
