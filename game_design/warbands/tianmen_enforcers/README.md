# 天门执法队（Tianmen Enforcers）

## 战帮定位

3人精英战帮。

天门执法队代表昼曜、律令、封印、净化、护阵与正面推进。

在 MVP 中，它是青丘使团的主要对照战帮：

- 青丘依靠狐火残留、掩影、裂隙与假身制造路线和目标变化。
- 天门依靠律令、律令领域、封印、净化和护阵压缩敌方行动空间。

当前 MVP 中，天门不使用战帮资源。

---

# 战帮背景

天门执法队隶属于巡界司，是天庭派往人界处理黄昏异常的前线小队。

天庭官方说法称：

- 夕灵异常会导致妖变。
- 青丘狐族会制造幻术并诱导凡人堕入夕界。
- 黄昏裂隙会吞噬村镇。
- 所有不受天命登记的夕妖，都必须被封印或押回天门审讯。

玄照奉命进入青丘谷，追查被记录为“夕灵异常源头”的青丘使团。

起初，天门执法队认为自己是在维护秩序。

但随着战役推进，玄照逐渐发现：

天庭封印的不只是妖。

还有历史、名字与真相。

# 队长机制：律令

律令不是状态。

律令不是标记。

律令是玄照发布的临时战场规则。

玄照存活时，每个大回合开始选择1条律令。

同一时间仅存在1条律令。

持续至本大回合结束。

当前 MVP 律令：

| 律令 | 触发 | 效果 |
|---|---|---|
| 禁行令 | 敌方第一次进入边缘据点或黄昏裂隙 | 该单位获得迷踪 |
| 戒严令 | 敌方第一次占领中央据点 | 中央据点生成律令领域 |
| 缉妖令 | 天门首次协同攻击同一目标 | 本次攻击 ATK_DICE +1 |

# 核心循环

天门的目标不是追求爆发，而是通过律令和地形控制，把敌方逼入可预测的正面对抗。

```text
占据中央 / 高台
↓
玄照颁布律令
↓
律令改变本大回合规则
↓
律令领域形成阵地优势
↓
封印、护阵、友方净化、占点
```

青丘的对抗方式是：

```text
利用掩影 / 裂隙 / 边缘格
↓
生成狐火残留
↓
收集狐火
↓
用假身与狐印制造围猎窗口
↓
偷点或集火
```

# 战帮成员

## 玄照

队长。

核心职责：颁布律令，改变本大回合规则。

玄照存活时，天门律令体系稳定运转。

玄照阵亡后，当前律令保留至本大回合结束，后续不能再颁布新律令。

## 白烬

支援成员。

核心职责：友方净化、治疗。

白烬负责维持队伍清明，让天门小队能够顶住青丘的迷踪和狐印干扰。

## 赤霄

前排成员。

核心职责：护阵、占点、承伤、守中央。

赤霄是天门维持阵地的前排。

# 战帮专属机制

## 律令领域 decree_zone

律令领域是天门的动态覆盖层。

基础效果：

- 与敌方有害覆盖层互斥，后生成者覆盖前者。
- 敌方单位经过律令领域时，获得迷踪。
- 敌方召唤物进入律令领域后，持续时间 -1。
- 天门单位在律令领域内 DEF_DICE +1。
- 天门单位在律令领域内防御时，支援面自动成功。

## 封印 seal

封印是负面控制状态。

效果：

- 目标无法使用主动技能。

当前 MVP 中，白烬只能进行友方净化，因此不能净化敌方单位身上的封印。

后续若出现可选择敌方单位的净化、驱散或解除控制能力，可以移除封印。

# 地形关系

天门强地形：

- 中央据点
- 高台
- 律令领域
- 障碍形成的防守通道

天门弱地形：

- 掩影
- 边缘据点
- 黄昏裂隙
- 狐火残留密集区域

地形闭环：

```text
青丘用狐火残留占用关键格
↓
天门用律令领域覆盖敌方有害覆盖层
↓
青丘用裂隙和掩影绕开律令领域
↓
天门用律令处罚越界行动，并通过封印限制核心技能
```
