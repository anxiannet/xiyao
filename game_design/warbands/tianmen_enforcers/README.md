# 天门执法队（Tianmen Enforcers）

## 战帮定位

3人精英战帮。

代表昼曜、律令、封印、净化与正面推进。

适合MVP教学中的第一支战帮。

---

# 战帮背景

天门执法队隶属于天巡司，是天庭派往人界处理夕火异常的前线小队。

玄照率队进入青丘谷，奉命追捕被记录为“夕妖源头”的青丘使团。

他们并不认为自己残酷。

他们相信：

- 夕火会导致妖变
- 青丘狐族会制造幻觉
- 黄昏裂隙会吞噬村镇
- 封印异常是保护百姓

但在追捕过程中，玄照逐渐发现：

天庭封印的不只是妖。

还有历史、名字与真相。

---

# 战帮成员

- 玄照：队长，前排控制，封印核心。
- 白烬：成员，净化治疗，状态反制。
- 赤霄：成员，护卫占点，地形镇守。

---

# 战帮核心能力

## 天门律令

当天门单位站在昼曜令台、中央据点或高台时，其封印和占点能力增强。

## 正面推进

天门执法队不擅长快速绕后。

他们通过控制关键地形，迫使青丘使团进入正面对抗。

---

# 推荐百宝囊

- 律令囊：封印控制。
- 圣辉囊：净化防守。
- 镇守囊：中央占点。
