# 《夕妖》游戏流程与初始设置

## 一局游戏目标

玩家控制一支完整小队，在3个大回合内争夺气运。

3个大回合结束后，气运更高者获胜。

---

# 初始设置

## 1. 选择模式

- 教学战役
- 自由对战
- AI对战

## 2. 选择小队

- 天门执法队：3人精英小队
- 青丘使团：4人机动小队

## 3. 生成地图

地图由程序生成六角网格、部署区、地形、覆盖层和据点。

背景只负责氛围，规则由格子和标签控制。

据点拥有归属状态。

据点初始归属可以是无归属、青丘或天门。

## 4. 部署单位

- 队长必须部署。
- 单位必须部署在本方部署区。
- 单位不可重叠。

## 5. 初始化资源

```text
青丘：狐火 = 0 / 5
```

每名单位每次激活拥有2 AP。

---

# 回合结构

一局共3个大回合。

每个大回合包含：

1. 行动阶段
2. 结算阶段

---

# 行动阶段

双方轮流激活单位。

每次激活1个单位。

单位可执行：

- 移动
- 攻击
- 核心技能
- 占领
- 待命

每个单位每大回合只能激活1次。

---

# AP消耗

| 动作 | 消耗 |
|---|---|
| 移动 | 1 AP |
| 攻击 | 1 AP |
| 核心技能 | 按技能文本 |
| 占领 | 1 AP |
| 待命 | 消耗剩余AP |

---

# 占领流程

单位执行占领动作时：

1. 选择一个自己所在范围内的据点。
2. 检查该据点范围内双方正式英雄单位数量。
3. 假身、召唤物、覆盖层和区域不计入占领。
4. 若执行占领方正式单位数量大于敌方正式单位数量，则该据点归属改为执行占领方。
5. 若数量相同，或执行占领方数量不大于敌方，则占领失败，据点归属不变。

单位离开据点后，不改变据点归属。

只有被对方明确占领时，据点归属才会改变。

---

# 攻防流程

1. 选择攻击来源。
2. 选择目标。
3. 检查距离、视线、地形和状态。
4. 计算攻击骰。
5. 攻击方掷骰。
6. 防御方掷骰。
7. 比较暴击与成功数。
8. 结算伤害、状态和击败。

---

# 据点与气运

| 来源 | 气运 |
|---|---|
| 中央据点归属方 | +2 |
| 边缘据点归属方 | +1 |
| 击败敌方成员 | +1 |
| 击败敌方队长 | +2 |

自伤击败不计为敌方击败。

---

# 结算阶段

依次处理：

1. 据点气运。
2. 状态倒计时。
3. 假身消失。
4. 狐火残留倒计时。
5. 勘验区消失。
6. 判断胜负。

---

# 同分规则

若3个大回合结束后同分，依次比较：

1. 存活单位更多者胜。
2. 队长存活者胜。
3. 控制中央据点者胜。
