# 《夕妖》命名决议 V2

## 已确认废弃

原名称：暮炁

正式废弃。

不得再作为资源名、本源名或百宝囊名使用。

---

# 四象本源

四象本源固定为：

- 昼曜
- 夜魄
- 晨辉
- 夕灵

对应关系：

```text
昼 ↔ 夜
晨 ↔ 夕
曜 ↔ 魄
辉 ↔ 灵
```

## 夕灵

夕灵是四象本源之一。

它代表黄昏、变化、灵性、化形和过渡。

在 MVP 规则中：

- 夕灵是世界观本源。
- 夕灵不是青丘战帮资源。
- 青丘战帮资源统一称为狐火。

---

# 青丘命名体系

## 狐火

狐火是青丘战帮资源。

青丘通过收集狐火残留获得狐火。

可用于：

- 化形
- 青丘技能循环
- 百宝囊效果

推荐写法：

```text
收集 1 点狐火
消耗 1 点狐火
狐火：3/5
```

不得写作：

```text
获得 1 点夕灵
夕灵：3/5
狐火状态
```

## 狐火残留

狐火残留是地格覆盖层。

代表狐火在地面、墙影或裂隙附近留下的痕迹。

规则含义：

- 青丘单位经过时，收集 1 点狐火。
- 天门单位经过时，获得迷踪。

## 狐印

狐印是青丘施加给敌方单位的标记状态。

规则含义：

- 青丘攻击带狐印的目标时，支援面自动成功。

狐印不是资源。

狐印不是地格覆盖层。

---

# 天门命名体系

## 昼曜

昼曜是四象本源之一。

代表秩序、显现、裁决和律法。

## 威仪

威仪是天门战帮资源。

天门通过律令领域、封印、清除狐火残留和控制中央据点获得威仪。

## 律令领域

律令领域是天门地格覆盖层。

用于压缩青丘路线，并让天门单位获得防御支援。

---

# 命名层级

```text
四象本源
├─ 昼曜 → 威仪 / 律令领域 / 封印 / 追缉
├─ 夜魄 → 后续势力扩展
├─ 晨辉 → 净化 / 护盾 / 治疗
└─ 夕灵 → 狐火 / 狐火残留 / 狐印 / 化形
```

# 后续文档规范

必须使用：

- 夕灵：四象本源
- 狐火：青丘战帮资源
- 狐火残留：地格覆盖层
- 狐印：单位标记状态
- 威仪：天门战帮资源
- 律令领域：天门覆盖层

不得再新增或继续使用：

- 暮炁
- 暮炁值
- 暮炁囊
- 暮灯
- 夕火（资源含义）
- 狐火状态
- 夕灵值（作为 MVP 青丘资源）
